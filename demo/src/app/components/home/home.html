Getting Started

Welcome to the Synerity UI Demo Application. This application showcases our comprehensive component library built with Angular and modern design principles.

🎨 Design System

Consistent design tokens, colors, and typography across all components.

♿ Accessibility

WCAG 2.1 AA compliant components with full keyboard navigation.

📱 Responsive

Mobile-first design that works seamlessly across all device sizes.

⚡ Performance

Optimized bundle sizes and lazy loading for optimal performance.

Component Categories

Foundation: Buttons, Inputs, Cards, Badges, Avatars
Forms: Checkboxes, Radio Buttons, Selects, Date Pickers, Color Pickers
Data: Tables, Lists, Pagination, Search, Filters
Overlay: Modals, Tooltips, Popovers, Dropdowns
Navigation: Menus, Breadcrumbs, Tabs, Sidebars
Media: Images, Videos, Carousels, Galleries

Features

  • 🎯 TypeScript Support: Full TypeScript support with comprehensive type definitions
  • 🔧 Customizable: Extensive theming and customization options
  • 🧪 Well Tested: Comprehensive test coverage with unit and integration tests
  • 📚 Documentation: Detailed documentation with examples and API references
  • 🚀 Modern Stack: Built with Angular 19+ and modern web standards
  • 🎨 Tailwind CSS: Utility-first CSS framework integration

Installation

npm install @synerity/ui

Then import the styles and components in your Angular application:

@import '@synerity/ui/synerity-ui.css';
import {{ '{' }} Button {{ '}' }} from '@synerity/ui';

Splitter Component

A resizable panel component that allows users to split content into adjustable sections. Perfect for creating layouts with draggable dividers, code editors, and dashboard panels.

Basic Usage

Left Panel

This is the left panel content. You can drag the gutter to resize the panels.

The splitter allows users to adjust the layout according to their preferences.

Right Panel

This is the right panel content. Try dragging the gutter between the panels.

The panels will resize dynamically as you drag the divider.

 
<sui-splitter [layout]="'horizontal'" [gutterSize]="4">
  <div slot="panel1">
    <h3>Left Panel</h3>
    <p>Panel content here...</p>
  </div>
  <div slot="panel2">
    <h3>Right Panel</h3>
    <p>Panel content here...</p>
  </div>
</sui-splitter>

API Reference

Property Type Default Description
layout 'horizontal' | 'vertical' 'horizontal' Orientation of the splitter
gutterSize number 4 Size of the draggable gutter in pixels
minSizes number[] [0, 0] Minimum sizes for panels as percentages
maxSizes number[] [100, 100] Maximum sizes for panels as percentages
style any {} Inline styles for the splitter
styleClass string '' CSS class names for the splitter

Events

Event Type Description
onResizeEnd EventEmitter<any> Fired when resizing is completed

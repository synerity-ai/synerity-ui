API Reference

Properties

@Input() disabled: boolean = false; // Disable the input @Input() name: string = ''; // Form field name @Input() inputId: string = ''; // Input element ID @Input() placeholder: string = ''; // Placeholder text @Input() minLength: number = 1; // Minimum characters before showing suggestions @Input() options: Array<string | { label: string; value: unknown }> = []; // Suggestion options @Output() change: EventEmitter<unknown>; // Value change event

Methods

onInput(event: Event): void; // Handle input changes onFocus(): void; // Handle focus event onBlur(): void; // Handle blur event onKeyDown(event: KeyboardEvent): void; // Handle keyboard navigation select(item: string | object): void; // Select an item clear(): void; // Clear the selected value

Keyboard Navigation

ArrowDown // Navigate to next suggestion ArrowUp // Navigate to previous suggestion Enter // Select highlighted suggestion Escape // Close dropdown

CSS Classes

.sui-auto-complete // Base component class .sui-auto-complete-small // Small size variant .sui-auto-complete-normal // Normal size variant (default) .sui-auto-complete-large // Large size variant .sui-auto-complete-error // Error state .sui-auto-complete-success // Success state

Card Component Demo

Cards are flexible containers for grouping related content and actions. They provide a clean way to display information with optional headers, content, and footers.

Simple card with default styling and basic content

Interactive Features

Cards with interactive behaviors and states

Clickable Card

Click anywhere on this card to trigger an action.
<sui-card header="Clickable Card" [clickable]="true" (click)="onCardClick($event)"> Click anywhere on this card... </sui-card>

Hoverable Card

Hover over this card to see the effect.
<sui-card header="Hoverable Card" [hoverable]="true"> Hover over this card to see the effect. </sui-card>

Loading Card

This card shows a loading state.

Disabled Card

This card is disabled and non-interactive.

API Reference

Properties

@Input() variant: 'default' | 'outlined' | 'elevated' | 'filled' = 'default'; // Visual style variant @Input() size: 'sm' | 'md' | 'lg' = 'md'; // Card size @Input() clickable: boolean = false; // Make card clickable @Input() disabled: boolean = false; // Disable card interactions @Input() header: string = ''; // Card header text @Input() subheader: string = ''; // Card subheader text @Input() showHeader: boolean = true; // Show/hide header section @Input() showFooter: boolean = false; // Show/hide footer section @Input() loading: boolean = false; // Show loading state @Input() hoverable: boolean = false; // Enable hover effects @Input() borderless: boolean = false; // Remove card borders @Output() click: EventEmitter<MouseEvent>; // Card click event @Output() headerClick: EventEmitter<MouseEvent>; // Header click event

Methods

onCardClick(event: MouseEvent): void; // Handle card clicks onHeaderClick(event: MouseEvent): void; // Handle header clicks

CSS Classes

.sui-card // Base card class .sui-card-sm // Small size .sui-card-md // Medium size .sui-card-lg // Large size .sui-card-default // Default variant .sui-card-outlined // Outlined variant .sui-card-elevated // Elevated variant .sui-card-filled // Filled variant .sui-card-clickable // Applied when clickable .sui-card-hoverable // Applied when hoverable .sui-card-disabled // Applied when disabled .sui-card-borderless // Applied when borderless

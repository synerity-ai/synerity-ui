API Reference

Properties

@Input() disabled: boolean = false; // Disable all dropdowns @Input() name: string = ''; // Form field name @Input() levelsPlaceholder: string[] = []; // Placeholder for each level @Input() options: HierarchicalOption[] = []; // Hierarchical data structure @Output() change: EventEmitter<unknown>; // Value change event

HierarchicalOption Interface

interface HierarchicalOption { label: string; // Display text value?: unknown; // Option value children?: HierarchicalOption[]; // Child options for next level }

Methods

levelOptions(level: number): HierarchicalOption[]; // Get options for specific level onSelect(level: number, idx: number): void; // Handle selection at level

CSS Classes

.sui-cascade-select // Base component class .sui-cascade-select-disabled // Disabled state .cascade-select-dropdown // Individual dropdown

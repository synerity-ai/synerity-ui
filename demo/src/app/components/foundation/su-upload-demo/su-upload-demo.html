Upload Component

Upload component provides drag-and-drop file upload functionality with validation, progress tracking, and customizable options.

Basic Upload

Simple file upload without restrictions

<sui-upload 
  [multiple]="false"
  (onUpload)="handleUpload($event)"
></sui-upload>

Image Upload

Multiple image files with size limit (5MB max)

<sui-upload 
  [multiple]="true"
  accept="image/*"
  [maxFileSize]="5242880"
  (onError)="handleError($event)"
></sui-upload>

Document Upload

Specific file types with validation

<sui-upload 
  [multiple]="true"
  accept=".pdf,.doc,.docx,.txt"
  [maxFileSize]="10485760"
></sui-upload>

Auto Upload

Files upload automatically when selected

<sui-upload 
  [multiple]="true"
  [auto]="true"
  (onBeforeUpload)="handleBeforeUpload($event)"
  (onProgress)="handleProgress($event)"
></sui-upload>

Disabled State

Upload component in disabled state

<sui-upload 
  [disabled]="true"
></sui-upload>

API Reference

Properties (Inputs)

Name Type Default Description
multiple boolean false Allow multiple file selection
accept string '' Accepted file types (e.g., 'image/*', '.pdf')
maxFileSize number 0 Maximum file size in bytes
disabled boolean false Disable the upload component
auto boolean false Auto-upload files when selected
customUpload boolean false Use custom upload handler

Events (Outputs)

Name Parameters Description
onUpload { files: File[] } Emitted when files are uploaded
onBeforeUpload { files: File[] } Emitted before upload starts
onProgress { progress: number } Emitted during upload progress
onError { file: File, error: string } Emitted on upload error
onSelect { files: File[] } Emitted when files are selected
onRemove { file: File } Emitted when file is removed

Features

  • Drag & Drop: Native drag-and-drop support
  • File Validation: Type and size validation
  • Progress Tracking: Upload progress monitoring
  • Error Handling: Comprehensive error reporting
  • Accessibility: ARIA labels and keyboard support
  • Responsive: Mobile-friendly design

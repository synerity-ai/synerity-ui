Stepper Component

A step-by-step navigation component that guides users through multi-step processes. Perfect for forms, wizards, onboarding flows, and complex user journeys.

Basic Usage

Step {{ activeIndex + 1 }}: {{ basicSteps[activeIndex]?.label }}

This is the content for step {{ activeIndex + 1 }}. Click on any step to navigate directly to it.

 
<sui-stepper 
  [steps]="steps" 
  [activeIndex]="activeIndex"
  [orientation]="'horizontal'"
  (onActiveIndexChange)="onActiveIndexChange($event)">
  <div class="step-content">
    <h3>Step Content</h3>
    <p>Content for current step...</p>
  </div>
</sui-stepper>

API Reference

Property Type Default Description
activeIndex number 0 Index of the currently active step
orientation 'horizontal' | 'vertical' 'horizontal' Orientation of the stepper
linear boolean false Whether the stepper is in linear mode
steps Array [] Array of step objects with label, content, disabled, completed properties
style any {} Inline styles for the stepper
styleClass string '' CSS class names for the stepper

Tree Select Component

Tree Select displays hierarchical data in a dropdown tree structure. Only leaf nodes (items without children) are selectable.

Basic Usage

Simple hierarchical selection

Selected: {{ selectedLocation || '(none)' }}

<sui-tree-select 
  [options]="locations" 
  [(ngModel)]="selectedLocation"
  placeholder="Choose a location"
></sui-tree-select>

Multiple Examples

Different hierarchical data sets

{{ selectedLocation || '(none)' }}

{{ selectedCategory || '(none)' }}

{{ selectedDepartment || '(none)' }}

// Define hierarchical data
locations = [
  {
    label: 'United States',
    children: [
      { label: 'California', value: 'CA' },
      { label: 'New York', value: 'NY' }
    ]
  },
  {
    label: 'Europe',
    children: [
      { label: 'United Kingdom', value: 'UK' },
      { label: 'Germany', value: 'DE' }
    ]
  }
];

<sui-tree-select [options]="locations" [(ngModel)]="selected"></sui-tree-select>

Disabled State

Non-interactive tree select

<sui-tree-select [options]="locations" [disabled]="true"></sui-tree-select>

API Reference

Properties (Inputs)

Name Type Default Description
options Array<{ label, value?, children? }> [] Hierarchical tree data
disabled boolean false Disabled state
name string '' Name attribute
placeholder string '' Placeholder text

Events (Outputs)

Name Parameters Description
change unknown Emitted when selection changes

Usage Notes

  • Hierarchical Data: Define parent nodes with children array
  • Leaf Nodes: Only leaf nodes (items without children) are selectable
  • Parent Nodes: Parent nodes (with children) are displayed but not selectable
  • Value Display: Shows the label of the selected leaf node
  • Dropdown: Click trigger to open/close the tree dropdown

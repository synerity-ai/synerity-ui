Float Label Component

Float Label provides an animated floating label for form inputs, textareas, and selects. The label floats above the input when focused or when the field has a value.

Basic Usage

Simple floating label with text input

<sui-float-label label="Username">
  <input type="text" [(ngModel)]="username" placeholder=" " />
</sui-float-label>

Multiple Input Types

Float labels work with different input types

<sui-float-label label="Email Address">
  <input type="email" [(ngModel)]="email" placeholder=" " />
</sui-float-label>

<sui-float-label label="Password">
  <input type="password" [(ngModel)]="password" placeholder=" " />
</sui-float-label>

With Textarea

Float label works with multiline textareas

<sui-float-label label="Message">
  <textarea [(ngModel)]="message" placeholder=" " rows="4"></textarea>
</sui-float-label>

With Select Dropdown

Float label works with select elements

<sui-float-label label="Country">
  <select [(ngModel)]="country">
    <option value="">Select a country</option>
    <option value="us">United States</option>
    <option value="uk">United Kingdom</option>
  </select>
</sui-float-label>

Required Field

Show required indicator with asterisk

<sui-float-label label="Full Name" [required]="true">
  <input type="text" [(ngModel)]="requiredField" placeholder=" " required />
</sui-float-label>

Error State

Show validation error state

Please enter a valid email address

<sui-float-label label="Email (Invalid)" styleClass="sui-float-label-error">
  <input type="email" [(ngModel)]="errorField" placeholder=" " />
</sui-float-label>

Success State

Show validation success state

Email is valid!

<sui-float-label label="Email (Valid)" styleClass="sui-float-label-success">
  <input type="email" [(ngModel)]="successField" placeholder=" " />
</sui-float-label>

Outlined Variant

Thicker border style

Non-interactive disabled input

API Reference

Properties (Inputs)

Name Type Default Description
label string '' Text label that floats above the input
for string '' ID of the associated input element
required boolean false Whether to show required indicator (*)
style any {} Inline styles to apply to the container
styleClass string '' Additional CSS class(es) to apply

CSS Classes

Class Description
sui-float-label Base float label class
sui-float-label-focused Applied when input is focused or has value
sui-float-label-required Required field indicator
sui-float-label-error Error state styling
sui-float-label-success Success state styling
sui-float-label-disabled Disabled state styling
sui-float-label-outlined Outlined variant with thicker border
sui-float-label-filled Filled variant (Material Design style)

Usage Notes

  • Placeholder Requirement: Always use placeholder=" " (single space) for proper label animation
  • Input Types: Works with input, textarea, and select elements
  • Label Animation: Label floats up when input is focused or has a value
  • Accessibility: Use the for attribute to associate label with input ID
  • Styling: Combine multiple styleClass values separated by spaces

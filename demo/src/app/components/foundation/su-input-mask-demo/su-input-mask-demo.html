Input Mask Component

Input Mask enforces specific input patterns using mask definitions. Supports phone numbers, dates, credit cards, SSN, and custom patterns.

Basic Usage

Phone number mask

Value: {{ phone || '(empty)' }}

<sui-input-mask 
  mask="(999) 999-9999"
  [(ngModel)]="phone"
  placeholder="(___) ___-____"
></sui-input-mask>

Understanding mask pattern characters

9 Numeric digit (0-9)
A Alphabetic character (A-Z, a-z)
* Alphanumeric (0-9, A-Z, a-z)
Other Literal characters (-, /, (, ), etc.)

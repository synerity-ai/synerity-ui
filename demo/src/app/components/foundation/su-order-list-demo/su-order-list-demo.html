Order List Component

Order List allows users to reorder items within a list using up/down controls or drag-and-drop.

Basic Usage

Reorder tasks using arrow buttons

<sui-order-list 
  [value]="tasks" 
  header="Task Priority Order"
  (onReorder)="handleReorder($event)"
></sui-order-list>

With Filtering

Filter items with search input

<sui-order-list 
  [value]="features" 
  [filter]="true"
  filterPlaceholder="Search features..."
></sui-order-list>

With Selection

Select items with checkboxes

Selected: {{ selectedTasks.length }} task(s) 

<sui-order-list 
  [value]="tasks" 
  (onSelectionChange)="handleSelectionChange($event)"
></sui-order-list>

API Reference

Properties (Inputs)

Name Type Default Description
value any[] [] Array of items to display
header string '' Header title for the list
filter boolean false Enable filtering
filterPlaceholder string 'Search' Placeholder for filter input
dragdrop boolean false Enable drag and drop (future)

Events (Outputs)

Name Parameters Description
onReorder { items, direction, index } Emitted when items are reordered
onSelectionChange any[] Emitted when selection changes

Controls

  • Move Up: Moves item one position up in the list
  • Move Down: Moves item one position down in the list
  • First/Last buttons: Disabled when item is at boundary
  • Selection: Check items to track selected state

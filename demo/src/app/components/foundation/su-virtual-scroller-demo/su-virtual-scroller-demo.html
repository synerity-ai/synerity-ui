API Reference

Properties (Inputs)

Name Type Default Description
value any[] [] Array of items to scroll
itemSize number 50 Height of each item in pixels
scrollHeight string '200px' Height of scroll viewport
scrollWidth string '100%' Width of scroll viewport
orientation 'vertical' | 'horizontal' 'vertical' Scroll direction
lazy boolean false Enable lazy loading
delay number 0 Delay for lazy load (ms)

Events (Outputs)

Name Parameters Description
onScroll { scrollTop, scrollLeft } Emitted when user scrolls
onScrollIndexChange { first, last } Emitted when visible range changes
onLazyLoad { first, last } Emitted for lazy loading

Performance Benefits

  • Optimized Rendering: Only renders items visible in viewport
  • Large Datasets: Handles 10,000+ items smoothly
  • Memory Efficient: Minimal DOM nodes at any time
  • Smooth Scrolling: 60fps performance with large lists

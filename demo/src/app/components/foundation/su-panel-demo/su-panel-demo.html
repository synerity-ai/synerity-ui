Panel Component

A versatile container component with optional header and toggleable content. Perfect for organizing content, creating collapsible sections, and building dashboard layouts.

Basic Usage

This is a basic panel with a header and content area.

You can put any content inside the panel body.

 
<sui-panel header="Basic Panel">
  <p>This is a basic panel with a header and content area.</p>
  <p>You can put any content inside the panel body.</p>
</sui-panel>

Toggleable Panel

This panel can be collapsed and expanded by clicking the header.

The toggle functionality is controlled by the toggleable property.

Current state: {{ isCollapsed ? 'Collapsed' : 'Expanded' }}

 
<sui-panel 
  header="Toggleable Panel" 
  [toggleable]="true"
  [collapsed]="isCollapsed"
  (onToggle)="onPanelToggle($event)">
  <p>Panel content here...</p>
</sui-panel>

Warning Panel

API Reference

Property Type Default Description
header string '' Header text for the panel
toggleable boolean false Whether the panel can be collapsed/expanded
collapsed boolean false Initial collapsed state of the panel
showHeader boolean true Whether to show the panel header
style any {} Inline styles for the panel
styleClass string '' CSS class names for the panel

Events

Event Type Description
onToggle EventEmitter<any> Fired when the panel is toggled (if toggleable)

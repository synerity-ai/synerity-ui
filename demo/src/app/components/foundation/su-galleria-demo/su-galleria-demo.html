Galleria Component

Galleria provides a fullscreen image gallery with thumbnails, navigation, and advanced viewing options. Perfect for showcasing high-resolution images and photo collections.

Basic Usage

Click on any image to open the fullscreen gallery

{{ image.title }} 
<sui-galleria 
  [value]="galleryImages"
  [visible]="galleryVisible"
  [showThumbnails]="true"
  [showIndicators]="true"
  (onImageChange)="onImageChange($event)"
></sui-galleria>

API Reference

Properties (Inputs)

Name Type Default Description
value GalleriaItem[] [] Array of gallery items
activeIndex number 0 Index of the currently active image
visible boolean false Gallery visibility state
showThumbnails boolean true Show thumbnail navigation
showIndicators boolean true Show dot indicators
circular boolean true Enable circular navigation
fullScreen boolean false Enable fullscreen mode

Events (Outputs)

Name Parameters Description
onImageChange number Emitted when the active image changes
onShow void Emitted when gallery is shown
onHide void Emitted when gallery is hidden

Item Data Structure

  • src: High-resolution image URL (required)
  • alt: Alt text for accessibility (optional)
  • title: Image title (optional)
  • caption: Image description (optional)
  • thumbnailSrc: Thumbnail image URL (optional, falls back to src)

Features

  • Fullscreen Display: Immersive viewing experience
  • Thumbnail Navigation: Quick image selection
  • Keyboard Navigation: Arrow keys and ESC support
  • Touch Gestures: Swipe navigation on mobile
  • Backdrop Blur: Modern glassmorphism effect
  • Responsive Design: Optimized for all screen sizes

Pick List Component

Pick List transfers items between two lists with selection support and transfer controls.

Basic Usage

Transfer cities between available and selected lists

<sui-pick-list 
  [source]="availableCities" 
  [target]="selectedCities"
  sourceHeader="Available Cities"
  targetHeader="Selected Cities"
></sui-pick-list>

With Filtering

Filter both source and target lists

<sui-pick-list 
  [source]="availableSkills" 
  [target]="selectedSkills"
  [filter]="true"
  filterPlaceholder="Search..."
></sui-pick-list>

API Reference

Properties (Inputs)

Name Type Default Description
source any[] [] Array of source items
target any[] [] Array of target items
sourceHeader string 'Available' Header for source list
targetHeader string 'Selected' Header for target list
filter boolean true Enable filtering
filterPlaceholder string 'Search' Placeholder for filter inputs
showSourceControls boolean true Show source to target controls
showTargetControls boolean true Show target to source controls

Events (Outputs)

Name Parameters Description
onMoveToTarget { items: any[] } Emitted when items moved to target
onMoveToSource { items: any[] } Emitted when items moved to source
onMoveAllToTarget { items: any[] } Emitted when all moved to target
onMoveAllToSource { items: any[] } Emitted when all moved to source

Transfer Controls

  • → (Right Arrow): Move selected items from source to target
  • ⇒ (Double Right): Move all items from source to target
  • ← (Left Arrow): Move selected items from target to source
  • ⇐ (Double Left): Move all items from target to source

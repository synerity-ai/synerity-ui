Input Number Component

Input Number provides numeric input with increment/decrement buttons. Supports min/max bounds, step values, and custom formatting.

Basic Usage

Simple numeric input with spinner

Value: {{ quantity }}

<sui-input-number [(ngModel)]="quantity"></sui-input-number>

With Constraints

Min/max bounds and step values

<!-- With min/max -->
<sui-input-number 
  [(ngModel)]="price"
  [min]="0"
  [max]="999.99"
  [step]="0.01"
></sui-input-number>

<!-- With step increment -->
<sui-input-number 
  [(ngModel)]="temperature"
  [step]="5"
></sui-input-number>

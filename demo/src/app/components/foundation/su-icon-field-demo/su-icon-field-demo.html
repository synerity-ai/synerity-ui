Icon Field Component

Icon Field adds decorative icons to input fields with flexible positioning (left or right). Supports different sizes and automatically adjusts input padding for proper icon placement.

API Reference

Properties (Inputs)

Name Type Default Description
icon string '' Icon character or emoji to display
position 'left' | 'right' 'left' Position of the icon relative to the input
size 'small' | 'normal' | 'large' 'normal' Size variant affecting icon and padding
style any {} Inline styles to apply to the container
styleClass string '' Additional CSS class(es) to apply

CSS Classes

Class Description
sui-icon-field Base icon field class
sui-icon-field-left Left positioned icon
sui-icon-field-right Right positioned icon
sui-icon-field-small Small size variant
sui-icon-field-normal Normal size variant (default)
sui-icon-field-large Large size variant
sui-icon-field-icon Icon element styling

Usage Notes

  • Icon Characters: Use emojis or icon font characters (e.g., Font Awesome, Material Icons)
  • Automatic Padding: Input padding is automatically adjusted based on icon position and size
  • Focus State: Icon color changes to blue when input is focused
  • Disabled State: Icon becomes gray and semi-transparent when input is disabled
  • Input Types: Works with all standard HTML input types (text, email, password, search, tel, url, date, etc.)
  • Responsive: Icon and padding scale appropriately across different screen sizes

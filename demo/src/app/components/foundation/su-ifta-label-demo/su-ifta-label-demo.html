IFTA Label Component

IFTA (If-True-False-Alt) Label displays conditional labels with an additional alternative state. Perfect for three-state scenarios like Approved/Pending/Rejected.

Basic Usage

Conditional label with alternative state

<sui-ifta-label [condition]="true" trueLabel="Approved" falseLabel="Rejected" altLabel="Pending"></sui-ifta-label>

Toggle between states to see true/false/alt labels

API Reference

Properties (Inputs)

Name Type Default Description
condition boolean true Determines which label to display
trueLabel string '' Label shown when condition is true
falseLabel string '' Label shown when condition is false and no alt label
altLabel string '' Alternative label shown when condition is false (takes precedence over falseLabel)
style any {} Inline styles
styleClass string '' Additional CSS classes

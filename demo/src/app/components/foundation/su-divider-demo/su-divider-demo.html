Divider Component

A visual separator component that creates horizontal or vertical dividers with different styles and alignments. Perfect for separating content sections and creating visual hierarchy.

Basic Usage

This is content above the divider.

This is content below the divider.

<sui-divider></sui-divider>

API Reference

Property Type Default Description
type 'solid' | 'dashed' | 'dotted' 'solid' Style of the divider line
layout 'horizontal' | 'vertical' 'horizontal' Orientation of the divider
align 'left' | 'center' | 'right' 'center' Alignment of the divider
style any {} Inline styles for the divider
styleClass string '' CSS class names for the divider

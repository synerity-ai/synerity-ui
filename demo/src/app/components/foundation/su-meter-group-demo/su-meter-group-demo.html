Meter Group Component

Meter Group displays proportional data as horizontal or vertical bars. Perfect for budget allocation, performance metrics, and data visualization.

Basic Usage

Simple horizontal meter with default colors

<sui-meter-group 
  [value]="simpleData"
  [showValue]="true"
  [showLabel]="true"
></sui-meter-group>

Sales Data with Colors

Quarterly sales with custom colors and icons

<sui-meter-group 
  [value]="salesData"
></sui-meter-group>

// Data structure
salesData = [
  { label: 'Q1', value: 25, color: '#3b82f6', icon: '📈' },
  { label: 'Q2', value: 35, color: '#10b981', icon: '📊' }
];

API Reference

Properties (Inputs)

Name Type Default Description
value MeterItem[] [] Array of meter items
min number 0 Minimum value for calculations
max number 100 Maximum value for calculations
showValue boolean true Display item values
showLabel boolean true Display item labels
orientation 'horizontal' | 'vertical' 'horizontal' Meter orientation

Data Structure

  • label: Display text for the meter item
  • value: Numeric value (determines bar width)
  • color: Custom color (optional, auto-generated if not provided)
  • icon: Icon character (optional)

Features

  • Proportional Display: Bar widths based on relative values
  • Color Coding: Automatic or custom colors
  • Icons: Support for emoji or character icons
  • Responsive: Adapts to container size
  • Interactive: Hover effects and scaling

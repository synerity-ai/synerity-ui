Carousel Component

Carousel displays a series of images or content in a rotating slideshow format. Perfect for product showcases, image galleries, and content presentation.

Basic Usage

Simple image carousel with navigation and indicators

Current slide: {{ currentIndex + 1 }} of {{ productImages.length }}

<sui-carousel 
  [value]="productImages"
  [showIndicators]="true"
  [showNavigators]="true"
  (onPageChange)="onPageChange($event)"
></sui-carousel>

Autoplay Carousel

Automatically advancing slideshow with 3-second intervals

<sui-carousel 
  [value]="travelImages"
  [autoplayInterval]="3000"
  [circular]="true"
></sui-carousel>

Indicators Only

Carousel with indicators but no navigation arrows

<sui-carousel 
  [value]="simpleImages"
  [showNavigators]="false"
  [showIndicators]="true"
></sui-carousel>

Switch between different carousel configurations

API Reference

Properties (Inputs)

Name Type Default Description
value CarouselItem[] [] Array of carousel items
activeIndex number 0 Index of the currently active slide
circular boolean true Enable circular navigation
autoplayInterval number 0 Autoplay interval in milliseconds (0 to disable)
showIndicators boolean true Show slide indicators
showNavigators boolean true Show navigation arrows

Events (Outputs)

Name Parameters Description
onPageChange number Emitted when the active slide changes

Item Data Structure

  • src: Image URL (required)
  • alt: Alt text for accessibility (optional)
  • title: Slide title (optional)
  • caption: Slide description (optional)

Features

  • Image Sliding: Smooth transitions between slides
  • Navigation: Previous/next arrow controls
  • Indicators: Dot indicators for direct navigation
  • Autoplay: Automatic slide advancement
  • Circular: Loop back to first slide after last
  • Responsive: Adapts to different screen sizes

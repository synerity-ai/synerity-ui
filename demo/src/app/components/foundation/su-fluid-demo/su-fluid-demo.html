Fluid Component

A responsive grid container component that creates flexible layouts with customizable columns and gutters. Perfect for creating responsive card grids, image galleries, and content layouts.

API Reference

Property Type Default Description
columns number 12 Number of columns in the grid
gutter number 16 Gutter size between items in pixels
responsive boolean true Whether the grid should be responsive
style any {} Inline styles for the fluid container
styleClass string '' CSS class names for the fluid container

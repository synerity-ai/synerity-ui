Paginator Component

Paginator provides navigation controls for paginated data with configurable page sizes and display options.

Basic Usage

Simple pagination with 100 total records

#{{ item.id }}

{{ item.title }}

{{ item.description }}

<sui-paginator 
  [rows]="10" 
  [totalRecords]="100"
  (onPageChange)="handlePageChange($event)"
></sui-paginator>

With Page Report

Display current page information

<sui-paginator 
  [showCurrentPageReport]="true"
  currentPageReportTemplate="Showing {first} to {last} of {totalRecords} items"
></sui-paginator>

Rows Per Page Options

Allow users to change page size

<sui-paginator 
  [showRowsPerPageOptions]="true"
  [rowsPerPageOptions]="[5, 10, 20, 50]"
></sui-paginator>

Custom Page Link Size

Control number of page links displayed

API Reference

Properties (Inputs)

Name Type Default Description
rows number 10 Number of rows per page
totalRecords number 0 Total number of records
pageLinkSize number 5 Number of page links to display
rowsPerPageOptions number[] [5, 10, 20, 30] Options for rows per page dropdown
showCurrentPageReport boolean false Show current page info
currentPageReportTemplate string '({currentPage} of {totalPages})' Template for page report
showFirstLastIcon boolean true Show first/last page buttons
showPageLinks boolean true Show page number links

Events (Outputs)

Name Parameters Description
onPageChange { first, rows, page, pageCount } Emitted when page changes

Template Tokens

  • {currentPage}: Current page number (1-based)
  • {totalPages}: Total number of pages
  • {first}: Index of first record on page
  • {last}: Index of last record on page
  • {totalRecords}: Total number of records

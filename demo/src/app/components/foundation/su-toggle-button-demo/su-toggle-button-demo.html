Toggle Button Component

Toggle Button provides mutually exclusive or multiple selection options in a button group format.

Basic Usage

Single selection toggle

Selected: {{ selectedView }}

<sui-toggle-button [options]="viewOptions" [(ngModel)]="selectedView"></sui-toggle-button>

With Icons

Buttons with icon decorations

Text Alignment

Text Format

Multiple Selection

Select multiple options

Selected: {{ selectedOptions.length > 0 ? selectedOptions.join(', ') : '(none)' }}

<sui-toggle-button [options]="options" [(ngModel)]="selected" [multiple]="true"></sui-toggle-button>

API Reference

Properties (Inputs)

Name Type Default Description
options Array<{ label, value, icon? }> [] Button options
multiple boolean false Allow multiple selections
size 'sm' | 'md' | 'lg' 'md' Size variant
variant 'default' | 'success' | 'warning' | 'danger' | 'purple' 'default' Color variant
style 'default' | 'outline' | 'ghost' 'default' Style variant
shape 'default' | 'rounded' | 'pill' 'default' Shape variant

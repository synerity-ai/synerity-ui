API Reference

Properties

@Input() disabled: boolean = false; // Disable the textarea @Input() name: string = ''; // Form field name @Input() inputId: string = ''; // Input element ID @Input() placeholder: string = ''; // Placeholder text @Input() rows: number = 3; // Number of visible rows @Input() autosize: boolean = false; // Auto-resize based on content @Output() change: EventEmitter<string>; // Value change event

Methods

onInput(event: Event): void; // Handle input changes onBlur(): void; // Handle blur event adjustHeight(): void; // Adjust height for autosize (private)

CSS Classes

.sui-textarea-container // Container wrapper .sui-textarea-input // Textarea input element

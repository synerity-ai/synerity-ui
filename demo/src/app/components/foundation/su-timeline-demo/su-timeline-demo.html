Timeline Component

Timeline displays a sequence of events in chronological order. Perfect for project tracking, order status, and process flows.

Basic Usage

Simple vertical timeline with status indicators

Selected: {{ selectedEvent.content }} ({{ selectedEvent.date }}) 

<sui-timeline 
  [value]="simpleEvents"
  (onEventClick)="handleEventClick($event)"
></sui-timeline>

Project Timeline

Complex project with various status types

<sui-timeline 
  [value]="projectEvents"
  align="alternate"
></sui-timeline>

API Reference

Properties (Inputs)

Name Type Default Description
value TimelineEvent[] [] Array of timeline events
align 'left' | 'right' | 'alternate' 'left' Content alignment for vertical timeline
layout 'vertical' | 'horizontal' 'vertical' Timeline layout direction
dataKey string '' Key for data binding

Events (Outputs)

Name Parameters Description
onEventClick TimelineEvent Emitted when event is clicked

Event Data Structure

  • status: 'completed' | 'success' | 'error' | 'warning' (optional)
  • content: Event description text
  • date: Event date/time (optional)
  • icon: Custom icon character (optional)
  • color: Custom color (optional)

Status Colors

  • completed: Blue (#3b82f6)
  • success: Green (#10b981)
  • error: Red (#ef4444)
  • warning: Yellow (#f59e0b)
  • default: Gray (#9ca3af)

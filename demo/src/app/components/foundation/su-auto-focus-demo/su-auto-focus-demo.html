API Reference

Properties

@Input() delay: number = 0; // Delay before focusing (milliseconds) @Input() condition: boolean = true; // Only focus if condition is true @Input() style: any = {}; // Custom styles @Input() styleClass: string = ''; // Custom CSS classes

Methods

focus(): void; // Programmatically focus the element

CSS Classes

.sui-auto-focus // Base component class

Usage Notes

// Auto-focus is a wrapper component // Place it around the element you want to focus // Commonly used for: // - First field in forms // - Search inputs // - Modal dialogs // - Error messages requiring attention

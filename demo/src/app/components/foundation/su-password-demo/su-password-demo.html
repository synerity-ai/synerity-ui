API Reference

Properties

@Input() disabled: boolean = false; // Disable the input @Input() name: string = ''; // Form field name @Input() inputId: string = ''; // Input element ID @Input() placeholder: string = ''; // Placeholder text @Input() minLength: number | null = null; // Minimum password length @Input() toggleMask: boolean = true; // Show/hide toggle button @Output() change: EventEmitter<string>; // Value change event

Methods

onInput(event: Event): void; // Handle input changes onBlur(): void; // Handle blur event toggleVisibility(): void; // Toggle password visibility

CSS Classes

.sui-password // Base component class .sui-password-disabled // Disabled state .sui-password-sm // Small size variant .sui-password-lg // Large size variant .sui-password-error // Error state .sui-password-success // Success state

Best Practices

// Always validate password strength // Use minLength for minimum security // Provide clear feedback to users // Consider adding strength indicator // Allow users to see password (toggleMask)

Chip Component

Chip component represents entities using icons, labels and images with optional remove functionality. Supports different colors, sizes, and interactive features.

Basic Usage

Simple chip with label

API Reference

Properties (Inputs)

Name Type Default Description
label string '' Text label for the chip
icon string '' Icon character or emoji to display
image string '' URL of the image to display (takes precedence over icon)
removable boolean false Whether to show a remove button
disabled boolean false Whether the chip is disabled
style any {} Inline styles to apply to the chip
styleClass string '' Additional CSS class(es) to apply

Events (Outputs)

Name Parameters Description
onRemove void Emitted when the remove button is clicked

CSS Classes

Class Description
sui-chip Base chip class
sui-chip-primary Primary color variant
sui-chip-success Success color variant
sui-chip-warning Warning color variant
sui-chip-danger Danger color variant
sui-chip-small Small size variant
sui-chip-large Large size variant
sui-chip-disabled Disabled state

Data View Component

Data View displays data in list or grid layouts with customizable templates for each item.

List Layout

Display data in a vertical list format

{{ product.image }}

{{ product.name }}

{{ product.category }}

⭐ {{ product.rating }}
\${{ product.price }} {{ product.inStock ? '✓ In Stock' : '✗ Out of Stock' }} 
<sui-data-view [value]="products" layout="list" [itemTemplate]="listItem"></sui-data-view>

<ng-template #listItem let-item>
  <div>&#123;&#123; item.name &#125;&#125;</div>
</ng-template>

Grid Layout

Display data in a responsive grid format

{{ product.image }}

{{ product.name }}

{{ product.category }}

\${{ product.price }} ⭐ {{ product.rating }}
{{ product.inStock ? 'In Stock' : 'Out of Stock' }} 
<sui-data-view [value]="products" layout="grid"></sui-data-view>

Switchable Layout

Toggle between list and grid views

API Reference

Properties (Inputs)

Name Type Default Description
value any[] [] Array of data items to display
layout 'list' | 'grid' 'list' Display layout mode
itemTemplate TemplateRef null Template for rendering each item
paginator boolean false Enable pagination
rows number 0 Number of rows per page
emptyMessage string 'No records found' Message for empty data

Events (Outputs)

Name Parameters Description
onPage { first: number, rows: number } Emitted when page changes
onLazyLoad any Emitted for lazy loading

Template Variables

  • Item Template: Access current item via let-item
  • Index: Access item index via let-index="index"
  • Grid Layout: Automatically responsive (1-4 columns based on screen size)
  • List Layout: Full-width items stacked vertically

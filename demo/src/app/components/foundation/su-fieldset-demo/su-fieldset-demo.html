Fieldset Component

A fieldset component that groups related form elements with a legend and optional collapsible functionality. Perfect for organizing complex forms into logical sections.

Basic Usage

<sui-fieldset legend="Basic Information">
  <label>Name:</label>
  <input type="text" placeholder="Enter your name">
  
  <label>Email:</label>
  <input type="email" placeholder="Enter your email">
</sui-fieldset>

Toggleable Fieldset

<sui-fieldset 
  legend="Advanced Settings" 
  [toggleable]="true" 
  [collapsed]="isCollapsed"
  (onToggle)="onFieldsetToggle($event)">
  <!-- Content -->
</sui-fieldset>

API Reference

Property Type Default Description
legend string '' Text displayed in the fieldset legend
toggleable boolean false Whether the fieldset can be collapsed/expanded
collapsed boolean false Initial collapsed state of the fieldset
style any {} Inline styles for the fieldset
styleClass string '' CSS class names for the fieldset

Events

Event Type Description
onToggle EventEmitter<any> Fired when the fieldset is toggled (if toggleable)

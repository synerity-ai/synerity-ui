Input Group Component

Input Group combines inputs with text addons or buttons. Perfect for URLs, prices, usernames, and search fields.

Basic Usage

Input with text addon

@ 
<sui-input-group>
  <span class="sui-input-group-addon">@</span>
  <input type="text" placeholder="username" />
</sui-input-group>

Usage Notes

  • Addons: Use <span class="sui-input-group-addon"> for text addons
  • Buttons: Can include <button> elements
  • Content: Use ng-content to project input elements

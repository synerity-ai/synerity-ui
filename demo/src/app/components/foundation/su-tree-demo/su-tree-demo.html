Tree Component

Tree displays hierarchical data in an expandable/collapsible structure with support for selection and filtering.

Basic Usage

Simple hierarchical tree structure

<sui-tree [value]="files"></sui-tree>

With Filtering

Filter tree nodes by label

<sui-tree 
  [value]="files" 
  [filter]="true"
  filterPlaceholder="Search files..."
></sui-tree>

Single Selection

Select a single node

Selected: {{ selectedNodes[0]?.label }} 

<sui-tree 
  [value]="organization" 
  selectionMode="single"
  (onSelectionChange)="handleSelectionChange($event)"
></sui-tree>

Multiple Selection with Checkboxes

Select multiple nodes using checkboxes

Selected {{ selectedNodes.length }} node(s) 

<sui-tree 
  [value]="files" 
  selectionMode="checkbox"
></sui-tree>

Organization Chart

Hierarchical organization structure

API Reference

Properties (Inputs)

Name Type Default Description
value TreeNode[] [] Array of tree nodes
selectionMode 'single' | 'multiple' | 'checkbox' | null null Node selection mode
filter boolean false Enable filtering
filterPlaceholder string 'Search' Placeholder for filter input
filterBy string 'label' Property to filter by

Events (Outputs)

Name Parameters Description
onNodeSelect TreeNode Emitted when node is selected
onNodeUnselect TreeNode Emitted when node is unselected
onNodeExpand TreeNode Emitted when node is expanded
onNodeCollapse TreeNode Emitted when node is collapsed
onSelectionChange TreeNode[] Emitted when selection changes

TreeNode Interface

  • key: Unique identifier for the node
  • label: Display text for the node
  • data: Optional data associated with node
  • children: Optional array of child nodes
  • expanded: Optional expanded state
  • selectable: Optional selectable flag

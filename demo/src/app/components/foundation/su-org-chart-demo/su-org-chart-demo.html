Organization Chart Component

Organization Chart displays hierarchical data structures with nodes, connections, and interactive features. Perfect for company org charts, team structures, and decision trees.

Basic Usage

Simple team structure with role information

Selected: {{ selectedNode.label }} ({{ selectedNode.data?.role }}) 

<sui-org-chart 
  [value]="teamStructure"
  (onNodeSelect)="handleNodeSelect($event)"
></sui-org-chart>

Company Organization

Complex company hierarchy with multiple levels

<sui-org-chart 
  [value]="companyOrg"
  [collapsible]="true"
  (onNodeExpand)="handleNodeExpand($event)"
></sui-org-chart>

API Reference

Properties (Inputs)

Name Type Default Description
value OrgNode[] [] Array of organization nodes
selectionMode 'single' | 'multiple' | null null Node selection behavior
collapsible boolean true Allow nodes to be collapsed
orientation 'vertical' | 'horizontal' 'vertical' Chart layout direction

Events (Outputs)

Name Parameters Description
onNodeSelect OrgNode Emitted when node is selected
onNodeUnselect OrgNode Emitted when node is unselected
onNodeExpand OrgNode Emitted when node is expanded
onNodeCollapse OrgNode Emitted when node is collapsed

Node Data Structure

  • key: Unique identifier for the node
  • label: Display name/title
  • data: Additional node information (role, department, etc.)
  • children: Array of child nodes (optional)
  • expanded: Initial expanded state (optional)
  • type: Node type for custom styling (optional)

Features

  • Hierarchical Display: Multi-level organization structures
  • Interactive Selection: Single or multiple node selection
  • Expand/Collapse: Collapsible node groups
  • Responsive Design: Adapts to different screen sizes
  • Custom Styling: Support for node types and themes

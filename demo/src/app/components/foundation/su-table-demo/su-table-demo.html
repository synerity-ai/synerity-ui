Table Component

Table displays data in tabular format with support for sorting, filtering, selection, and pagination.

Basic Usage

Simple data table with columns

<sui-table [value]="products" [columns]="columns"></sui-table>

Sortable Table

Click column headers to sort

<sui-table 
  [value]="products" 
  [columns]="columns"
  (onSort)="handleSort($event)"
></sui-table>

columns = [
  { field: 'name', header: 'Product Name', sortable: true },
  { field: 'price', header: 'Price', sortable: true }
];

Filterable Table

Filter columns using input fields

Single Selection

Select a single row

Selected: {{ selectedProducts[0]?.name }} 

<sui-table 
  [value]="products" 
  [columns]="columns"
  selectionMode="single"
  (onSelectionChange)="handleSelectionChange($event)"
></sui-table>

Multiple Selection

Select multiple rows with checkboxes

Selected {{ selectedProducts.length }} product(s) 

<sui-table 
  [value]="products" 
  [columns]="columns"
  selectionMode="multiple"
></sui-table>

API Reference

Properties (Inputs)

Name Type Default Description
value any[] [] Array of data to display
columns Column[] [] Array of column definitions
selectionMode 'single' | 'multiple' | null null Row selection mode
paginator boolean false Enable pagination
rows number 10 Number of rows per page
emptyMessage string 'No records found' Message for empty table

Events (Outputs)

Name Parameters Description
onSort { field: string, order: 'asc' | 'desc' } Emitted when column is sorted
onFilter { filters: object } Emitted when filter changes
onRowSelect any Emitted when row is selected
onSelectionChange any[] Emitted when selection changes

Column Definition

  • field: Property name in data object
  • header: Display label for column header
  • sortable: Enable sorting for column (optional)
  • filterable: Enable filtering for column (optional)

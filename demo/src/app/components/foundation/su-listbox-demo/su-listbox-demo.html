Listbox Component

Listbox provides a scrollable list of selectable options with keyboard navigation support.

Basic Usage

Simple listbox selection

Selected: {{ selectedCity || '(none)' }}

<sui-listbox [options]="cities" [(ngModel)]="selectedCity"></sui-listbox>

API Reference

Properties (Inputs)

Name Type Default Description
options Array<{ label, value, disabled? }> [] List of selectable options
disabled boolean false Disable entire listbox

Events (Outputs)

Name Parameters Description
change unknown Emitted when selection changes

Keyboard Navigation

  • Arrow Up/Down: Navigate through options
  • Enter: Select highlighted option
  • Focus: Click or tab to focus listbox

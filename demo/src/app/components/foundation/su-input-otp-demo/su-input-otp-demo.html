Input OTP Component

Input OTP provides a one-time password input with individual digit boxes. Supports different lengths, numeric/text modes, and auto-focus navigation.

Basic Usage

6-digit OTP input (default)

Value: {{ otp6 || '(empty)' }}

<sui-input-otp [(ngModel)]="otp6" (complete)="handleComplete($event)"></sui-input-otp>

Different Lengths

Customize the number of digits

<sui-input-otp [length]="4" [(ngModel)]="otp4"></sui-input-otp>
<sui-input-otp [length]="6" [(ngModel)]="otp6"></sui-input-otp>
<sui-input-otp [length]="8" [(ngModel)]="otp8"></sui-input-otp>

Input Types

Numeric only vs alphanumeric

<sui-input-otp [numericOnly]="true"></sui-input-otp>
<sui-input-otp [numericOnly]="false"></sui-input-otp>

API Reference

Properties (Inputs)

Name Type Default Description
length number 6 Number of OTP digits
numericOnly boolean true Allow only numeric input
inputType 'text' | 'tel' 'tel' Input type attribute
disabled boolean false Disabled state
name string '' Name attribute

Events (Outputs)

Name Parameters Description
change string Emitted when any digit changes
complete string Emitted when all digits are filled

Features

  • Auto-Navigation: Automatically moves to next digit after input
  • Backspace: Deletes current digit and moves to previous
  • Paste Support: Paste entire OTP code at once
  • Keyboard: Full keyboard navigation support

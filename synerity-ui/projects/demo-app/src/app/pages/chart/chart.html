How to Use Chart

Installation

Import the Chart component from the UI library:

import { Chart } from 'ui-lib';

// In your component imports
imports: [Chart]

Basic Usage

Use the component with chart data and options:

<sui-chart 
  type="bar"
  [data]="chartData"
  [options]="chartOptions"
  width="600"
  height="300"
></sui-chart>

Component Properties

  • type: Chart type ('line', 'bar', 'pie', 'doughnut', 'polarArea', 'radar')
  • data: Chart data object with labels and datasets
  • options: Chart configuration options
  • width: Chart width in pixels
  • height: Chart height in pixels
  • style: Custom CSS styles
  • styleClass: Custom CSS class

Chart Types

  • Line: Line charts for trends and time series data
  • Bar: Bar charts for categorical data comparison
  • Pie: Pie charts for proportional data
  • Doughnut: Doughnut charts similar to pie charts
  • Polar Area: Polar area charts for radial data
  • Radar: Radar charts for multivariate data

Data Structure

Chart data follows this structure:

chartData = {
  labels: ['January', 'February', 'March'],
  datasets: [
    {
      label: 'Sales',
      data: [12, 19, 3],
      backgroundColor: ['red', 'blue', 'green'],
      borderColor: ['red', 'blue', 'green'],
      borderWidth: 1
    }
  ]
};

Options Configuration

Chart options control appearance and behavior:

chartOptions = {
  responsive: true,
  maintainAspectRatio: false,
  plugins: {
    legend: { position: 'top' },
    title: { display: true, text: 'Chart Title' }
  },
  scales: {
    y: { beginAtZero: true }
  }
};

Interfaces

ChartType

type ChartType = 
  | 'line' 
  | 'bar' 
  | 'pie' 
  | 'doughnut' 
  | 'polarArea' 
  | 'radar';

ChartData

interface ChartData {
  labels: string[];
  datasets: {
    label: string;
    data: number[];
    backgroundColor?: string | string[];
    borderColor?: string | string[];
    borderWidth?: number;
  }[];
}

ChartOptions

interface ChartOptions {
  responsive?: boolean;
  maintainAspectRatio?: boolean;
  plugins?: any;
  scales?: any;
}

Methods

  • updateChart(): Updates the chart with new data
  • getChartType(): Returns the current chart type
  • getChartData(): Returns the current chart data
  • getChartOptions(): Returns the current chart options

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Keyboard support for chart interactions
  • Focus Management: Proper focus handling
  • Screen Reader: Descriptive labels and alt text

Styling

The component uses CSS classes that can be customized:

  • .sui-chart: Main container
  • .sui-chart-canvas: Chart canvas styling
  • .sui-chart-title: Chart title styling
  • .sui-chart-legend: Legend styling
  • .sui-chart-tooltip: Tooltip styling

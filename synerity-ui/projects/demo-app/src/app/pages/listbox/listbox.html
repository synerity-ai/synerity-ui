How to Use ListBox

Installation

Import the ListBox component from the UI library:

import { Listbox } from 'ui-lib';

// In your component imports
imports: [Listbox]

Basic Usage

Use the component with a simple array of options:

<sui-listbox
  [(ngModel)]="selectedValue"
  [options]="options"
  (change)="onSelectionChange($event)"
></sui-listbox>

Form Integration

The ListBox component supports two-way data binding with Angular forms:

<form [formGroup]="myForm">
  <sui-listbox
    formControlName="selectedOption"
    [options]="options"
  ></sui-listbox>
</form>

Documentation

Component Overview

The ListBox component provides a list of selectable options with keyboard navigation support. It's ideal for forms where users need to select from a predefined list of values.

Props

Property Type Default Description
options Array<{label: string, value: any, disabled?: boolean}> [] Array of selectable options
disabled boolean false Disables the entire component
ngModel any null Selected value (two-way binding)

Events

Event Type Description
change EventEmitter<any> Fired when selection changes

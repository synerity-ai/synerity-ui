How to Use ListBox

Installation

Import the ListBox component from the UI library:

import { Listbox } from 'ui-lib';

// In your component imports
imports: [Listbox]

Basic Usage

Use the component with a simple array of options:

<sui-listbox
  [(ngModel)]="selectedValue"
  [options]="options"
  (change)="onSelectionChange($event)"
></sui-listbox>

Component Properties

  • options: Array of option objects with label and value
  • disabled: Boolean to disable the entire component
  • ngModel: Two-way binding for the selected value

Events

  • change: Emitted when selection changes

Keyboard Navigation

  • Arrow Up/Down: Navigate through options
  • Enter: Select the active option

API Documentation

Properties

Property Type Default Description
options Array<{label: string, value: any, disabled?: boolean}> [] Array of selectable options
disabled boolean false Disables the entire component
ngModel any null Selected value (two-way binding)

Events

Event Type Description
change EventEmitter<any> Fired when selection changes

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Full keyboard support with arrow keys and Enter
  • Focus Management: Proper focus handling and visual indicators

Styling

The component uses CSS classes that can be customized:

  • .listbox-container: Main container
  • .listbox-option: Individual option styling
  • .listbox-option-selected: Selected option styling
  • .listbox-option-disabled: Disabled option styling

SplitButton Component

Button with a dropdown menu that provides additional actions while maintaining a primary action.

Basic SplitButton

Click the main button for the primary action, or the dropdown arrow for additional options.

Last action: {{ lastAction }} 
<sui-split-button 
  label="Save"
  icon="💾"
  [items]="basicItems"
  buttonStyle="primary"
  (onClick)="onSplitButtonClick()"
  (itemClick)="onSplitButtonItemClick($event)"
></sui-split-button>

SplitButton Variants

SplitButton supports different button styles: primary, secondary, success, info, warning, help, and danger. 

<sui-split-button 
  label="Success"
  [items]="variantItems"
  buttonStyle="success"
  (onClick)="onSplitButtonClick()"
  (itemClick)="onSplitButtonItemClick($event)"
></sui-split-button>

SplitButton States

SplitButton supports disabled state and individual menu items can also be disabled or act as separators. 

<sui-split-button 
  label="Normal"
  [items]="stateItems"
  buttonStyle="primary"
  [disabled]="false"
  (onClick)="onSplitButtonClick()"
  (itemClick)="onSplitButtonItemClick($event)"
></sui-split-button>

Installation

npm install @synerity/ui

Basic Usage

The SplitButton component provides a main button with a dropdown menu for additional actions. 

<sui-split-button 
  label="Save"
  icon="💾"
  [items]="menuItems"
  buttonStyle="primary"
  (onClick)="onMainClick()"
  (itemClick)="onItemClick($event)"
></sui-split-button>

Menu Items Configuration

Each menu item can have a label, icon, command function, disabled state, or act as a separator: 

menuItems = [
  { 
    label: 'Save', 
    icon: '💾', 
    command: () => this.save() 
  },
  { 
    label: 'Save As...', 
    icon: '📁', 
    command: () => this.saveAs() 
  },
  { separator: true },
  { 
    label: 'Export', 
    icon: '📤', 
    command: () => this.export() 
  },
  { 
    label: 'Disabled Action', 
    icon: '🚫', 
    disabled: true 
  }
];

Button Styles

SplitButton supports different visual styles:

  • primary: Default primary button style
  • secondary: Secondary button style
  • success: Success/green button style
  • info: Info/blue button style
  • warning: Warning/yellow button style
  • help: Help/purple button style
  • danger: Danger/red button style

Props

Property Type Default Description
label string '' Label for the main button
icon string '' Icon for the main button
items Array<SplitButtonItem> [] Array of menu items
buttonStyle 'primary' | 'secondary' | 'success' | 'info' | 'warning' | 'help' | 'danger' 'primary' Visual style of the button
disabled boolean false Whether the SplitButton is disabled

Events

Event Type Description
onClick void Emitted when the main button is clicked
itemClick SplitButtonItem Emitted when a menu item is clicked

SplitButtonItem Interface

Property Type Description
label string Label for the menu item
icon string Optional icon for the menu item
command () => void Optional function to execute when clicked
disabled boolean Whether the item is disabled
separator boolean Whether the item acts as a separator

Accessibility

  • Keyboard navigation support with Tab, Enter, and Escape keys
  • ARIA attributes for screen readers
  • Focus management when opening/closing dropdown
  • Proper button semantics for all interactive elements
  • Disabled state support for individual menu items

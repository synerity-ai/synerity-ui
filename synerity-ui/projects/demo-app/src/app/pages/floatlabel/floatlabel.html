Float Label

Floating label component that provides animated labels for form inputs with focus and value states.

Basic Usage

The Float Label component provides animated floating labels for form inputs.

1. Import the Component

import { FloatLabel as SuiFloatLabel } from '@synerity/ui';

2. Add to Component Imports

@Component({
  imports: [CommonModule, FormsModule, SuiFloatLabel],
  // ...
})

3. Use in Template

<sui-float-label
  label="Enter your name"
  for="my-input"
  [required]="true">
  <input 
    type="text" 
    id="my-input"
    [(ngModel)]="inputValue"
    class="w-full px-3 py-2 border border-gray-300 rounded-md">
</sui-float-label>

API Reference

Properties

Property Type Default Description
label string '' The label text to display
for string '' The ID of the associated input element
required boolean false Whether the field is required
style any {} Inline styles for the component

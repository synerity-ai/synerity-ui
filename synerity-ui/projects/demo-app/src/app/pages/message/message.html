Message

A component to display inline messages with different severity levels and optional close functionality.

Basic Message

A simple message component with default styling and content.

<sui-message 
  severity="info" 
  text="This is a basic message component.">
</sui-message>

Info Message

An informational message with blue styling to convey neutral information.

<sui-message 
  severity="info" 
  text="This is an informational message.">
</sui-message>

Success Message

A success message with green styling to indicate successful operations.

<sui-message 
  severity="success" 
  text="Operation completed successfully!">
</sui-message>

Warning Message

A warning message with orange styling to alert users about potential issues.

<sui-message 
  severity="warn" 
  text="Please review your input before proceeding.">
</sui-message>

Error Message

An error message with red styling to indicate critical issues or failures.

<sui-message 
  severity="error" 
  text="An error occurred while processing your request.">
</sui-message>

Closable Message

A message with a close button that allows users to dismiss it.

<sui-message 
  severity="info" 
  text="This message can be closed by clicking the X button."
  [closable]="true"
  (onClose)="onMessageClose()">
</sui-message>

Installation

The Message component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Message component in your Angular application:

import { Message } from '@synerity/ui';

Then use it in your template:

<sui-message severity="info" text="Your message text here"> </sui-message>

Severity Levels

The Message component supports different severity levels:

// Available severity levels <sui-message severity="info" text="Info message"></sui-message> <sui-message severity="success" text="Success message"></sui-message> <sui-message severity="warn" text="Warning message"></sui-message> <sui-message severity="error" text="Error message"></sui-message>

Component Properties

Prop Type Default Description
severity string 'info' Severity level of the message (info, success, warn, error)
text string null Text content of the message
closable boolean false Whether the message can be closed
style object null Inline style of the component
styleClass string null Style class of the component

Events

Event Description
onClose Emitted when the message is closed

Overview

The Message component is used to display inline messages with different severity levels. It provides a consistent way to show informational, success, warning, and error messages to users.

Messages are essential for providing feedback to users about the status of their actions and system states.

Panel

Collapsible content panel with header, body, and footer sections.

Basic Panel

This is the content area of the panel. You can put any content here including forms, text, images, and other components.

 
<sui-panel header="Basic Panel">
  <div class="p-4">
    <p>Panel content goes here</p>
  </div>
</sui-panel>

Collapsible Panel

This panel can be collapsed and expanded by clicking the header.

 
<sui-panel 
  header="Collapsible Panel"
  [toggleable]="true"
  [collapsed]="panelCollapsed"
  (onToggle)="togglePanel()">
  <div class="p-4">
    <p>Panel content goes here</p>
  </div>
</sui-panel>

Installation

The Panel component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Panel component in your Angular application:

import { Panel } from '@synerity/ui';

@Component({
  template: `
    <sui-panel header="My Panel">
      <div class="p-4">
        <p>Panel content</p>
      </div>
    </sui-panel>
  `
})
export class MyComponent {}

Panel Features

Header Section

Display title and optional toggle button

Body Section

Main content area with flexible layout

Footer Section

Optional footer for actions or additional info

Collapsible

Toggle panel visibility with smooth animations

Props

Prop Type Default Description
header string '' Panel header text
toggleable boolean false Whether the panel can be collapsed
collapsed boolean false Whether the panel is collapsed
style any {} Inline styles for the panel
styleClass string '' CSS class for the panel

Events

Event Type Description
onToggle EventEmitter<any> Emitted when panel is toggled

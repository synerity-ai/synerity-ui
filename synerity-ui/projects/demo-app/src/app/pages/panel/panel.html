Toggleable Panel

This panel can be collapsed and expanded by clicking the header. The content will be hidden when collapsed.

Feature 1

Collapsible content for better space management.

Feature 2

Smooth animations and transitions.

!

Click the header to toggle this panel's visibility.

 
<sui-panel 
  header="Toggleable Panel" 
  [toggleable]="true"
  [collapsed]="panelCollapsed"
  (onToggle)="onPanelToggle($event)">
  <div class="space-y-4">
    <p class="text-gray-600">This panel can be collapsed and expanded.</p>

Installation

The Panel component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
header string '' Header text for the panel
toggleable boolean false Whether the panel can be collapsed/expanded
collapsed boolean false Initial collapsed state

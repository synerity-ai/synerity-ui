Toast

A notification component that displays messages in an overlay with different severity levels and customizable positioning.

Basic Toast

A simple toast notification with default styling and positioning.

<sui-toast></sui-toast>

// In component
showToast(severity: string, summary: string, detail: string) {
  this.toastService.add({
    severity: severity,
    summary: summary,
    detail: detail
  });
}

Info Toast

An informational toast with blue styling to convey neutral information.

this.toastService.add({
  severity: 'info',
  summary: 'Information',
  detail: 'This is an informational toast message'
});

Success Toast

A success toast with green styling to indicate successful operations.

this.toastService.add({
  severity: 'success',
  summary: 'Success',
  detail: 'Operation completed successfully!'
});

Warning Toast

A warning toast with orange styling to alert users about potential issues.

this.toastService.add({
  severity: 'warn',
  summary: 'Warning',
  detail: 'Please review your input before proceeding.'
});

Error Toast

An error toast with red styling to indicate critical issues or failures.

this.toastService.add({
  severity: 'error',
  summary: 'Error',
  detail: 'An error occurred while processing your request.'
});

Custom Toast

A toast with custom styling, duration, and positioning options.

this.toastService.add({
  severity: 'info',
  summary: 'Custom Toast',
  detail: 'This toast has custom styling and behavior.',
  life: 5000,
  styleClass: 'custom-toast',
  position: 'top-right'
});

Installation

The Toast component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Toast component in your Angular application:

import { Toast } from '@synerity/ui';

Add the toast component to your template:

<sui-toast></sui-toast>

Inject the ToastService and use it to show toasts:

constructor(private toastService: ToastService) {} showToast() { this.toastService.add({ severity: 'info', summary: 'Message', detail: 'Toast message content' }); }

Service Methods

The ToastService provides several methods for managing toasts:

// Add a toast this.toastService.add(message); // Add multiple toasts this.toastService.addAll(messages); // Clear all toasts this.toastService.clear(); // Clear specific toast this.toastService.clear(key);

Component Properties

Prop Type Default Description
position string 'top-right' Position of the toast container
key string null Unique key for the toast container
baseZIndex number 0 Base z-index value for toast layering
style object null Inline style of the component
styleClass string null Style class of the component

Message Properties

Prop Type Default Description
severity string 'info' Severity level (info, success, warn, error)
summary string null Summary text of the message
detail string null Detail text of the message
life number 3000 Time in milliseconds to display the message
closable boolean true Whether the message can be closed

Overview

The Toast component is used to display notification messages in an overlay. It provides a non-intrusive way to show feedback to users with different severity levels and customizable positioning.

Toasts are essential for providing real-time feedback to users about the status of their actions and system events.

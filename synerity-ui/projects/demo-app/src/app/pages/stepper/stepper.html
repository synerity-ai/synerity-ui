Stepper

Step-by-step navigation component that guides users through a multi-step process with visual progress indicators.

Basic Horizontal Stepper

Step 1: Personal Information

Please provide your personal details to get started.

Step 2: Contact Information

We need your contact details to reach you.

Step 3: Review & Submit

Please review your information before submitting.

All information has been collected. Click submit to complete the process.

 
<sui-stepper 
  [steps]="basicSteps"
  [activeIndex]="activeIndex"
  orientation="horizontal"
  (onActiveIndexChange)="onActiveIndexChange($event)">
  
  <div *ngIf="activeIndex === 0" class="space-y-4">
    <h4 class="text-lg font-medium text-gray-900">Step 1: Personal Information</h4>
    <p class="text-gray-600">Please provide your personal details.</p>
    <div class="grid grid-cols-2 gap-4">
      <input type="text" placeholder="First Name" class="px-3 py-2 border border-gray-300 rounded-md">
      <input type="text" placeholder="Last Name" class="px-3 py-2 border border-gray-300 rounded-md">
    </div>
  </div>
</sui-stepper>

Vertical Stepper

Setup Account

Create your account and set up your profile.

Configure Settings

Customize your application settings and preferences.

Import Data

Import your existing data or start fresh.

Complete Setup

Review everything and complete the setup process.

 
<sui-stepper 
  [steps]="verticalSteps"
  [activeIndex]="verticalActiveIndex"
  orientation="vertical"
  (onActiveIndexChange)="onVerticalActiveIndexChange($event)">
  
  <div *ngIf="verticalActiveIndex === 0" class="space-y-4">
    <h4 class="text-lg font-medium text-gray-900">Setup Account</h4>
    <p class="text-gray-600">Create your account and set up your profile.</p>
  </div>
</sui-stepper>

Linear Stepper

Step 1: Basic Info

Complete this step to proceed to the next one.

Step 2: Advanced Settings

Configure advanced settings for your account.

Step 3: Final Review

Review all settings and complete the process.

 
<sui-stepper 
  [steps]="linearSteps"
  [activeIndex]="linearActiveIndex"
  [linear]="true"
  orientation="horizontal"
  (onActiveIndexChange)="onLinearActiveIndexChange($event)">
  
  <div *ngIf="linearActiveIndex === 0" class="space-y-4">
    <h4 class="text-lg font-medium text-gray-900">Step 1: Basic Info</h4>
    <p class="text-gray-600">Complete this step to proceed.</p>
    <button (click)="completeLinearStep(0)" class="px-4 py-2 text-sm bg-primary-600 text-white rounded-md">
      Complete Step
    </button>
  </div>
</sui-stepper>

Installation

The Stepper component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
activeIndex number 0 Index of the currently active step
orientation 'horizontal' | 'vertical' 'horizontal' Orientation of the stepper
linear boolean false Whether steps must be completed in order

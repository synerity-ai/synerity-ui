Stepper

Basic Stepper

<sui-stepper 
  [steps]="stepperItems"
  [activeIndex]="currentStep"
  (onActiveIndexChange)="goToStep($event)">

Installation

The Stepper component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Stepper component in your Angular application:

import { Stepper } from '@synerity/ui';

@Component({
  template: `
    <sui-stepper 
      [steps]="steps"
      [activeIndex]="currentStep"
      (onActiveIndexChange)="onStepChange($event)">
    </sui-stepper>
  `
})
export class MyComponent {}

Stepper Features

Step Navigation

Click on any step to navigate directly

Clear indication of current step and progress

Adapts to different screen sizes

Installation

npm install @synerity/ui-lib

Import

import { ScrollPanel } from '@synerity/ui-lib';

@Component({
  imports: [ScrollPanel],
  // ...
})

Basic Usage

<sui-scroll-panel scrollHeight="200px">
  <!-- Content goes here -->
</sui-scroll-panel>

Features

  • ✓ Customizable scroll dimensions
  • ✓ Vertical and horizontal scrolling
  • ✓ Scroll event handling
  • ✓ Programmatic scroll control
  • ✓ Custom styling support

Props

Name Type Default Description
scrollHeight string '200px' Height of the scroll panel
scrollWidth string '100%' Width of the scroll panel
showScrollbars boolean true Show or hide scrollbars
style any {} Inline styles for the scroll panel
styleClass string '' CSS class for the scroll panel

Events

Name Parameters Description
onScroll { scrollTop, scrollLeft, scrollHeight, scrollWidth, clientHeight, clientWidth } Fired when scroll position changes

Methods

Name Parameters Description
scrollToTop none Scroll to the top of the panel
scrollToBottom none Scroll to the bottom of the panel
scrollToLeft none Scroll to the left of the panel
scrollToRight none Scroll to the right of the panel

Installation

The ScrollPanel component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the ScrollPanel component in your Angular application:

import { ScrollPanel } from '@synerity/ui';

@Component({
  template: `
    <sui-scroll-panel scrollHeight="200px" scrollWidth="100%">
      <!-- Your content here -->
    </sui-scroll-panel>
  `
})
export class MyComponent {}

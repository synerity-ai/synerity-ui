ScrollPanel

Container component with custom scrollbars and scroll control methods for better content organization.

Installation

The ScrollPanel component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
scrollHeight string '200px' Height of the scroll panel
scrollWidth string '100%' Width of the scroll panel
showScrollbars boolean true Whether to show scrollbars

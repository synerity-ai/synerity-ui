Installation

The Sidebar component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Sidebar component in your Angular application:

import { Sidebar } from '@synerity/ui';

@Component({
  template: `
    <div class="p-4 bg-gray-100 rounded-lg" 
      [visible]="sidebarVisible"
      [position]="'left'"
      (onShow)="onShow($event)"
      (onHide)="onHide($event)">
      <div>Sidebar content</div>
    </div class="p-4 bg-gray-100 rounded-lg">
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Sidebar uses Tailwind CSS classes for styling. Here are the key classes:

div class="p-4 bg-gray-100 rounded-lg" Base class for the sidebar container
div class="p-4 bg-gray-100 rounded-lg"-content Class for the sidebar content area
div class="p-4 bg-gray-100 rounded-lg"-mask Class for the sidebar backdrop mask

Props

Prop Type Default Description
visible boolean false Whether the sidebar is visible
position 'left' | 'right' | 'top' | 'bottom' 'left' Position of the sidebar
style any {} Inline styles for the sidebar container
styleClass string '' Style class for the sidebar container

Events

Event Type Description
onShow EventEmitter<Event> Emitted when the sidebar is shown
onHide EventEmitter<Event> Emitted when the sidebar is hidden

Props

Prop Type Default Description
variant 'primary' | 'secondary' | 'outline' | 'ghost' | 'danger' 'primary' Visual style variant of the button
size 'sm' | 'md' | 'lg' 'md' Size of the button
disabled boolean false Whether the button is disabled
loading boolean false Whether the button shows loading state
type 'button' | 'submit' | 'reset' 'button' HTML button type
fullWidth boolean false Whether the button takes full width of its container
icon string | null null Icon to display in the button (emoji or icon name)
iconPosition 'left' | 'right' 'left' Position of the icon relative to the text

Events

Event Type Description
click MouseEvent Emitted when the button is clicked

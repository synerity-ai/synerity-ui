Button

Interactive button components with various styles, sizes, and states for user actions.

Button States

Normal Button {{ isLoading ? 'Loading...' : 'Loading Button' }} Disabled Button Full Width Button 
<!-- Normal Button -->
<sui-button 
  type="button"
  variant="primary"
  (click)="onClick()">
  Normal Button
</sui-button>

<!-- Loading Button -->
<sui-button 
  type="button"
  variant="primary"
  [loading]="isLoading"
  (click)="onLoadingClick()">
  {{ isLoading ? 'Loading...' : 'Loading Button' }}
</sui-button>

<!-- Disabled Button -->
<sui-button 
  type="button"
  variant="primary"
  [disabled]="true"
  (click)="onDisabledClick()">
  Disabled Button
</sui-button>

Installation

The Button component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Button component in your Angular application:

Styling with Tailwind CSS

Button uses Tailwind CSS classes for styling. Here are the key classes:

px-4 py-2 Horizontal and vertical padding
rounded-md Border radius
font-medium Font weight
transition-colors Smooth color transitions
hover:bg-blue-600 Hover state styling

Props

Prop Type Default Description
type string 'button' HTML button type (button, submit, reset)
variant 'primary' | 'secondary' | 'outline' | 'ghost' | 'danger' 'primary' Button style variant
size 'sm' | 'md' | 'lg' 'md' Button size
disabled boolean false Disable the button
loading boolean false Show loading state
fullWidth boolean false Make button full width

Events

Event Type Description
click EventEmitter<Event> Emitted when the button is clicked

Button

Interactive buttons with multiple variants, sizes, and states for user actions.

Basic Variants

<!-- Primary Button -->
<button class="px-4 py-2 bg-blue-600 text-white rounded-md hover:bg-blue-700 focus:outline-none focus:ring-2 focus:ring-blue-500 focus:ring-offset-2 transition-colors">
  Primary
</button>

<!-- Secondary Button -->
<button class="px-4 py-2 bg-gray-600 text-white rounded-md hover:bg-gray-700 focus:outline-none focus:ring-2 focus:ring-gray-500 focus:ring-offset-2 transition-colors">
  Secondary
</button>

<!-- Success Button -->
<button class="px-4 py-2 bg-green-600 text-white rounded-md hover:bg-green-700 focus:outline-none focus:ring-2 focus:ring-green-500 focus:ring-offset-2 transition-colors">
  Success
</button>

Installation

The Button component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Button component in your Angular application:

import {{ '{' }} Button {{ '}' }} from '@synerity/ui';

{{ '@' }}Component({{ '{' }}
  template: `
    <button class="px-4 py-2 bg-blue-600 text-white rounded-md hover:bg-blue-700">

Styling with Tailwind CSS

Buttons use Tailwind CSS classes for styling. Here are the key classes:

px-4 py-2 Horizontal and vertical padding
bg-blue-600 Background color
text-white Text color
rounded-md Border radius
hover:bg-blue-700 Hover state

Props

Prop Type Default Description
type string 'button' HTML button type attribute
disabled boolean false Whether the button is disabled
loading boolean false Whether the button is in loading state

Events

Event Type Description
onClick EventEmitter<MouseEvent> Emitted when the button is clicked

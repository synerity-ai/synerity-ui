Captcha

A security verification component with various challenge types and difficulty levels.

Basic Captcha

Basic math captcha with easy difficulty
<sui-captcha 
  [type]="'math'"
  [difficulty]="'easy'"
  [refreshButton]="true"
  (onVerify)="onVerify($event)"
  (onRefresh)="onRefresh()"
></sui-captcha>

Installation

Import the Captcha component in your module or standalone component:

import { Captcha } from 'ui-lib';

Basic Usage

Use the Captcha component to add security verification to your forms:

<sui-captcha 
  [type]="'math'"
  [difficulty]="'easy'"
  [refreshButton]="true"
  (onVerify)="onVerify($event)"
  (onRefresh)="onRefresh()"
></sui-captcha>

Styling Guidelines

  • Use appropriate difficulty levels based on your security requirements
  • Always provide a refresh button for better user experience
  • Consider accessibility when choosing challenge types
  • Test captcha functionality across different devices and browsers

Props

Property Type Default Description
type 'math' | 'text' | 'image' 'math' Type of captcha challenge
difficulty 'easy' | 'medium' | 'hard' 'easy' Difficulty level of the challenge
refreshButton boolean true Whether to show refresh button
customChallenge object - Custom challenge configuration

Events

Event Type Description
onVerify boolean Emitted when captcha is verified
onRefresh void Emitted when captcha is refreshed

Accessibility

  • Captcha challenges are designed to be accessible to users with disabilities
  • Alternative text is provided for image-based challenges
  • Keyboard navigation is supported for all interactive elements
  • Screen reader compatibility is maintained across all challenge types

Basic Usage

The Toggle Switch component provides an intuitive way to toggle between on/off states.

1. Import the Component

import { ToggleSwitch as SuiToggleSwitch } from '@synerity/ui';

2. Add to Component Imports

@Component({
  imports: [CommonModule, FormsModule, SuiToggleSwitch],
  // ...
})

3. Use in Template

<sui-toggle-switch
  [(ngModel)]="toggleValue"
  (change)="onToggleChange($event)"
  inputId="my-toggle">
</sui-toggle-switch>

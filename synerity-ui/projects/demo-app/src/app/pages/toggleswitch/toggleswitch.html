Installation

Import the ToggleSwitch component in your Angular module:

import { ToggleSwitch } from '@synerity/ui-lib';

@NgModule({
  imports: [
    ToggleSwitch
  ]
})

Basic Usage

Use the ToggleSwitch component in your template:

<sui-toggle-switch
  [(ngModel)]="toggleValue"
  (change)="onToggleChange($event)"
  inputId="toggle"
  name="toggle">
</sui-toggle-switch>

Form Integration

The ToggleSwitch implements ControlValueAccessor and works seamlessly with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-toggle-switch
    formControlName="toggle"
    inputId="toggle">
  </sui-toggle-switch>
</form>

Component Overview

The ToggleSwitch component provides an intuitive way to toggle between on/off states. It features a modern switch design with smooth animations and supports various interactive states. The component implements ControlValueAccessor for seamless integration with Angular forms.

Props

Property Type Default Description
ngModel boolean false Two-way binding for the toggle state
disabled boolean false Disables the toggle switch interaction
inputId string - Unique identifier for the toggle element
name string - Name attribute for the toggle element

Events

Event Type Description
change boolean Fired when the toggle state changes

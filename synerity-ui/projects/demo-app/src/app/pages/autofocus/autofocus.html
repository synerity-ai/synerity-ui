AutoFocus

A component that automatically focuses an element when it becomes visible or meets certain conditions.

Basic AutoFocus

A basic auto-focus that triggers immediately when the component loads.

<sui-auto-focus [delay]="basicDelay" [condition]="basicCondition">
  <input
    type="text"
    placeholder="This input will be focused automatically"
    class="px-4 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-transparent">
</sui-auto-focus>

Delay AutoFocus

Auto-focus with a delay to allow for smooth transitions.

<sui-auto-focus [delay]="1000" [condition]="true">
  <input
    type="text"
    placeholder="This input will be focused after 1 second"
    class="px-4 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-green-500 focus:border-transparent">
</sui-auto-focus>

Conditional AutoFocus

Auto-focus that only triggers when a condition is met.

<sui-auto-focus [delay]="0" [condition]="true">
  <input
    type="text"
    placeholder="This input will be focused when condition is true"
    class="px-4 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-purple-500 focus:border-transparent">
</sui-auto-focus>

Getting Started

Import the AutoFocus component and add it to your template:

import {{ '{' }} AutoFocus {{ '}' }} from 'ui-lib';

<sui-auto-focus [delay]="0" [condition]="true">
  <input type="text" placeholder="This will be focused">
</sui-auto-focus>

Basic auto-focus usage:

<sui-auto-focus
  [delay]="0"
  [condition]="true">
  <input
    type="text"
    placeholder="This input will be focused automatically"
    class="px-4 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-transparent">
</sui-auto-focus>

API Reference

Properties

Property Type Default Description
delay number 0 Delay in milliseconds before focusing
condition boolean true Whether to trigger auto-focus

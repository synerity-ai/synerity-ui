Auto Focus

Component that automatically focuses on an element when it becomes visible, perfect for improving user experience in forms and modals.

Basic Auto Focus

Click the button below to show a form. The first input will automatically receive focus.

Interactive: The first input field automatically receives focus when the form appears. 

<sui-auto-focus>
  <input 
    type="text" 
    placeholder="This input will be focused automatically"
    class="w-full px-3 py-2 border border-gray-300 rounded-md">
</sui-auto-focus>

Conditional Auto Focus

Toggle the condition and show the form to see conditional auto focus in action.

Auto Focus Condition: {{ focusCondition ? 'Enabled' : 'Disabled' }}

Modal Dialog

Conditional: Auto focus only occurs when the condition is true. 

<sui-auto-focus [condition]="focusCondition">
  <input 
    type="text" 
    placeholder="This input will be focused only if condition is true"
    class="w-full px-3 py-2 border border-gray-300 rounded-md">
</sui-auto-focus>

Delayed Auto Focus

Click the button to show a form with delayed auto focus (2 second delay).

Watch for the focus to appear after 2 seconds...

Delayed: Auto focus occurs after the specified delay, useful for animations or transitions. 

<sui-auto-focus [delay]="2000">
  <input 
    type="text" 
    placeholder="This input will be focused after 2 seconds"
    class="w-full px-3 py-2 border border-gray-300 rounded-md">
</sui-auto-focus>

Installation

The Auto Focus component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Auto Focus component in your Angular application:

import { AutoFocus } from '@synerity/ui';

@Component({
  selector: 'app-example',
  imports: [AutoFocus],
  template: `
    <sui-auto-focus>
      <input type="text" placeholder="Auto-focused input">
    </sui-auto-focus>
  `
})
export class ExampleComponent {}

Props

Property Type Default Description
delay number 0 Delay in milliseconds before focusing the element
condition boolean true Whether auto focus should be applied
style object {} Custom inline styles to apply to the wrapper element
styleClass string '' Additional CSS classes to apply to the wrapper element

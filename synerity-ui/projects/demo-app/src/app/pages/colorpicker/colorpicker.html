ColorPicker

A color picker component for selecting colors with various input formats and presets.

Basic ColorPicker

Selected Color: {{ basicColor }}

  [(ngModel)]="basicColor"
  (onChange)="onColorChange($event)"
  size="medium">
</sui-color-picker>

Installation

Import the ColorPicker component in your module or standalone component:

import { ColorPicker } from 'ui-lib'; // In your component imports imports: [ColorPicker, FormsModule]

Basic Usage

Use the ColorPicker component with ngModel for two-way data binding:

<sui-color-picker
  [(ngModel)]="selectedColor"
  (onChange)="onColorChange($event)"
  size="medium">
</sui-color-picker>

Form Integration

Integrate with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-color-picker
    formControlName="selectedColor"
    (onChange)="onColorChange($event)"
    size="medium">
  </sui-color-picker>
</form>

ColorPicker Component

The ColorPicker component provides various ways to select colors including HTML5 color input, hex text input, and RGB/HSL number inputs with visual color previews.

Props

Prop Type Default Description
ngModel string - Two-way binding for the selected color value
size string medium Size of the color picker (small, medium, large)
disabled boolean false Disables the color picker
placeholder string - Placeholder text for text inputs

Events

Event Type Description
onChange string Emitted when the color value changes

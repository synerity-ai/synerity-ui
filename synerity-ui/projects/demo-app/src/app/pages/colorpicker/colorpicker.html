Features

Color Formats

Support for HEX, RGB, and HSL color formats

Inline Mode

Display color picker inline without popup

Form Integration

Full Angular forms support with validation

Accessibility

Keyboard navigation and screen reader support

Props

Prop Type Default Description
value string '#000000' Selected color value
format 'hex' | 'rgb' | 'hsl' 'hex' Color format for display
inline boolean false Display color picker inline
disabled boolean false Disable the color picker
style object {{ '{' }} {{ '}' }} Inline style object
styleClass string '' CSS class name

Events

Event Parameters Description
onChange string Callback to invoke when value changes
onSelect string Callback to invoke when color is selected

Accessibility

  • • Proper labeling and color contrast
  • • Keyboard navigation support
  • • Screen reader compatibility
  • • ARIA attributes for color selection
  • • Focus management and visual indicators

Basic Usage

The ColorPicker component provides various ways to select colors:

1. Synerity Color Picker

<sui-color-picker 
  [(ngModel)]="selectedColor"
  (onChange)="onColorChange($event)"
  size="medium">
</sui-color-picker>

2. Hex Color Input

<input type="text" [(ngModel)]="hexColor" placeholder="#000000">

3. RGB Color Input

<input type="number" [(ngModel)]="rgbColor.r" min="0" max="255">
<input type="number" [(ngModel)]="rgbColor.g" min="0" max="255">
<input type="number" [(ngModel)]="rgbColor.b" min="0" max="255">

Component Integration

To use ColorPicker in your Angular component:

1. Import FormsModule

Import FormsModule from @angular/forms to enable two-way data binding.

2. Define Color Properties

Define color properties in your component class for storing selected colors.

3. Handle Color Changes

Create event handlers to respond to color selection changes.

API Reference

HTML5 Color Input

Attribute Type Description
type "color" Specifies the input type as color picker
ngModel string Two-way binding for the selected color value
disabled boolean Disables the color picker

Text Input (Hex Colors)

Attribute Type Description
type "text" Text input for hex color values
placeholder string Placeholder text (e.g., "#000000")
pattern string Regex pattern for hex color validation

Number Input (RGB Colors)

Attribute Type Description
type "number" Number input for RGB values
min number Minimum value (0 for RGB)
max number Maximum value (255 for RGB)

Events

Event Type Description
change Event Fired when color value changes
input Event Fired on every input change
blur Event Fired when input loses focus

Supported Color Formats

Hex Colors

6-digit hexadecimal color codes:

  • #FF0000 - Red
  • #00FF00 - Green
  • #0000FF - Blue
  • #FFFFFF - White
  • #000000 - Black

RGB Colors

Red, Green, Blue values (0-255):

  • rgb(255, 0, 0) - Red
  • rgb(0, 255, 0) - Green
  • rgb(0, 0, 255) - Blue

HSL Colors

Hue, Saturation, Lightness values:

  • hsl(0, 100%, 50%) - Red
  • hsl(120, 100%, 50%) - Green
  • hsl(240, 100%, 50%) - Blue

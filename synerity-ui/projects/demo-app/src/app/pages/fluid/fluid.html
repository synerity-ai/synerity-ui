Installation

The Fluid component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Fluid component in your Angular application:

import { Fluid } from '@synerity/ui';

@Component({
  selector: 'app-example',
  imports: [Fluid],
  template: `
    <sui-fluid [columns]="12" [gutter]="16">
      <div>Column 1</div>
      <div>Column 2</div>
      <div>Column 3</div>
    </sui-fluid>
  `
})
export class ExampleComponent {}

Props

Property Type Default Description
columns number 12 Number of columns in the grid
gutter number 16 Gutter size in pixels between columns
responsive boolean true Whether the grid should be responsive

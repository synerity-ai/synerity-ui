Getting Started

Import the Fluid component and add it to your template:

import {{ '{' }} Fluid {{ '}' }} from 'ui-lib';

<sui-fluid [columns]="12" [gutter]="16">
  <div class="sui-fluid-column">Column 1</div>
  <div class="sui-fluid-column">Column 2</div>
</sui-fluid>

Basic fluid grid usage:

<sui-fluid
  [columns]="12"
  [gutter]="16"
  [responsive]="true">
  <div class="sui-fluid-column bg-blue-100 p-2 rounded text-center">Column 1</div>
  <div class="sui-fluid-column bg-green-100 p-2 rounded text-center">Column 2</div>
  <div class="sui-fluid-column bg-purple-100 p-2 rounded text-center">Column 3</div>
</sui-fluid>

API Reference

Properties

Property Type Default Description
columns number 12 Number of columns in the grid
gutter number 16 Gutter size in pixels
responsive boolean true Whether the grid is responsive

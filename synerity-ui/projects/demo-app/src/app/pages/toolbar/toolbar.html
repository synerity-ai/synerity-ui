Installation

The Toolbar component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Toolbar component in your Angular application:

import { Toolbar } from '@synerity/ui';

@Component({
  template: `
    <sui-toolbar [leftItems]="toolbarItems"></sui-toolbar>
  `
})
export class MyComponent {}

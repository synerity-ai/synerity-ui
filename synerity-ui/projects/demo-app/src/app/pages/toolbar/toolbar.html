Toolbar Component

Action button container that organizes toolbar items into left, center, and right sections with support for separators and disabled states.

Basic Toolbar

  [rightItems]="basicRightItems"
  (onItemClick)="onToolbarItemClick($event)"

    Document Editor
  </div>

Installation

npm install @synerity/ui

Basic Usage

The Toolbar component organizes action buttons into left, center, and right sections. 

<sui-toolbar 
  [leftItems]="leftItems"
  [rightItems]="rightItems"
  (onItemClick)="onItemClick($event)"
>
  <div class="text-center">
    Center Content
  </div>
</sui-toolbar>

Orientations

Toolbar supports two orientations:

  • horizontal: Default horizontal layout (left to right)
  • vertical: Vertical layout (top to bottom)

Use Cases

  • Text Editors: Formatting and editing tools
  • File Managers: File operations and navigation
  • Image Editors: Drawing and editing tools
  • Data Tables: Action buttons for table operations
  • Form Builders: Form element insertion tools

Props

Property Type Default Description
leftItems Array<ToolbarItem> [] Array of items for the left section
rightItems Array<ToolbarItem> [] Array of items for the right section
orientation 'horizontal' | 'vertical' 'horizontal' Orientation of the toolbar
style any {{ '{}' }} Inline styles for the toolbar
styleClass string '' CSS class for custom styling

Events

Event Type Description
onItemClick ToolbarItem Emitted when a toolbar item is clicked

ToolbarItem Interface

Property Type Description
label string Optional label for the item
icon string Optional icon for the item
command () => void Optional function to execute when clicked
disabled boolean Whether the item is disabled
separator boolean Whether the item acts as a separator

Accessibility

  • Keyboard navigation support with Tab and Enter keys
  • ARIA attributes for screen readers
  • Focus management for toolbar items
  • Proper button semantics for all interactive elements
  • Disabled state support for individual items

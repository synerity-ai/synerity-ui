Galleria

A galleria component for displaying images in a full-screen gallery with thumbnails and navigation.

Basic Galleria

A galleria with thumbnails and indicators. Click on an image to open the gallery.

<sui-galleria
  [value]="basicGalleriaItems"
  [visible]="false"
  [showThumbnails]="true"
  [showIndicators]="true"
  [circular]="true"
  (onImageChange)="onGalleriaImageChange($event)"
  (onShow)="onGalleriaShow()"
  (onHide)="onGalleriaHide()">
</sui-galleria>

Getting Started

Import the Galleria component and add it to your template:

import {{ '{' }} Galleria {{ '}' }} from 'ui-lib';

<sui-galleria [value]="galleriaItems"></sui-galleria>

Define your galleria items:

galleriaItems = [
  {{ '{' }} 
    src: 'image1.jpg', 
    alt: 'Image 1', 
    title: 'First Image',
    thumbnailSrc: 'thumb1.jpg'
  {{ '}' }},
  {{ '{' }} 
    src: 'image2.jpg', 
    alt: 'Image 2', 
    title: 'Second Image',
    thumbnailSrc: 'thumb2.jpg'
  {{ '}' }}
];

API Reference

Properties

Property Type Default Description
value GalleriaItem[] [] Array of galleria items
activeIndex number 0 Index of the active image
visible boolean false Whether the galleria is visible
showThumbnails boolean true Whether to show thumbnails
showIndicators boolean true Whether to show page indicators
circular boolean true Whether to loop infinitely
fullScreen boolean false Whether to display in full screen

Events

Event Type Description
onImageChange number Emitted when the active image changes
onShow void Emitted when the galleria is shown
onHide void Emitted when the galleria is hidden

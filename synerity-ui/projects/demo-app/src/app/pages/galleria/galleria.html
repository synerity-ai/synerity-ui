Galleria

A responsive image gallery component with thumbnail navigation and fullscreen viewing.

Basic Galleria

<sui-galleria 
  [value]="galleriaItems"
  (onChange)="onGalleriaChange($event)">
  <ng-template let-item pTemplate="item">
    <img [src]="item.src" [alt]="item.alt" class="w-full h-64 object-cover rounded-lg">
  </ng-template>
  <ng-template let-item pTemplate="thumbnail">
    <img [src]="item.thumbnailImageSrc" [alt]="item.alt" class="w-20 h-16 object-cover rounded">
  </ng-template>
</sui-galleria>

Thumbnail Galleria

Galleria with thumbnail navigation and fullscreen viewing.

<sui-galleria 
  [value]="galleriaItems"
  [showThumbnails]="true"
  [showIndicators]="true"
  (onChange)="onGalleriaChange($event)">
  <ng-template let-item pTemplate="item">
    <img [src]="item.src" [alt]="item.alt" class="w-full h-64 object-cover rounded-lg">
  </ng-template>
  <ng-template let-item pTemplate="thumbnail">
    <img [src]="item.thumbnailImageSrc" [alt]="item.alt" class="w-20 h-16 object-cover rounded">
  </ng-template>
</sui-galleria>

Installation

The Galleria component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Galleria component in your Angular application:

import { Galleria } from '@synerity/ui';

@Component({
  template: `
    <sui-galleria 
      [value]="galleriaItems"
      (onChange)="onChange($event)">
      <ng-template let-item pTemplate="item">
        <img [src]="item.src" [alt]="item.alt">
      </ng-template>
    </sui-galleria>
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Galleria uses Tailwind CSS classes for styling. Here are the key classes:

sui-galleria Base class for the galleria container
sui-galleria-item Class for individual gallery items
sui-galleria-thumbnail Class for thumbnail images

Props

Prop Type Default Description
value Array<any> [] Array of items to display in the galleria
showThumbnails boolean true Whether to show thumbnail navigation
showIndicators boolean true Whether to show indicator dots
style any {} Inline styles for the galleria container
styleClass string '' Style class for the galleria container

Events

Event Type Description
onChange EventEmitter<any> Emitted when the active image changes

Galleria

Advanced image gallery component with lightbox functionality, thumbnails, and navigation controls.

Installation

The Galleria component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Galleria component in your Angular application:

import { Galleria } from '@synerity/ui';

@Component({
  template: `
    <!-- Thumbnail Grid -->
    <div class="grid grid-cols-4 gap-4">
      <img *ngFor="let item of images; let i = index" 
           [src]="item.thumbnailSrc" 
           (click)="showGalleria(i)">
    </div>
    
    <!-- Galleria Component -->
    <sui-galleria [value]="images"></sui-galleria>
  `
})
export class MyComponent {}

Galleria Item Structure

Each galleria item can have the following properties:

src Required. Full-size image source URL
thumbnailSrc Optional. Thumbnail image source URL
alt Optional. Alternative text for accessibility
title Optional. Title displayed in caption
caption Optional. Caption text displayed below title

Configuration Options

Thumbnails

<sui-galleria [showThumbnails]="true">

Show/hide thumbnail strip at the bottom of the gallery.

Indicators

<sui-galleria [showIndicators]="true">

Show/hide dot indicators for direct image navigation.

Circular Navigation

<sui-galleria [circular]="true">

Enable continuous looping through images.

Fullscreen Mode

<sui-galleria [fullScreen]="true">

Display gallery in fullscreen mode.

Event Handling

Handle galleria events using the provided event emitters:

<sui-galleria 
  [value]="images"
  (onImageChange)="onImageChange($event)"
  (onShow)="onShow($event)"
  (onHide)="onHide($event)">
</sui-galleria>

onImageChange(activeIndex: number) {
  console.log('Current image:', activeIndex);
  console.log('Current item:', this.images[activeIndex]);
}

onShow() {
  console.log('Galleria opened');
}

onHide() {
  console.log('Galleria closed');
}

Props

Prop Type Default Description
value Array<GalleriaItem> [] Array of galleria items to display
activeIndex number 0 Index of the currently active image
visible boolean false Show/hide the galleria overlay
showThumbnails boolean true Show thumbnail strip
showIndicators boolean true Show dot indicators
circular boolean true Enable circular navigation
fullScreen boolean false Display in fullscreen mode
style any {} Inline styles for the galleria
styleClass string '' CSS class name for custom styling

Events

Event Type Description
onImageChange EventEmitter<number> Emitted when the active image changes
onShow EventEmitter<void> Emitted when the galleria is shown
onHide EventEmitter<void> Emitted when the galleria is hidden

GalleriaItem Interface

interface GalleriaItem {
  src: string;               // Full-size image source URL (required)
  thumbnailSrc?: string;     // Thumbnail image source URL
  alt?: string;              // Alternative text for accessibility
  title?: string;            // Title displayed in caption
  caption?: string;          // Caption text displayed below title
}

RadioButton

Single-choice selection components with support for radio groups, disabled states, and form integration.

Basic Usage

Selected: {{ selectedColor || 'None' }}

<!-- Color Selection -->
<label class="flex items-center">
  <sui-radio-button 
    name="color"
    value="red"
    [(ngModel)]="selectedColor"
    (change)="onRadioChange($event, 'color')">
  </sui-radio-button>
  <span class="ml-2 text-sm text-gray-700">Red</span>
</label>

Props

Prop Type Default Description
name string '' Name attribute for radio group identification
value unknown null Value of the radio button when selected
disabled boolean false Whether the radio button is disabled
inputId string '' ID attribute for the radio button element

Events

Event Type Description
change EventEmitter<unknown> Emitted when the radio button value changes

Accessibility

  • • Proper ARIA attributes for radio button groups
  • • Keyboard navigation support (arrow keys, tab)
  • • Screen reader announcements for selection changes
  • • Focus management and visual focus indicators
  • • Support for disabled state with proper accessibility
  • • Radio group semantics for single selection

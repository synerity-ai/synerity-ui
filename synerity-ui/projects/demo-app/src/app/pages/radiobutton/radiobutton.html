Radio Button

Single selection input components that allow users to choose one option from a set of mutually exclusive choices.

Features

Form Integration

Full Angular forms support with validation

Accessibility

Built with ARIA labels and keyboard navigation

Customizable

Configurable values, names, and disabled states

Group Support

Mutually exclusive selection within groups

Props

Prop Type Default Description
ngModel any null Two-way binding for radio button value
name string '' Name attribute for grouping radio buttons
value any null Value of the radio button
disabled boolean false Disable the radio button
inputId string '' ID attribute for the radio button

Events

Event Parameters Description
change any Callback to invoke when selection changes

Accessibility

  • • Proper labeling with associated text
  • • Keyboard navigation support (Arrow keys)
  • • Screen reader compatibility
  • • ARIA attributes for form integration
  • • Focus management and visual indicators

How to Use SplitButton

Installation

Import the SplitButton component from the UI library:

import { SplitButton } from 'ui-lib';

// In your component imports
imports: [SplitButton]

Basic Usage

Use the component with a label and menu items:

<sui-split-button 
  label="Save"
  [items]="menuItems"
  (onClick)="onMainClick()"
  (itemClick)="onItemClick($event)"
></sui-split-button>

Component Properties

  • disabled: Boolean to disable the button
  • label: Text label for the main button
  • icon: Icon for the main button
  • items: Array of menu items for the dropdown
  • buttonStyle: Button style variant

Events

  • onClick: Emitted when the main button is clicked
  • itemClick: Emitted when a menu item is clicked

Menu Item Structure

Menu items support various properties:

menuItems = [
  { label: 'Save', icon: 'üíæ', command: () => this.save() },
  { label: 'Save As', icon: 'üìÅ', command: () => this.saveAs() },
  { label: '', separator: true }, // Separator item
  { label: 'Export', icon: 'üì§', command: () => this.export() },
  { label: 'Disabled', icon: 'üö´', command: () => this.disabled(), disabled: true }
];

Button Styles

Available button style variants:

  • primary: Primary button style (default)
  • secondary: Secondary button style
  • success: Success button style
  • info: Info button style
  • warning: Warning button style
  • help: Help button style
  • danger: Danger button style

API Documentation

Properties

Property Type Default Description
disabled boolean false Disable the button
label string '' Button label text
icon string '' Button icon
items SplitButtonItem[] [] Menu items array
buttonStyle ButtonStyle 'primary' Button style variant

Events

Event Type Description
onClick EventEmitter<void> Fired when main button is clicked
itemClick EventEmitter<SplitButtonItem> Fired when menu item is clicked

Interfaces

SplitButtonItem

interface SplitButtonItem {
  label: string;
  icon?: string;
  command?: () => void;
  disabled?: boolean;
  separator?: boolean;
}

ButtonStyle

type ButtonStyle = 
  | 'primary' 
  | 'secondary' 
  | 'success' 
  | 'info' 
  | 'warning' 
  | 'help' 
  | 'danger';

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Full keyboard support for button and menu
  • Focus Management: Proper focus handling and visual indicators
  • Screen Reader: Descriptive labels for all actions

Styling

The component uses CSS classes that can be customized:

  • .sui-split-button: Main container
  • .sui-split-button-main: Main button styling
  • .sui-split-button-dropdown: Dropdown button styling
  • .sui-split-button-menu: Dropdown menu container
  • .sui-split-button-item: Menu item styling
  • .sui-split-button-item-disabled: Disabled item styling
  • .sui-split-button-separator: Separator styling

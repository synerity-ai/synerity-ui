Skeleton

Loading placeholder component that mimics the structure of content while it's being loaded.

Basic Skeleton

Loading State: Basic skeleton placeholders with different widths to simulate text content. 

<sui-skeleton width="100%" height="1rem"></sui-skeleton>
<sui-skeleton width="80%" height="1rem"></sui-skeleton>
<sui-skeleton width="60%" height="1rem"></sui-skeleton>

Shape Variants

Rectangle

Rounded

Circle

Flexible Shapes: Customize border radius for different UI elements like cards, avatars, and buttons. 

<!-- Rectangle -->
<sui-skeleton width="100%" height="2rem" borderRadius="4px"></sui-skeleton>

<!-- Rounded -->
<sui-skeleton width="100%" height="2rem" borderRadius="12px"></sui-skeleton>

<!-- Circle -->
<sui-skeleton width="3rem" height="3rem" borderRadius="50%"></sui-skeleton>

Animation Variants

Default Animation

No Animation

Pulse Effect

Animation Options: Choose between shimmer animation, pulse effect, or no animation based on your design needs. 

<!-- Default shimmer animation -->
<sui-skeleton width="100%" height="1rem" [animation]="true"></sui-skeleton>

<!-- No animation -->
<sui-skeleton width="100%" height="1rem" [animation]="false"></sui-skeleton>

<!-- Pulse animation -->
<sui-skeleton width="100%" height="1rem" styleClass="pulse"></sui-skeleton>

Installation

The Skeleton component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Skeleton component in your Angular application:

import { Skeleton } from '@synerity/ui';

@Component({
  template: `
    <sui-skeleton width="100%" height="1rem"></sui-skeleton>
  `
})
export class MyComponent {}

Configuration Options

Size and Shape

<sui-skeleton width="200px" height="20px" borderRadius="8px">

Control the dimensions and border radius of the skeleton.

Animation Control

<sui-skeleton [animation]="true|false">

Enable or disable the shimmer animation effect.

Custom Styling

<sui-skeleton styleClass="pulse primary">

Apply custom CSS classes for additional styling variants.

Inline Styles

<sui-skeleton [style]="{backgroundColor: '#f0f0f0'}">

Apply inline styles for precise control over appearance.

Best Practices

Match Content Structure

Design your skeleton to closely match the actual content structure:

<!-- For a user profile card -->
<div class="profile-skeleton">
  <sui-skeleton width="3rem" height="3rem" borderRadius="50%"></sui-skeleton>
  <sui-skeleton width="8rem" height="1.25rem"></sui-skeleton>
  <sui-skeleton width="6rem" height="1rem"></sui-skeleton>
</div>

Use Appropriate Timing

Show skeleton for at least 200ms to avoid flickering, and hide it when content loads:

// Show skeleton immediately
const skeletonTimer = setTimeout(() => {
  showSkeleton = false;
  showContent = true;
}, 200);

Consider Accessibility

Provide loading state announcements for screen readers:

<sui-skeleton 
  width="100%" 
  height="1rem"
  [style]="{'aria-label': 'Loading content...'}">
</sui-skeleton>

Props

Prop Type Default Description
width string '100%' Width of the skeleton element
height string '1rem' Height of the skeleton element
borderRadius string '4px' Border radius of the skeleton element
animation boolean true Enable shimmer animation effect
style any {} Inline styles for the skeleton
styleClass string '' CSS class name for custom styling

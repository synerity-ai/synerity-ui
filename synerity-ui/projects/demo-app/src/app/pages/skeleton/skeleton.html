Skeleton

A loading placeholder component that displays animated placeholders while content is being loaded.

Basic Skeleton

Basic skeleton placeholders with default settings.

Basic: Default skeleton placeholders with animated shimmer effect. 

<sui-skeleton width="100%" height="1rem"></sui-skeleton>
<sui-skeleton width="80%" height="1rem"></sui-skeleton>
<sui-skeleton width="60%" height="1rem"></sui-skeleton>

Different Shapes

Skeleton placeholders with different border radius for various shapes.

Rectangle

Rounded

Circle

Shapes: Different border radius values to create rectangles, rounded corners, and circles. 

<sui-skeleton width="60px" height="60px" borderRadius="4px"></sui-skeleton>
<sui-skeleton width="60px" height="60px" borderRadius="8px"></sui-skeleton>
<sui-skeleton width="60px" height="60px" borderRadius="50%"></sui-skeleton>

Animation Control

Skeleton placeholders with and without animation.

With Animation (Default)

Without Animation

Animation: Control the shimmer animation effect with the animation property. 

<sui-skeleton width="100%" height="1rem" [animation]="true"></sui-skeleton>
<sui-skeleton width="100%" height="1rem" [animation]="false"></sui-skeleton>

Installation

npm install ui-lib

Basic Usage

Import the Skeleton component and use it in your template:

import { Skeleton } from 'ui-lib';

// In your template
<sui-skeleton width="100%" height="1rem"></sui-skeleton>

Common Use Cases

Loading States

Show skeleton placeholders while data is being fetched from APIs

Content Placeholders

Display loading placeholders for images, text, and other content

Layout Preservation

Maintain layout structure while content is loading

User Experience

Improve perceived performance and reduce loading anxiety

Properties

Property Type Default Description
width string '100%' Width of the skeleton placeholder
height string '1rem' Height of the skeleton placeholder
borderRadius string '4px' Border radius for rounded corners
animation boolean true Enable or disable the shimmer animation
style any {} Inline styles for the component
styleClass string '' CSS class for the component

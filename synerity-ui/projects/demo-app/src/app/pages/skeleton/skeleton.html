Getting Started

Import the Skeleton component and add it to your template:

import { Skeleton } from 'ui-lib';

<sui-skeleton [width]="100%" [height]="1rem" [animation]="true">
</sui-skeleton>

Basic skeleton usage:

<sui-skeleton
  [width]="200px"
  [height]="20px"
  [borderRadius]="8px"
  [animation]="true"
  [style]="{ backgroundColor: '#f3f4f6' }">
</sui-skeleton>

API Reference

Properties

Property Type Default Description
width string '100%' Width of the skeleton placeholder
height string '1rem' Height of the skeleton placeholder
borderRadius string '4px' Border radius of the skeleton placeholder
animation boolean true Enable or disable the loading animation
style object {} Inline styles for the skeleton
styleClass string '' CSS class for the skeleton

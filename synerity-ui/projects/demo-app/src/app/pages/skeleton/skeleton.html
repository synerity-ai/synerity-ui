Skeleton

A placeholder component that shows loading states with animated placeholders.

Basic Skeleton

<sui-skeleton 
  [width]="skeletonWidth"
  [height]="skeletonHeight"
  (onClick)="onSkeletonClick($event)">
</sui-skeleton>

Custom Skeleton

Skeleton with custom dimensions and shapes.

<sui-skeleton 
  width="200px"
  height="20px"
  (onClick)="onSkeletonClick($event)">
</sui-skeleton>

Animation Skeleton

Skeleton with animation effects for better loading experience.

<sui-skeleton 
  [width]="skeletonWidth"
  [height]="skeletonHeight"
  [animation]="'wave'"
  (onClick)="onSkeletonClick($event)">
</sui-skeleton>

Installation

The Skeleton component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Skeleton component in your Angular application:

import { Skeleton } from '@synerity/ui';

@Component({
  template: `
    <sui-skeleton 
      [width]="'200px'"
      [height]="'20px'"
      (onClick)="onClick($event)">
    </sui-skeleton>
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Skeleton uses Tailwind CSS classes for styling. Here are the key classes:

sui-skeleton Base class for the skeleton container
sui-skeleton-animated Class for animated skeleton effects
sui-skeleton-wave Class for wave animation effect

Props

Prop Type Default Description
width string '100%' Width of the skeleton placeholder
height string '1rem' Height of the skeleton placeholder
animation string 'none' Animation type for the skeleton (none, wave, pulse)
style any {} Inline styles for the skeleton container
styleClass string '' Style class for the skeleton container

Events

Event Type Description
onClick EventEmitter<Event> Emitted when the skeleton is clicked

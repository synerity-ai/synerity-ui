Image

Enhanced image component with preview, zoom, error handling, and accessibility features.

Basic Image

Standard: Basic image display with width/height controls and custom styling options. 

<sui-image 
  [src]="imageSrc"
  [alt]="imageAlt"
  [width]="'300px'"
  [height]="'200px'"
  (onLoad)="onImageLoad($event)"
  (onError)="onImageError($event)">
</sui-image>

<!-- With custom styling -->
<sui-image 
  [src]="imageSrc"
  [alt]="imageAlt"
  [width]="'200px'"
  [height]="'200px'"
  styleClass="rounded circle shadow">
</sui-image>

Preview Image

Interactive: Click on the image to open it in a fullscreen preview overlay. 

<sui-image 
  [src]="previewImageSrc"
  [alt]="previewImageAlt"
  [width]="'250px'"
  [height]="'150px'"
  [preview]="true"
  styleClass="rounded shadow">
</sui-image>

<!-- With hover effects -->
<sui-image 
  [src]="previewImageSrc"
  [alt]="previewImageAlt"
  [width]="'200px'"
  [height]="'150px'"
  [preview]="true"
  styleClass="rounded border hover-scale">
</sui-image>

Zoom Image

Zoom Effect: Hover over the image to see the zoom effect. Perfect for detailed images. 

<sui-image 
  [src]="zoomImageSrc"
  [alt]="zoomImageAlt"
  [width]="'300px'"
  [height]="'200px'"
  [zoom]="true"
  styleClass="rounded shadow">
</sui-image>

<!-- With brightness effect -->
<sui-image 
  [src]="zoomImageSrc"
  [alt]="zoomImageAlt"
  [width]="'250px'"
  [height]="'200px'"
  [zoom]="true"
  styleClass="rounded border hover-brightness">
</sui-image>

Error Handling

Graceful Degradation: Shows appropriate placeholders when images fail to load or have no source. 

<!-- Error handling -->
<sui-image 
  [src]="errorImageSrc"
  [alt]="errorImageAlt"
  [width]="'250px'"
  [height]="'150px'"
  (onError)="onImageError($event)">
</sui-image>

<!-- No source -->
<sui-image 
  [src]="''"
  [alt]="'No source image'"
  [width]="'250px'"
  [height]="'150px'">
</sui-image>

Installation

The Image component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Image component in your Angular application:

import { Image } from '@synerity/ui';

@Component({
  template: `
    <sui-image [src]="imageSrc" [alt]="imageAlt"></sui-image>
  `,
  data: {
    imageSrc: 'path/to/image.jpg',
    imageAlt: 'Image description'
  }
})
export class MyComponent {}

Configuration Options

Preview Mode

<sui-image [preview]="true">

Enable click-to-preview functionality with fullscreen overlay.

Zoom Effect

<sui-image [zoom]="true">

Enable hover zoom effect for detailed image viewing.

Loading Strategy

<sui-image [loading]="'lazy'">

Set loading strategy: 'lazy' (default) or 'eager'.

Custom Styling

<sui-image styleClass="rounded shadow border">

Apply custom CSS classes for styling variants.

Event Handling

Handle image events using the provided event emitters:

<sui-image 
  [src]="imageSrc"
  [alt]="imageAlt"
  (onLoad)="onImageLoad($event)"
  (onError)="onImageError($event)">
</sui-image>

onImageLoad(event: Event) {
  console.log('Image loaded successfully');
  // Handle successful image load
}

onImageError(event: Event) {
  console.log('Image failed to load');
  // Handle image load error
}

Styling Variants

rounded Add rounded corners
circle Make image circular
shadow Add drop shadow
border Add border
aspect-square Force square aspect ratio
aspect-video Force 16:9 aspect ratio
hover-scale Scale on hover
hover-brightness Brighten on hover

Props

Prop Type Default Description
src string '' Image source URL
alt string '' Alternative text for accessibility
width string '' Image width (CSS value)
height string '' Image height (CSS value)
preview boolean false Enable preview functionality
zoom boolean false Enable zoom effect on hover
loading 'lazy' | 'eager' 'lazy' Image loading strategy
style any {} Inline styles for the image
styleClass string '' CSS class name for custom styling

Events

Event Type Description
onLoad EventEmitter<Event> Emitted when the image loads successfully
onError EventEmitter<Event> Emitted when the image fails to load

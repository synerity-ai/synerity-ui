Image

A responsive image component with preview, zoom, and loading states.

Basic Image

<sui-image 
  [src]="imageSrc"
  [alt]="imageAlt"
  (onError)="onImageError($event)"
  (onLoad)="onImageLoad($event)">
</sui-image>

Preview Image

Image with preview functionality and zoom capabilities.

<sui-image 
  [src]="imageSrc"
  [alt]="imageAlt"
  [preview]="true"
  (onError)="onImageError($event)"
  (onLoad)="onImageLoad($event)">
</sui-image>

Installation

The Image component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Image component in your Angular application:

import { Image } from '@synerity/ui';

@Component({
  template: `
    <sui-image 
      [src]="imageSrc"
      [alt]="imageAlt"
      (onError)="onError($event)"
      (onLoad)="onLoad($event)">
    </sui-image>
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Image uses Tailwind CSS classes for styling. Here are the key classes:

sui-image Base class for the image container
sui-image-preview Class for the preview overlay
sui-image-loading Class for the loading state

Props

Prop Type Default Description
src string '' Source URL of the image
alt string '' Alt text for the image
preview boolean false Whether to enable preview functionality
style any {} Inline styles for the image container
styleClass string '' Style class for the image container

Events

Event Type Description
onLoad EventEmitter<Event> Emitted when the image is loaded
onError EventEmitter<Event> Emitted when the image fails to load

Image

An image component with preview, zoom, and error handling capabilities.

Basic Image

A simple image with basic properties.

<sui-image
  [src]="basicImageSrc"
  [alt]="basicImageAlt"
  [width]="'400px'"
  [height]="'300px'"
  (onLoad)="onImageLoad($event)"
  (onError)="onImageError($event)">
</sui-image>

Getting Started

Import the Image component and add it to your template:

import {{ '{' }} Image {{ '}' }} from 'ui-lib';

<sui-image [src]="imageSrc" [alt]="imageAlt"></sui-image>

Basic image usage:

<sui-image
  [src]="'path/to/image.jpg'"
  [alt]="'Image description'"
  [width]="'400px'"
  [height]="'300px'"
  [preview]="true"
  [zoom]="true"
  (onLoad)="onImageLoad($event)"
  (onError)="onImageError($event)">
</sui-image>

API Reference

Properties

Property Type Default Description
src string '' Image source URL
alt string '' Alternative text for the image
width string '' Image width
height string '' Image height
preview boolean false Whether to enable preview functionality
zoom boolean false Whether to enable zoom functionality
loading 'lazy' | 'eager' 'lazy' Image loading strategy

Events

Event Type Description
onLoad Event Emitted when the image loads successfully
onError Event Emitted when the image fails to load

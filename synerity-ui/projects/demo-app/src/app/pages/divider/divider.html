Installation

The Divider component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Divider component in your Angular application:

import { Divider } from '@synerity/ui';

@Component({
  template: `
    <p>Content above</p>
    <sui-divider></sui-divider>
    <p>Content below</p>
  `
})
export class MyComponent {}

Props

Prop Type Default Description
layout 'horizontal' | 'vertical' 'horizontal' Divider orientation
align 'left' | 'center' | 'right' 'center' Text alignment for text dividers
type 'solid' | 'dashed' | 'dotted' 'solid' Divider line style
style any {} Inline styles for the divider
styleClass string '' CSS class for the divider

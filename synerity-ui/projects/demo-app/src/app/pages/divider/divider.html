Installation

npm install @synerity/ui

Basic Usage

The Divider component provides a simple way to separate content sections with visual lines. 

<sui-divider></sui-divider>

Customization

Customize the divider appearance with different types, layouts, and alignments: 

<sui-divider 
  type="dashed"
  layout="horizontal"
  align="center"
  styleClass="my-custom-class"
></sui-divider>

Use Cases

  • Content Separation: Separate different sections of content
  • Form Layout: Divide form fields into logical groups
  • Navigation: Create visual breaks in navigation menus
  • Card Layout: Separate header, body, and footer sections
  • List Items: Create visual separation between list items

Props

Property Type Default Description
type 'solid' | 'dashed' | 'dotted' 'solid' Style of the divider line
layout 'horizontal' | 'vertical' 'horizontal' Orientation of the divider
align 'left' | 'center' | 'right' 'center' Alignment for horizontal dividers
style any {{ '{}' }} Inline styles for the divider
styleClass string '' CSS class for custom styling

Basic Usage

The Cascade Select component provides a hierarchical selection interface for nested data structures.

1. Import the Component

import { CascadeSelect as SuiCascadeSelect } from '@synerity/ui';

2. Add to Component Imports

@Component({
  imports: [CommonModule, FormsModule, SuiCascadeSelect],
  // ...
})

3. Use in Template

<sui-cascade-select
  [(ngModel)]="selectedValue"
  (change)="onCascadeSelectChange($event)"
  [options]="cascadeOptions"
  [levelsPlaceholder]="['Select Level 1', 'Select Level 2']"
  name="my-cascade-select">
</sui-cascade-select>

API Reference

Properties

Property Type Default Description
disabled boolean false Whether the cascade select is disabled
name string '' Name attribute for the input
levelsPlaceholder string[] [] Placeholder text for each level
options Array<object> [] Hierarchical options data structure

Events

Event Type Description
change EventEmitter<unknown> Emitted when the cascade select value changes

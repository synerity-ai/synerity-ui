Knob

A circular input component for selecting values with visual feedback and customizable styling.

Basic Knob

Current Value: {{ basicValue }}
<!-- Basic Knob -->
<sui-knob 
  [value]="knobValue"
  [min]="0"
  [max]="100"
  [step]="1"
  [size]="120"
  [strokeWidth]="12"
  [showValue]="true"
  (onChange)="onKnobChange($event)"
></sui-knob>

Installation

The Knob component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Key Features

Knob provides several useful features for value selection:

mouse Click and drag to change values
touch Full touch device support
visual Progress ring shows current value
customizable Size, colors, and appearance
accessibility Keyboard navigation and screen reader support

Props

Prop Type Default Description
value number 0 Current value of the knob
min number 0 Minimum value
max number 100 Maximum value
step number 1 Step size for value changes
size number 100 Size of the knob in pixels
strokeWidth number 14 Width of the progress stroke
disabled boolean false Disable the knob
showValue boolean true Show the current value

Events

Event Type Description
onChange EventEmitter<number> Emitted when the knob value changes

Features

Circular Interface

Intuitive circular input for value selection

Drag & Click

Support for both drag and click interactions

Customizable Range

Configurable min, max, and step values

Visual Feedback

Progress indication and value display

Props

Prop Type Default Description
value number 0 Current value of the knob
min number 0 Minimum value
max number 100 Maximum value
step number 1 Step increment for value changes
size number 100 Size of the knob in pixels
strokeWidth number 14 Width of the progress stroke
disabled boolean false Disable the knob component
readonly boolean false Make the knob readonly
showValue boolean true Show the current value in the center
style any {{ '{' }}{{ '}' }} Inline styles for the knob
styleClass string '' CSS class for the knob

Events

Event Parameters Description
onChange number Callback when knob value changes

Knob

How to Use Knob

Installation

Import the Knob component from the UI library:

import { Knob } from 'ui-lib';

// In your component imports
imports: [Knob]

Basic Usage

Use the component with basic configuration:

<sui-knob 
  [value]="knobValue"
  [min]="0"
  [max]="100"
  (onChange)="onKnobChange($event)"
></sui-knob>

Component Properties

  • value: Current value of the knob
  • min: Minimum value (default: 0)
  • max: Maximum value (default: 100)
  • step: Step size for value changes (default: 1)
  • size: Size of the knob in pixels (default: 100)
  • strokeWidth: Width of the progress stroke (default: 14)
  • disabled: Boolean to disable the knob
  • readonly: Boolean to make knob read-only
  • showValue: Boolean to show the current value (default: true)
  • style: Custom CSS styles
  • styleClass: Custom CSS class

Events

  • onChange: Emitted when the value changes

Features

  • Mouse Interaction: Click and drag to change values
  • Touch Support: Full touch device support
  • Visual Feedback: Progress ring shows current value
  • Customizable Styling: Size, colors, and appearance
  • Accessibility: Keyboard navigation and screen reader support
  • Responsive: Works on all screen sizes

Styling

Customize the knob appearance using CSS:

.custom-knob {
  --knob-color: #3b82f6;
  --knob-bg: #f3f4f6;
  --knob-text: #1f2937;
}

API Documentation

Properties

Property Type Default Description
value number 0 Current value
min number 0 Minimum value
max number 100 Maximum value
step number 1 Step size
size number 100 Size in pixels
strokeWidth number 14 Progress stroke width
disabled boolean false Disable the knob
readonly boolean false Make knob read-only
showValue boolean true Show current value
style any {} Custom styles
styleClass string '' Custom CSS class

Events

Event Type Description
onChange EventEmitter<number> Fired when value changes

Methods

  • getKnobClass(): Returns CSS classes for the knob
  • getKnobStyle(): Returns inline styles for the knob
  • getProgressStyle(): Returns styles for the progress ring
  • getDisplayValue(): Returns the formatted display value
  • isDisabled(): Checks if knob is disabled
  • isReadonly(): Checks if knob is read-only
  • shouldShowValue(): Checks if value should be displayed

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Arrow keys for value changes
  • Focus Management: Proper focus handling
  • Screen Reader: Descriptive labels and announcements

Styling

The component uses CSS classes that can be customized:

  • .sui-knob: Main container
  • .sui-knob-disabled: Disabled state styling
  • .sui-knob-readonly: Read-only state styling
  • .sui-knob-value: Value display styling
  • .sui-knob-progress: Progress ring styling

Examples

Basic Usage

export class ExampleComponent {
  knobValue = 50;
  
  onKnobChange(newValue: number) {
    this.knobValue = newValue;
  }
}

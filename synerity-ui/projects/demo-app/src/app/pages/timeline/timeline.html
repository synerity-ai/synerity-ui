Timeline

Timeline components for displaying chronological events with customizable layouts and status indicators.

Basic Timeline

<sui-timeline 
  [value]="basicTimelineData"
  align="left"
  (onEventClick)="onEventClick($event)">
</sui-timeline>

Alternate Timeline

<sui-timeline 
  [value]="alternateTimelineData"
  align="alternate"
  (onEventClick)="onEventClick($event)">
</sui-timeline>

Horizontal Timeline

<sui-timeline 
  [value]="horizontalTimelineData"
  layout="horizontal"
  (onEventClick)="onEventClick($event)">
</sui-timeline>

Custom Icons Timeline

<sui-timeline 
  [value]="customIconsTimelineData"
  align="alternate"
  (onEventClick)="onEventClick($event)">
</sui-timeline>

Installation

npm install @synerity/ui-lib

Import

import {{ '{' }} Timeline {{ '}' }} from '@synerity/ui-lib';

@Component({{ '{' }}
  imports: [Timeline],
  // ...
{{ '}' }})

Basic Usage

<sui-timeline [value]="timelineData"></sui-timeline>

Features

  • ✓ Vertical and horizontal layouts
  • ✓ Left, right, and alternate alignments
  • ✓ Custom icons and status indicators
  • ✓ Event click handling
  • ✓ Responsive design

Props

Name Type Default Description
value Array<TimelineEvent> [] Array of timeline events
align 'left' | 'right' | 'alternate' 'left' Alignment of timeline events
layout 'vertical' | 'horizontal' 'vertical' Layout orientation

Events

Name Parameters Description
onEventClick TimelineEvent Fired when a timeline event is clicked

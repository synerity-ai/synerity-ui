Timeline

Display a sequence of events in chronological order with vertical or horizontal layouts.

Vertical Timeline

<sui-timeline 
  [value]="timelineData"
  layout="vertical"
  (onEventClick)="onEventClick($event)">
</sui-timeline>

Horizontal Timeline

<sui-timeline 
  [value]="timelineData"
  layout="horizontal"
  (onEventClick)="onEventClick($event)">
</sui-timeline>

Timeline with Status

<sui-timeline 
  [value]="timelineData"
  layout="vertical"
  (onEventClick)="onEventClick($event)">
</sui-timeline>

Installation

The Timeline component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
value TimelineEvent[] [] Array of timeline events
layout 'vertical' | 'horizontal' 'vertical' Timeline layout orientation
align 'left' | 'right' | 'alternate' 'left' Content alignment for vertical layout

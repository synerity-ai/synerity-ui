Popover

Contextual popup content that appears relative to a target element with customizable positioning and behavior.

Basic Popover

Popover Content

This is a basic popover with some content.

 
<sui-popover
  [visible]="basicPopoverVisible"
  [position]="basicPopoverConfig.position"
  [dismissible]="basicPopoverConfig.dismissible"
  [showCloseIcon]="basicPopoverConfig.showCloseIcon"
  (onShow)="onPopoverShow()"
  (onHide)="onPopoverHide()">
  
  <button>Show Popover</button>
  
  <div slot="content">
    <h4>Popover Content</h4>
    <p>This is a basic popover with some content.</p>
  </div>
</sui-popover>

Popover

A contextual overlay component that displays additional content relative to a trigger element.

Basic Popover

A simple popover that appears when clicking the trigger element.

This is a bottom positioned popover with some content.

This is a top positioned popover with some content.

 
<sui-popover position="bottom">
  <button>Bottom Popover</button>
  <div slot="content">
    <p>This is a bottom positioned popover.</p>
  </div>
</sui-popover>

Installation

The Popover component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Popover component in your Angular application:

import { Popover } from '@synerity/ui';

Then use it in your template:

<sui-popover position="bottom"> <button>Trigger Button</button> <div slot="content"> <p>Popover content goes here.</p> </div> </sui-popover>

Different Positions

The popover can be positioned in different directions:

<!-- Bottom Popover --> <sui-popover position="bottom"></sui-popover> <!-- Top Popover --> <sui-popover position="top"></sui-popover> <!-- Left Popover --> <sui-popover position="left"></sui-popover> <!-- Right Popover --> <sui-popover position="right"></sui-popover>

Arrow Indicator

Popovers automatically include arrow indicators:

<sui-popover position="bottom"> <button>Trigger Button</button> <div slot="content"> <p>Popover with arrow indicator.</p> </div> </sui-popover>

Component Properties

Prop Type Default Description
position 'top' | 'bottom' | 'left' | 'right' 'bottom' Position of the popover relative to trigger
dismissible boolean true Whether the popover can be dismissed by clicking outside
trigger 'click' | 'hover' | 'focus' 'click' Trigger event for showing popover
disabled boolean false Whether the popover is disabled

Events

Event Description
onShow Emitted when the popover is shown
onHide Emitted when the popover is hidden

Overview

The Popover component provides a contextual overlay that displays additional content relative to a trigger element. It's perfect for showing supplementary information, actions, or content without navigating away from the current page.

Popovers are positioned relative to their trigger element and can be configured to appear on any side with optional arrow indicators.

FocusTrap

A component that traps focus within a specific area, preventing users from tabbing outside.

Basic FocusTrap

A basic focus trap that keeps focus within the container.

 
<sui-focus-trap [enabled]="basicEnabled" [autoFocus]="basicAutoFocus">
  <div class="flex space-x-4">
    <button class="px-4 py-2 bg-blue-500 text-white rounded hover:bg-blue-600">Button 1</button>
    <button class="px-4 py-2 bg-green-500 text-white rounded hover:bg-green-600">Button 2</button>
    <button class="px-4 py-2 bg-purple-500 text-white rounded hover:bg-purple-600">Button 3</button>
  </div>
</sui-focus-trap>

Custom FocusTrap

Custom focus trap with different configuration.

 
<sui-focus-trap [enabled]="customEnabled" [autoFocus]="customAutoFocus">
  <div class="flex space-x-4">
    <input type="text" placeholder="Input field" class="px-3 py-2 border border-gray-300 rounded-lg">
    <button class="px-4 py-2 bg-indigo-500 text-white rounded hover:bg-indigo-600">Submit</button>
    <select class="px-3 py-2 border border-gray-300 rounded-lg">
      <option>Option 1</option>
      <option>Option 2</option>
    </select>
  </div>
</sui-focus-trap>

Form FocusTrap

Focus trap applied to a form with multiple input elements.

 
<sui-focus-trap [enabled]="formEnabled" [autoFocus]="formAutoFocus">
  <form class="space-y-4">
    <div>
      <label class="block text-sm font-medium text-gray-700 mb-1">Name</label>
      <input type="text" placeholder="Enter your name" class="w-full px-3 py-2 border border-gray-300 rounded-lg">
    </div>
    <div>
      <label class="block text-sm font-medium text-gray-700 mb-1">Email</label>
      <input type="email" placeholder="Enter your email" class="w-full px-3 py-2 border border-gray-300 rounded-lg">
    </div>
    <div class="flex space-x-4">
      <button type="submit" class="px-4 py-2 bg-blue-500 text-white rounded hover:bg-blue-600">Submit</button>
      <button type="button" class="px-4 py-2 bg-gray-500 text-white rounded hover:bg-gray-600">Cancel</button>
    </div>
  </form>
</sui-focus-trap>

Getting Started

Import the FocusTrap component and add it to your template:

import {{ '{' }} FocusTrap {{ '}' }} from 'ui-lib';

<sui-focus-trap [enabled]="true" [autoFocus]="true">
  <div>Content to trap focus</div>
</sui-focus-trap>

Basic focus trap usage:

<sui-focus-trap
  [enabled]="true"
  [autoFocus]="true">
  <div class="flex space-x-4">
    <button>Button 1</button>
    <button>Button 2</button>
    <button>Button 3</button>
  </div>
</sui-focus-trap>

API Reference

Properties

Property Type Default Description
enabled boolean true Whether the focus trap is enabled
autoFocus boolean true Whether to automatically focus the first element

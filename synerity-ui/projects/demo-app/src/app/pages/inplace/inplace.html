Inplace

A component that allows inline editing of content with a simple click.

Basic Inplace

A basic inplace component for inline editing.

{{ basicValue }}
Status: {{ basicActive ? 'Active' : 'Display' }} 
<sui-inplace
  [active]="basicActive"
  (onActivate)="onBasicActivate()"
  (onDeactivate)="onBasicDeactivate()">
  <div slot="display" class="text-lg font-medium text-gray-700 cursor-pointer hover:text-blue-600 transition-colors">
    {{ basicValue }}
  </div>
  <div slot="content">
    <input
      type="text"
      [(ngModel)]="basicValue"
      class="px-3 py-2 border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-transparent">
  </div>
</sui-inplace>

Getting Started

Import the Inplace component and add it to your template:

import { Inplace } from 'ui-lib';

<sui-inplace [active]="false" (onActivate)="onActivate()" (onDeactivate)="onDeactivate()">
  <div slot="display">Display content</div>
  <div slot="content">
    <input type="text" [(ngModel)]="value">
  </div>
</sui-inplace>

Basic inplace usage:

<sui-inplace
  [active]="active"
  [disabled]="false"
  [closable]="true"
  (onActivate)="onActivate()"
  (onDeactivate)="onDeactivate()">
  <div slot="display" class="cursor-pointer hover:text-blue-600">
    {{ displayValue }}
  </div>
  <div slot="content">
    <input
      type="text"
      [(ngModel)]="displayValue"
      class="px-3 py-2 border rounded-lg">
  </div>
</sui-inplace>

API Reference

Properties

Property Type Default Description
active boolean false Whether the component is in active mode
disabled boolean false Whether the component is disabled
closable boolean true Whether the component can be closed

Events

Event Type Description
onActivate EventEmitter<void> Fired when component is activated
onDeactivate EventEmitter<void> Fired when component is deactivated

Templates

slot="display"

Content slot for the display state of the inplace component.

slot="content"

Content slot for the active state of the inplace component.

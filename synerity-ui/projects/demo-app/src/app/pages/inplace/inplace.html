Inplace

A component that allows inline editing of content, switching between display and edit modes seamlessly.

Basic Inplace

Click on the content below to activate inline editing mode.

Click to Edit

This is the display content. Click anywhere to edit it inline.

Edit Mode

You are now in edit mode. Make your changes and click outside to save.

Basic: Click to activate edit mode, click outside to deactivate. 

<sui-inplace [active]="isActive" [closable]="isClosable">
  <ng-template #display>
    <div class="p-4 bg-blue-50 border border-blue-200 rounded-lg cursor-pointer">
      <h4 class="text-lg font-semibold text-blue-900 mb-2">Click to Edit</h4>
      <p class="text-blue-700">This is the display content.</p>
    </div>
  </ng-template>
  
  <ng-template #content>
    <div class="p-4 bg-blue-100 border border-blue-300 rounded-lg">
      <h4 class="text-lg font-semibold text-blue-900 mb-2">Edit Mode</h4>
      <input type="text" value="Editable content" class="w-full px-3 py-2 border border-blue-300 rounded-md">
    </div>
  </ng-template>
</sui-inplace>

Closable Inplace

Inplace with a close button for explicit control over the edit mode.

Click to Edit

This inplace has a close button for better control.

Edit Mode

Make your changes and use the close button to save.

Closable: Includes a close button for explicit control over edit mode. 

<sui-inplace [active]="isActive" [closable]="true">
  <ng-template #display>
    <div class="p-4 bg-green-50 border border-green-200 rounded-lg cursor-pointer">
      <h4 class="text-lg font-semibold text-green-900 mb-2">Click to Edit</h4>
      <p class="text-green-700">This inplace has a close button.</p>
    </div>
  </ng-template>
  
  <ng-template #content>
    <div class="p-4 bg-green-100 border border-green-300 rounded-lg">
      <div class="flex justify-between items-start mb-3">
        <h4 class="text-lg font-semibold text-green-900">Edit Mode</h4>
        <button (click)="onDeactivate()" class="px-3 py-1 bg-green-600 text-white text-sm rounded">Close</button>
      </div>
      <input type="text" value="Editable content" class="w-full px-3 py-2 border border-green-300 rounded-md">
    </div>
  </ng-template>
</sui-inplace>

Installation

The Inplace component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Inplace component in your Angular application:

import { Inplace } from '@synerity/ui';

@Component({
  selector: 'app-example',
  imports: [Inplace],
  template: `
    <sui-inplace [active]="isActive">
      <ng-template #display>
        <div>Click to edit</div>
      </ng-template>
      <ng-template #content>
        <input type="text" value="Editable content">
      </ng-template>
    </sui-inplace>
  `
})
export class ExampleComponent {
  isActive = false;
}

Props

Property Type Default Description
active boolean false Whether the inplace is in edit mode
closable boolean true Whether the inplace can be closed
disabled boolean false Whether the inplace is disabled

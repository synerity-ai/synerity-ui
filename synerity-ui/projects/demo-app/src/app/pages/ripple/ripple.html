Ripple

A material design ripple effect component for providing visual feedback on user interactions.

Basic Ripple

Basic ripple effects on buttons with default settings.

Primary Button
Secondary Button

Basic: Default ripple effects with white color and 600ms duration. 

<sui-ripple 
  class="ripple-button"
  (click)="onRippleClick()"
>
  <button class="px-6 py-3 bg-blue-500 text-white rounded-lg">
    Click Me
  </button>
</sui-ripple>

Different Colors

Ripple effects with different colors to match various button styles.

White Ripple
Dark Ripple
Yellow Ripple

Color Variants: Different ripple colors to match button backgrounds and provide better visual contrast. 

<sui-ripple 
  color="rgba(255, 255, 255, 0.8)"
  class="ripple-button"
  (click)="onRippleClick()"
>
  <button class="px-6 py-3 bg-red-500 text-white rounded-lg">
    Red Ripple
  </button>
</sui-ripple>

Custom Duration and Cards

Ripple effects with custom durations and on card elements.

Fast Ripple

300ms duration

 Fast Animation

Slow Ripple

1200ms duration

 Slow Animation

Custom Duration: Different animation speeds for various interaction types - fast for buttons, slow for cards. 

<sui-ripple 
  [duration]="300"
  color="rgba(59, 130, 246, 0.4)"
  class="ripple-card"
  (click)="onRippleClick()"
>
  <div class="p-6 bg-white rounded-lg shadow-md cursor-pointer">
    <h3>Fast Ripple</h3>
    <p>300ms duration</p>
  </div>
</sui-ripple>

Installation

npm install ui-lib

Basic Usage

Import the Ripple component and wrap any clickable element:

import {{ '{' }} Ripple {{ '}' }} from 'ui-lib';

// In your template
<sui-ripple>
  <button class="my-button">Click Me</button>
</sui-ripple>

Common Use Cases

Buttons

Add visual feedback to button clicks with material design ripple effects

Cards

Enhance card interaction feedback for better user experience

List Items

Provide touch feedback on mobile devices and desktop interactions

Navigation Elements

Improve user interaction feedback for menu items and links

Properties

Property Type Default Description
disabled boolean false Disable the ripple effect
color string 'rgba(255, 255, 255, 0.6)' Color of the ripple effect
duration number 600 Animation duration in milliseconds
style any {} Inline styles for the component
styleClass string '' CSS class for the component

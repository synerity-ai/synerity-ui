Ripple

A Material Design ripple effect component for interactive elements.

Basic Ripple

A basic ripple effect on a button element.

Status: {{ basicDisabled ? 'Disabled' : 'Enabled' }} 
<button
  sui-ripple
  [enabled]="basicEnabled"
  [color]="basicColor"
  class="px-6 py-3 bg-blue-600 text-white rounded-lg font-medium hover:bg-blue-700 transition-colors">
  Click for Ripple Effect
</button>

Getting Started

Import the Ripple component and add it to your template:

import { Ripple } from 'ui-lib';

<button sui-ripple [enabled]="true" [color]="rgba(0, 0, 0, 0.1)">
  Click me
</button>

Basic ripple usage:

<button
  sui-ripple
  [enabled]="true"
  [color]="rgba(59, 130, 246, 0.3)"
  [duration]="600"
  class="px-4 py-2 bg-blue-600 text-white rounded-lg">
  Button with Ripple
</button>

API Reference

Properties

Property Type Default Description
enabled boolean true Controls ripple effect activation
color string 'rgba(0, 0, 0, 0.1)' Color of the ripple effect
duration number 600 Duration of the ripple animation in milliseconds
radius number 50 Maximum radius of the ripple effect

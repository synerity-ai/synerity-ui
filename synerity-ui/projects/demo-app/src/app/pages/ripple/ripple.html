How to Use Ripple

Installation

Import the Ripple component from the UI library:

import { Ripple } from 'ui-lib';

// In your component imports
imports: [Ripple]

Basic Usage

Wrap any clickable element with the ripple component:

<sui-ripple>
  <button class="my-button">Click Me</button>
</sui-ripple>

Component Properties

  • disabled: Boolean to disable the ripple effect
  • color: Color of the ripple effect (CSS color value)
  • duration: Duration of the ripple animation in milliseconds
  • style: Custom CSS styles
  • styleClass: Custom CSS class

Common Use Cases

  • Buttons: Add visual feedback to button clicks
  • Cards: Enhance card interaction feedback
  • List Items: Provide touch feedback on mobile
  • Navigation Elements: Improve user interaction feedback
  • Custom Components: Add material design effects to any element

Best Practices

  • Appropriate Colors: Choose ripple colors that contrast well with the background
  • Duration: Use shorter durations for buttons (300-600ms), longer for cards (600-1000ms)
  • Accessibility: Ensure ripple doesn't interfere with focus indicators
  • Performance: Ripple effects are automatically cleaned up after animation
  • Consistency: Use consistent ripple styling across your application

API Documentation

Properties

Property Type Default Description
disabled boolean false Disable the ripple effect
color string 'rgba(255, 255, 255, 0.6)' Color of the ripple effect
duration number 600 Animation duration in milliseconds
style any {} Custom CSS styles
styleClass string '' Custom CSS class

Events

The component forwards click events from the wrapped element.

Methods

  • getRippleClass(): Returns CSS classes for the ripple container
  • getRippleStyle(): Returns inline styles for the ripple container
  • isDisabled(): Checks if ripple is disabled

Accessibility

  • ARIA: Preserves accessibility attributes of wrapped elements
  • Keyboard Navigation: Works with keyboard interactions
  • Screen Reader: Does not interfere with screen reader announcements
  • Focus Management: Maintains focus on wrapped elements

Styling

The component uses CSS classes that can be customized:

  • .sui-ripple: Main container
  • .sui-ripple-disabled: Disabled state styling
  • .sui-ripple-effect: The animated ripple element

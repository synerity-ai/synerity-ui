How to Use TreeSelect

Installation

Import the TreeSelect component from the UI library:

import { TreeSelect } from 'ui-lib';

// In your component imports
imports: [TreeSelect]

Basic Usage

Use the component with hierarchical data:

<sui-tree-select
  [(ngModel)]="selectedValue"
  [options]="options"
  placeholder="Select an option..."
  (change)="onSelectionChange($event)"
></sui-tree-select>

Component Properties

  • options: Array of hierarchical option objects with label, value, and children
  • placeholder: Placeholder text when no option is selected
  • disabled: Boolean to disable the entire component
  • name: String for form control name
  • ngModel: Two-way binding for the selected value

Events

  • change: Emitted when selection changes

Option Structure

options = [
  {
    label: 'Category 1',
    children: [
      { label: 'Option 1', value: 'option1' },
      { label: 'Option 2', value: 'option2' }
    ]
  },
  {
    label: 'Category 2',
    children: [
      { label: 'Option 3', value: 'option3' }
    ]
  }
];

Important Notes

  • Only leaf nodes (options without children) are selectable
  • Parent nodes are used for organization and navigation only
  • The component automatically handles tree expansion and collapse

API Documentation

Properties

Property Type Default Description
options Array<{label: string, value?: any, children?: any[]}> [] Array of hierarchical options
placeholder string '' Placeholder text when no option is selected
disabled boolean false Disables the entire component
name string '' Name attribute for form control
ngModel any null Selected value (two-way binding)

Events

Event Type Description
change EventEmitter<any> Fired when selection changes

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Arrow keys for tree navigation, Enter for selection
  • Focus Management: Proper focus handling and visual indicators
  • Screen Reader: Descriptive labels for all options and hierarchy

Styling

The component uses CSS classes that can be customized:

  • .tree-select-container: Main container
  • .tree-select-dropdown: Dropdown panel
  • .tree-select-option: Individual option styling
  • .tree-select-option-selected: Selected option styling
  • .tree-select-option-disabled: Disabled option styling

Examples

Basic Usage

export class ExampleComponent {
  selectedValue = null;
  options = [
    {
      label: 'Frontend',
      children: [
        { label: 'Angular', value: 'angular' },
        { label: 'React', value: 'react' }
      ]
    }
  ];
}

          { label: 'UI/UX Designers', value: 'ui-ux' },
          { label: 'React Developers', value: 'react-dev' }
        ]
      }
    ]
  }
];

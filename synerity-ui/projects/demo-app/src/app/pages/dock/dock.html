Dock

A dock component for displaying a set of items in a dock-like interface with various positions.

Basic Dock

A basic dock positioned at the bottom with application icons.

<sui-dock 
  [model]="dockItems" 
  position="bottom"
  (onItemClick)="onDockItemClick($event)">
</sui-dock>

Different Positions

Dock can be positioned at top, bottom, left, or right.

Top Position

Left Position

Right Position

// Different positions
<sui-dock [model]="items" position="top"></sui-dock>
<sui-dock [model]="items" position="bottom"></sui-dock>
<sui-dock [model]="items" position="left"></sui-dock>
<sui-dock [model]="items" position="right"></sui-dock>

Dock with Custom Items

Dock with custom icons and labels for different applications.

// Component data
customDockItems = [
  { 
    label: 'Finder', 
    icon: 'pi pi-folder',
    command: () => this.openFinder() 
  },
  { 
    label: 'Safari', 
    icon: 'pi pi-globe',
    command: () => this.openSafari() 
  },
  { 
    label: 'Mail', 
    icon: 'pi pi-envelope',
    command: () => this.openMail() 
  }
];

<sui-dock 
  [model]="customDockItems" 
  position="bottom"
  (onItemClick)="onDockItemClick($event)">
</sui-dock>

Installation

The Dock component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Dock component in your Angular application:

import { Dock } from '@synerity/ui';

Then use it in your template:

<sui-dock [model]="dockItems" position="bottom" (onItemClick)="onDockItemClick($event)"> </sui-dock>

Data Structure

Define your dock items in your component:

// Component data dockItems = [ { label: 'Finder', icon: 'pi pi-folder', command: () => this.openFinder() }, { label: 'Safari', icon: 'pi pi-globe', command: () => this.openSafari() }, { label: 'Mail', icon: 'pi pi-envelope', command: () => this.openMail() } ];

Positions

The dock can be positioned at different sides of the screen:

// Available positions position="top" // Top of the screen position="bottom" // Bottom of the screen (default) position="left" // Left side of the screen position="right" // Right side of the screen

Component Properties

Prop Type Default Description
model MenuItem[] [] Array of menu items for the dock
position string bottom Position of the dock (top, bottom, left, right)
style object null Inline style of the component
styleClass string null Style class of the component

Events

Event Description
onItemClick Emitted when a dock item is clicked

Overview

The Dock component provides a dock-like interface similar to macOS dock, where users can access frequently used applications or functions. It can be positioned at any side of the screen and supports custom icons and labels.

The dock is perfect for creating application launchers, quick access menus, or navigation bars with a modern, sleek appearance.

Features

Increment/Decrement

Built-in +/- buttons for easy value adjustment

Validation

Min/max constraints and step validation

Form Integration

Full Angular forms support with validation

Keyboard Support

Arrow keys for increment/decrement

Props

Prop Type Default Description
ngModel number | null null Two-way binding for number value
min number | null null Minimum allowed value
max number | null null Maximum allowed value
step number 1 Increment/decrement step value
disabled boolean false Disable the input
placeholder string '' Placeholder text to display
name string '' Name attribute for form submission
inputId string '' ID attribute for the input

Events

Event Parameters Description
change number | null Callback to invoke when value changes

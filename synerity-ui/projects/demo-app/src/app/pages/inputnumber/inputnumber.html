How to Use InputNumber

Installation

Import the InputNumber component in your Angular module:

import { InputNumber } from '@synerity/ui-lib';

@NgModule({
  imports: [
    InputNumber
  ]
})
export class YourModule { }

Basic Usage

Use the InputNumber component in your template:

<sui-input-number
  [(ngModel)]="numberValue"
  (change)="onNumberChange($event)"
  placeholder="Enter a number"
  [min]="0"
  [max]="100"
  [step]="1">
</sui-input-number>

Form Integration

The InputNumber implements ControlValueAccessor and works seamlessly with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-input-number
    formControlName="quantity"
    placeholder="Enter quantity"
    [min]="1"
    [max]="100">
  </sui-input-number>
</form>

InputNumber Documentation

Component Overview

The InputNumber component provides a specialized input for numeric values with built-in validation, min/max constraints, and step controls. It ensures type safety by emitting number or null values.

Props

Property Type Default Description
disabled boolean false Disables the input
name string '' Name attribute for the input
inputId string '' ID attribute for the input
min number | null null Minimum allowed value
max number | null null Maximum allowed value
step number 1 Step value for increments
placeholder string '' Placeholder text

Events

Event Type Description
change EventEmitter<number | null> Emitted when the value changes

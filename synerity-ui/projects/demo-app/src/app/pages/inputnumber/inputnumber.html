InputNumber

Numeric input components with increment/decrement controls, validation, and form integration.

Basic Usage

<!-- Basic Number -->
<sui-input-number 
  [(ngModel)]="basicNumber"
  placeholder="Enter number"
  name="basicNumber"
  (change)="onNumberChange($event, 'basic')">
</sui-input-number>

<!-- With Placeholder -->
<sui-input-number 
  [(ngModel)]="placeholderNumber"
  placeholder="Type a number here..."
  name="placeholderNumber">
</sui-input-number>

<!-- With Step -->
<sui-input-number 
  [(ngModel)]="stepNumber"
  [step]="5"
  placeholder="Step by 5"
  name="stepNumber">
</sui-input-number>

Number States

<!-- Normal State -->
<sui-input-number 
  [(ngModel)]="basicNumber"
  placeholder="Normal number"
  name="normalNumber">
</sui-input-number>

<!-- Disabled State -->
<sui-input-number 
  [(ngModel)]="disabledNumber"
  [disabled]="true"
  placeholder="Disabled number"
  name="disabledNumber">
</sui-input-number>

<!-- With Value -->
<sui-input-number 
  [(ngModel)]="basicNumber"
  placeholder="Pre-filled number"
  name="filledNumber">
</sui-input-number>

Props

Prop Type Default Description
placeholder string '' Placeholder text when number is empty
disabled boolean false Whether the number input is disabled
min number null Minimum allowed value
max number null Maximum allowed value
step number 1 Step value for increment/decrement
name string '' Name attribute for form handling
inputId string '' ID attribute for the number input element

Events

Event Type Description
change EventEmitter<number | null> Emitted when the number value changes

Accessibility

  • • Proper ARIA attributes for number inputs
  • • Keyboard navigation support (arrow keys, tab)
  • • Screen reader announcements for value changes
  • • Focus management and visual focus indicators
  • • Support for disabled state with proper accessibility
  • • Increment/decrement buttons with proper labeling

DataView

Flexible data display component that can render data in list or grid layouts with customizable templates.

List Layout

<sui-data-view 
  [value]="data"
  layout="list"
  [itemTemplate]="listItemTemplate">
</sui-data-view>

Grid Layout

<sui-data-view 
  [value]="data"
  layout="grid"
  [itemTemplate]="gridItemTemplate">
</sui-data-view>

Installation

The DataView component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
value any[] [] Array of data objects to display
layout 'list' | 'grid' 'list' Display layout for the data

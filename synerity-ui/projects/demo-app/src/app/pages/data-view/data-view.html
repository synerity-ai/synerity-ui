How to Use DataView

Installation

Import the DataView component from the UI library:

import { DataView } from 'ui-lib';

// In your component imports
imports: [DataView]

Basic Usage

Use the component with data and custom templates:

<sui-data-view 
  [value]="products"
  layout="list"
  emptyMessage="No products found"
>
  <ng-template #itemTemplate let-product>
    <div class="product-item">
      <h3>Product Name</h3>
      <p>Product Description</p>
      <span>Product Price</span>
    </div>
  </ng-template>
</sui-data-view>

Component Properties

  • value: Array of data items to display
  • layout: Display layout ('list' or 'grid')
  • paginator: Boolean to enable pagination
  • rows: Number of items per page
  • first: Index of first item on current page
  • totalRecords: Total number of records
  • emptyMessage: Message to display when no data

Events

  • onPage: Emitted when pagination changes
  • onLazyLoad: Emitted for lazy loading scenarios

Template Usage

The component uses Angular templates for custom item rendering:

  • itemTemplate: Template for rendering each data item
  • Template Context: Each item is passed as template context
  • Flexible Layout: Templates work with both list and grid layouts

Layout Modes

  • List Layout: Vertical list of items, good for detailed views
  • Grid Layout: Grid of items, good for card-based displays

Pagination

Enable pagination for large datasets:

<sui-data-view 
  [value]="paginatedData"
  [paginator]="true"
  [rows]="10"
  [first]="0"
  [totalRecords]="100"
  (onPage)="onPage($event)"
></sui-data-view>

API Documentation

Properties

Property Type Default Description
value any[] [] Array of data items
layout 'list' | 'grid' 'list' Display layout mode
paginator boolean false Enable pagination
rows number 0 Number of items per page
first number 0 Index of first item
totalRecords number 0 Total number of records
emptyMessage string 'No records found' Empty state message

Events

Event Type Description
onPage EventEmitter<{first: number, rows: number, page: number}> Fired when pagination changes
onLazyLoad EventEmitter<{first: number, rows: number}> Fired for lazy loading

Templates

itemTemplate

Template for rendering each data item:

<ng-template #itemTemplate let-item>
  <div class="item">
    <!-- Custom item content -->
  </div>
</ng-template>

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Full keyboard support for pagination
  • Focus Management: Proper focus handling
  • Screen Reader: Descriptive labels and announcements

Styling

The component uses CSS classes that can be customized:

  • .sui-data-view: Main container
  • .sui-data-view-list: List layout container
  • .sui-data-view-grid: Grid layout container
  • .sui-data-view-item: Individual item container
  • .sui-data-view-empty: Empty state styling
  • .sui-data-view-paginator: Pagination controls

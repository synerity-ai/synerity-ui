Textarea

Multi-line text input components with support for auto-sizing, placeholders, and various configurations.

Basic Textarea

<sui-textarea 
  [(ngModel)]="basicValue"
  placeholder="Enter your message here..."
  [rows]="4">
</sui-textarea>

Placeholder Textarea

<sui-textarea 
  [(ngModel)]="placeholderValue"
  placeholder="Type your detailed description here...">
</sui-textarea>

Disabled Textarea

<sui-textarea 
  [(ngModel)]="disabledValue"
  [disabled]="true"
  [rows]="3">
</sui-textarea>

Auto-size Textarea

<sui-textarea 
  [(ngModel)]="autosizeValue"
  [autosize]="true"
  placeholder="This textarea will automatically resize as you type...">
</sui-textarea>

Interactive Demo

<sui-textarea 
  [(ngModel)]="interactiveValue"
  [placeholder]="interactivePlaceholder"
  [rows]="interactiveRows"
  [disabled]="interactiveDisabled"
  [autosize]="interactiveAutosize">
</sui-textarea>

Installation

npm install @synerity/ui

Import

import { Textarea } from '@synerity/ui';

@Component({
  imports: [Textarea],
  // ...
})

Basic Usage

<sui-textarea 
  [(ngModel)]="message"
  placeholder="Enter your message..."
  [rows]="4">
</sui-textarea>

Features

Auto-sizing

Automatically adjust height based on content

Form Integration

Full Angular forms support with validation

Customizable

Configurable rows, placeholders, and states

Accessible

Built with accessibility best practices

Props

Prop Type Default Description
ngModel string '' Two-way binding for textarea value
placeholder string '' Placeholder text to display
rows number 3 Number of visible text lines
disabled boolean false Disable the textarea
autosize boolean false Automatically resize based on content
name string '' Name attribute for form submission
inputId string '' ID attribute for the textarea

Events

Event Parameters Description
change string Callback to invoke when value changes

Accessibility

  • • Proper labeling with placeholder text
  • • Keyboard navigation support
  • • Screen reader compatibility
  • • Focus management
  • • ARIA attributes for form integration

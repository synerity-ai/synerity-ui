Textarea

Multi-line text input components with support for auto-sizing, placeholders, and various configurations.

Basic Usage

Characters: {{ basicValue.length }}

<!-- Basic Textarea -->
<sui-textarea 
  [(ngModel)]="basicValue"
  placeholder="Enter your text here..."
  name="basicTextarea">
</sui-textarea>

<!-- With Placeholder -->
<sui-textarea 
  [(ngModel)]="placeholderValue"
  placeholder="Type your message here..."
  name="placeholderTextarea">
</sui-textarea>

<!-- Custom Rows -->
<sui-textarea 
  [(ngModel)]="rowsValue"
  [rows]="5"
  placeholder="5 rows textarea..."
  name="rowsTextarea">
</sui-textarea>

Textarea Features

Auto-expands as you type

<!-- Auto-sizing Textarea -->
<sui-textarea 
  [(ngModel)]="autosizeValue"
  [autosize]="true"
  placeholder="This textarea grows with content..."
  name="autosizeTextarea">
</sui-textarea>

<!-- Large Textarea -->
<sui-textarea 
  [(ngModel)]="rowsValue"
  [rows]="8"
  placeholder="Large textarea for longer content..."
  name="largeTextarea">
</sui-textarea>

Props

Prop Type Default Description
placeholder string '' Placeholder text when textarea is empty
rows number 3 Number of visible text lines
disabled boolean false Whether the textarea is disabled
autosize boolean false Whether the textarea auto-resizes based on content
name string '' Name attribute for form handling
inputId string '' ID attribute for the textarea element

Events

Event Type Description
change EventEmitter<string> Emitted when the textarea value changes

Accessibility

  • • Proper ARIA attributes for form controls
  • • Keyboard navigation support
  • • Screen reader announcements for value changes
  • • Focus management and visual focus indicators
  • • Support for disabled state with proper accessibility

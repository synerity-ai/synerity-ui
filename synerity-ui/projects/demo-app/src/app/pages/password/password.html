Password

A password input component with strength validation, visibility toggle, and confirmation features.

<sui-password 
  [(ngModel)]="basicPassword"
  (change)="onPasswordChange($event)"

</sui-password>

Installation

Import the Password component in your Angular module:

import { Password } from '@synerity/ui-lib';

@NgModule({
  imports: [
    Password
  ]
})

Basic Usage

Use the Password component in your template:

<sui-password
  [(ngModel)]="password"
  (change)="onPasswordChange($event)"
  placeholder="Enter password"
  [toggleMask]="true"
  inputId="password-input"
  name="password">
</sui-password>

Form Integration

The Password implements ControlValueAccessor and works seamlessly with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-password
    formControlName="password"
    placeholder="Enter password"
    [toggleMask]="true">
  </sui-password>
</form>

Component Overview

The Password component provides a secure password input field with visibility toggle functionality. It implements ControlValueAccessor for seamless integration with Angular forms and provides built-in security features.

Props

Property Type Default Description
ngModel string - Two-way binding for the password value
placeholder string - Placeholder text for the input
toggleMask boolean false Enables password visibility toggle button
disabled boolean false Disables the password input
readonly boolean false Makes the password input readonly
required boolean false Makes the password input required
inputId string - Unique identifier for the input element
name string - Name attribute for the input element

Events

Event Type Description
change string Fired when password value changes
focus FocusEvent Fired when password input gains focus
blur FocusEvent Fired when password input loses focus

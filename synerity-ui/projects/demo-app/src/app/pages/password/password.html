Built-in password masking and validation

Props

Prop Type Default Description
ngModel string '' Two-way binding for password value
placeholder string '' Placeholder text to display
disabled boolean false Disable the password input
minLength number null Minimum password length
toggleMask boolean true Enable/disable visibility toggle
name string '' Name attribute for form submission
inputId string '' ID attribute for the input

Events

Event Parameters Description
change string Callback to invoke when value changes

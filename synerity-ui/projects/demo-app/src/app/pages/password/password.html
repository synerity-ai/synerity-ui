Password

Secure password input components with toggle visibility, validation, and form integration.

Basic Usage

<!-- Basic Password -->
<sui-password 
  [(ngModel)]="basicPassword"
  placeholder="Enter password"
  name="basicPassword"
  (change)="onPasswordChange($event, 'basic')">
</sui-password>

<!-- With Placeholder -->
<sui-password 
  [(ngModel)]="placeholderPassword"
  placeholder="Type your password here..."
  name="placeholderPassword">
</sui-password>

<!-- No Toggle -->
<sui-password 
  [(ngModel)]="noTogglePassword"
  [toggleMask]="false"
  placeholder="Password without toggle"
  name="noTogglePassword">
</sui-password>

Password States

<!-- Normal State -->
<sui-password 
  [(ngModel)]="basicPassword"
  placeholder="Normal password"
  name="normalPassword">
</sui-password>

<!-- Disabled State -->
<sui-password 
  [(ngModel)]="disabledPassword"
  [disabled]="true"
  placeholder="Disabled password"
  name="disabledPassword">
</sui-password>

<!-- With Value -->
<sui-password 
  [(ngModel)]="basicPassword"
  placeholder="Pre-filled password"
  name="filledPassword">
</sui-password>

Props

Prop Type Default Description
placeholder string '' Placeholder text when password is empty
disabled boolean false Whether the password input is disabled
toggleMask boolean true Whether to show password visibility toggle
minLength number null Minimum password length
name string '' Name attribute for form handling
inputId string '' ID attribute for the password input element

Events

Event Type Description
change EventEmitter<string> Emitted when the password value changes

Accessibility

  • • Proper ARIA attributes for password inputs
  • • Keyboard navigation support
  • • Screen reader announcements for value changes
  • • Focus management and visual focus indicators
  • • Support for disabled state with proper accessibility
  • • Password visibility toggle with proper labeling

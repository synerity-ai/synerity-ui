Password

A password input component with strength validation, visibility toggle, and confirmation features.

Basic Password

Length: {{ basicPassword.length }} characters 
<sui-password 
  [(ngModel)]="basicPassword"
  (change)="onPasswordChange($event)"
  placeholder="Enter password"
  inputId="basic-password"
  name="basicPassword"
  [toggleMask]="true">
</sui-password>

Password Confirmation

Passwords match
Passwords do not match 
<!-- Password Input -->
<sui-password 
  [(ngModel)]="basicPassword"
  (change)="onPasswordChange($event)"
  placeholder="Enter password"
  inputId="password-input"
  name="password"
  [toggleMask]="true">
</sui-password>

<!-- Confirm Password Input -->
<sui-password 
  [(ngModel)]="confirmPassword"
  (change)="onConfirmPasswordChange($event)"
  placeholder="Confirm password"
  inputId="confirm-password"
  name="confirmPassword"
  [toggleMask]="true">
</sui-password>

<!-- Validation Messages -->
<div *ngIf="passwordsMatch()" class="text-green-600">
  Passwords match
</div>
<div *ngIf="passwordsDoNotMatch()" class="text-red-600">
  Passwords do not match
</div>

Basic Usage

The Password component provides secure password input with various features:

2. Password Confirmation

Implement password confirmation with validation feedback.

3. Password Strength

Add strength validation with visual indicators.

4. Password Requirements

Show password requirements with real-time validation.

API Reference

Password Input Element

Attribute Type Description
type "password" | "text" Input type for password or text display
ngModel string Two-way binding for the password value
placeholder string Placeholder text for the input
disabled boolean Disables the password input

Events

Event Type Description
input Event Fired when password value changes
change Event Fired when password input loses focus
focus Event Fired when password input gains focus

ConfirmDialog

Modal dialog component for confirming user actions with customizable messages and buttons.

Basic ConfirmDialog

Last action: {{ lastAction }} 
<sui-confirm-dialog
  [visible]="basicDialogVisible"
  header="Confirm Action"
  message="Are you sure you want to proceed with this action?"
  acceptLabel="Yes, Continue"
  rejectLabel="Cancel"
  (onAccept)="onBasicAccept()"
  (onReject)="onBasicReject()"
  (onHide)="onBasicHide()">
</sui-confirm-dialog>

Installation

The ConfirmDialog component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
visible boolean false Whether the dialog is visible
header string 'Confirm' Header text of the dialog
message string 'Are you sure?' Message text of the dialog

Confirm Dialog

A modal dialog component for confirming user actions with accept and reject options.

Basic Confirm Dialog

Confirm Dialog Component - Placeholder

<div class="p-4 bg-gray-100 rounded-lg" 
  [visible]="confirmDialogVisible"
  [message]="confirmMessage"
  (onAccept)="onConfirmDialogAccept($event)"
  (onReject)="onConfirmDialogReject($event)">
</div class="p-4 bg-gray-100 rounded-lg">

Installation

The Confirm Dialog component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Confirm Dialog component in your Angular application:

import { ConfirmDialog } from '@synerity/ui';

@Component({
  template: `
    <div class="p-4 bg-gray-100 rounded-lg" 
      [visible]="dialogVisible"
      [message]="'Are you sure?'"
      (onAccept)="onAccept($event)"
      (onReject)="onReject($event)">
    </div class="p-4 bg-gray-100 rounded-lg">
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Confirm Dialog uses Tailwind CSS classes for styling. Here are the key classes:

div class="p-4 bg-gray-100 rounded-lg" Base class for the confirm dialog container
div class="p-4 bg-gray-100 rounded-lg"-content Class for the dialog content area
div class="p-4 bg-gray-100 rounded-lg"-buttons Class for the dialog buttons container

Props

Prop Type Default Description
visible boolean false Whether the confirm dialog is visible
message string '' Message to display in the dialog
acceptLabel string 'Accept' Label for the accept button
rejectLabel string 'Reject' Label for the reject button
style any {} Inline styles for the confirm dialog container
styleClass string '' Style class for the confirm dialog container

Events

Event Type Description
onAccept EventEmitter<Event> Emitted when the accept button is clicked
onReject EventEmitter<Event> Emitted when the reject button is clicked

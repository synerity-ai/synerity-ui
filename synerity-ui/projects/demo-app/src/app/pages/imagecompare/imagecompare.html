Image Compare

Interactive image comparison component that allows users to compare two images side by side with a draggable slider.

Basic Image Compare

Interactive: Drag the slider to compare the before and after images. Perfect for showing transformations, edits, or comparisons. 

<sui-image-compare 
  [beforeSrc]="beforeImageSrc"
  [afterSrc]="afterImageSrc"
  [beforeAlt]="beforeImageAlt"
  [afterAlt]="afterImageAlt">
</sui-image-compare>

Installation

The Image Compare component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Image Compare component in your Angular application:

import { ImageCompare } from '@synerity/ui';

@Component({
  selector: 'app-example',
  imports: [ImageCompare],
  template: `
    <sui-image-compare 
      [beforeSrc]="beforeImage"
      [afterSrc]="afterImage">
    </sui-image-compare>
  `
})
export class ExampleComponent {
  beforeImage = 'path/to/before.jpg';
  afterImage = 'path/to/after.jpg';
}

Props

Property Type Default Description
beforeSrc string - URL of the before image (required)
afterSrc string - URL of the after image (required)
beforeAlt string - Alt text for the before image
afterAlt string - Alt text for the after image

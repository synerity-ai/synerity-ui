ImageCompare

An image comparison component that allows users to compare two images using a draggable slider.

Basic ImageCompare

A basic image comparison with horizontal orientation.

<sui-image-compare
  [beforeSrc]="basicBeforeSrc"
  [afterSrc]="basicAfterSrc"
  [beforeAlt]="basicBeforeAlt"
  [afterAlt]="basicAfterAlt"
  [orientation]="'horizontal'"
  [sliderPosition]="50">
</sui-image-compare>

Getting Started

Import the ImageCompare component and add it to your template:

import {{ '{' }} ImageCompare {{ '}' }} from 'ui-lib';

<sui-image-compare [beforeSrc]="beforeImage" [afterSrc]="afterImage"></sui-image-compare>

Basic image comparison usage:

<sui-image-compare
  [beforeSrc]="'path/to/before.jpg'"
  [afterSrc]="'path/to/after.jpg'"
  [beforeAlt]="'Before Image'"
  [afterAlt]="'After Image'"
  [orientation]="'horizontal'"
  [sliderPosition]="50">
</sui-image-compare>

API Reference

Properties

Property Type Default Description
beforeSrc string '' Source URL of the before image
afterSrc string '' Source URL of the after image
beforeAlt string 'Before' Alternative text for the before image
afterAlt string 'After' Alternative text for the after image
orientation 'horizontal' | 'vertical' 'horizontal' Orientation of the comparison slider
sliderPosition number 50 Initial position of the slider (0-100)

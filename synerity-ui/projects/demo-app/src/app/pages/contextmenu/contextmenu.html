Installation

The Context Menu component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Context Menu component in your Angular application:

import { ContextMenu } from '@synerity/ui';

@Component({
  template: `
    <div class="p-4 bg-gray-100 rounded-lg" 
      [model]="contextMenuItems">
    </div class="p-4 bg-gray-100 rounded-lg">
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Context Menu uses Tailwind CSS classes for styling. Here are the key classes:

div class="p-4 bg-gray-100 rounded-lg" Base class for the context menu container
div class="p-4 bg-gray-100 rounded-lg"-item Class for individual menu items
div class="p-4 bg-gray-100 rounded-lg"-item-active Class for the active menu item

Props

Prop Type Default Description
model Array<MenuItem> [] Array of menu items to display
style any {} Inline styles for the context menu container
styleClass string '' Style class for the context menu container

Events

Event Type Description
onItemClick EventEmitter<MenuItem> Emitted when a menu item is clicked

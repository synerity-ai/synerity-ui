Context Menu

A contextual menu component that appears on right-click with various menu items and submenus.

Basic Context Menu

Right-click on the area below to see the context menu.

Right-click here to open context menu

<div (contextmenu)="showContextMenu($event)">
  Right-click here
</div>

<sui-contextmenu 
  #contextMenu
  [model]="contextMenuItems"
  (onItemClick)="onContextMenuClick($event)">
</sui-contextmenu>

Context Menu with Icons

Context menu with icons for better visual hierarchy.

Right-click here to open context menu with icons

// Component data
iconContextMenuItems = [
  { label: 'Copy', icon: 'pi pi-copy', command: () => this.copy() },
  { label: 'Paste', icon: 'pi pi-paste', command: () => this.paste() },
  { label: 'Delete', icon: 'pi pi-trash', command: () => this.delete() }
];

<div (contextmenu)="showIconContextMenu($event)">
  Right-click here
</div>

Context Menu with Submenus

Context menu with nested submenus for complex actions.

Right-click here to open context menu with submenus

// Component data
submenuContextMenuItems = [
  { 
    label: 'Edit', 
    icon: 'pi pi-pencil',
    items: [
      { label: 'Cut', command: () => this.cut() },
      { label: 'Copy', command: () => this.copy() },
      { label: 'Paste', command: () => this.paste() }
    ]
  },
  { 
    label: 'View', 
    icon: 'pi pi-eye',
    items: [
      { label: 'Zoom In', command: () => this.zoomIn() },
      { label: 'Zoom Out', command: () => this.zoomOut() }
    ]
  }
];

Installation

The ContextMenu component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the ContextMenu component in your Angular application:

import { ContextMenu } from '@synerity/ui';

Then use it in your template:

<div (contextmenu)="showContextMenu($event)"> Right-click target </div> <sui-contextmenu #contextMenu [model]="contextMenuItems" (onItemClick)="onContextMenuClick($event)"> </sui-contextmenu>

Data Structure

Define your context menu items in your component:

// Component data contextMenuItems = [ { label: 'Copy', command: () => this.copy() }, { label: 'Paste', command: () => this.paste() }, { label: 'Delete', command: () => this.delete() } ]; // Show context menu showContextMenu(event: MouseEvent): void { this.contextMenu.show(event); }

Component Properties

Prop Type Default Description
model MenuItem[] [] Array of menu items for the context menu
style object null Inline style of the component
styleClass string null Style class of the component

Events

Event Description
onItemClick Emitted when a context menu item is clicked
onShow Emitted when the context menu is shown
onHide Emitted when the context menu is hidden

Overview

The ContextMenu component provides a contextual menu that appears when users right-click on a target element. It's perfect for providing contextual actions, shortcuts, and additional functionality without cluttering the main interface.

Context menus support nested submenus, icons, separators, and custom styling to create rich interaction experiences.

Installation

The ContextMenu component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the ContextMenu component in your Angular application:

import { ContextMenu } from '@synerity/ui';

@Component({
  template: `
    <sui-context-menu #contextMenu [model]="menuItems"></sui-context-menu>
    <div (contextmenu)="showContextMenu($event)">Right-click me!</div>
  `
})
export class MyComponent {
  @ViewChild('contextMenu') contextMenu!: ContextMenu;
  
  menuItems = [
    { label: 'Edit', command: () => console.log('Edit') },
    { label: 'Delete', command: () => console.log('Delete') }
  ];
  
  showContextMenu(event: MouseEvent) {
    this.contextMenu.show(event);
  }
}

Menu Structure

The context menu expects an array of menu items with the following structure:

label Required. Display text for the menu item
icon Optional. Icon to display before the label
items Optional. Array of submenu items
command Optional. Function to execute when clicked
disabled Optional. Boolean to disable the menu item
separator Optional. Boolean to show a separator line

Props

Prop Type Default Description
model Array<MenuItem> [] Array of menu items to display
appendTo 'body' | 'target' | ElementRef | string 'body' DOM element to append the menu to
style any {} Inline styles for the context menu
styleClass string '' CSS class name for custom styling

Events

Event Type Description
onShow EventEmitter<void> Emitted when the context menu is shown
onHide EventEmitter<void> Emitted when the context menu is hidden

Methods

show(event: MouseEvent)

Shows the context menu at the specified mouse event position.

hide()

Hides the context menu.

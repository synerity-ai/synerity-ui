Tooltip

A lightweight overlay component that displays helpful information on hover or focus.

Basic Tooltips

Simple tooltips that appear on hover or focus.

<sui-tooltip tooltip="This is a tooltip" position="top">
  <button>Hover me</button>
</sui-tooltip>

Custom Styled Tooltips

Tooltips with custom styling and longer content.

<sui-tooltip 
  tooltip="This is a longer tooltip with more detailed information." 
  position="top">
  <button>Custom Button</button>
</sui-tooltip>

Disabled Tooltips

Tooltips can be disabled when needed.

<sui-tooltip 
  tooltip="This tooltip is disabled" 
  [disabled]="true">
  <button>Disabled Button</button>
</sui-tooltip>

Installation

The Tooltip component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Tooltip component in your Angular application:

import { Tooltip } from '@synerity/ui';

Then use it in your template:

<sui-tooltip tooltip="Helpful information" position="top"> <button>Hover me</button> </sui-tooltip>

Different Positions

The tooltip can be positioned in different directions:

<!-- Top Tooltip --> <sui-tooltip tooltip="Top tooltip" position="top"></sui-tooltip> <!-- Bottom Tooltip --> <sui-tooltip tooltip="Bottom tooltip" position="bottom"></sui-tooltip> <!-- Left Tooltip --> <sui-tooltip tooltip="Left tooltip" position="left"></sui-tooltip> <!-- Right Tooltip --> <sui-tooltip tooltip="Right tooltip" position="right"></sui-tooltip>

Disabled State

Disable tooltips when needed:

<sui-tooltip tooltip="This tooltip is disabled" [disabled]="true"> <button>Disabled Button</button> </sui-tooltip>

Component Properties

Prop Type Default Description
tooltip string '' Tooltip text content
position 'top' | 'bottom' | 'left' | 'right' 'top' Position of the tooltip
disabled boolean false Whether the tooltip is disabled
delay number 0 Delay in milliseconds before showing tooltip

Events

Event Description
onShow Emitted when the tooltip is shown
onHide Emitted when the tooltip is hidden

Overview

The Tooltip component provides a lightweight overlay that displays helpful information when users hover over or focus on an element. Tooltips are perfect for providing additional context, explanations, or guidance without cluttering the interface.

Tooltips automatically position themselves relative to their trigger element and can be configured to appear on any side with customizable delays and styling.

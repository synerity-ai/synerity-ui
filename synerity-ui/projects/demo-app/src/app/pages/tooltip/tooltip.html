Tooltip

Contextual information overlay that appears on hover or focus.

Installation

The Tooltip component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Use the tooltip directive on any element:

<button 
  sui-tooltip="Your tooltip text"
  class="px-4 py-2 bg-blue-600 text-white rounded-md">
  Hover me
</button>

Positioning

Control tooltip position with the tooltipPosition attribute:

<button 
  sui-tooltip="Top tooltip"
  tooltipPosition="top">
  Top
</button>

<button 
  sui-tooltip="Bottom tooltip"
  tooltipPosition="bottom">
  Bottom
</button>

Props

Prop Type Default Description
sui-tooltip string '' Tooltip text content
tooltipPosition 'top' | 'bottom' | 'left' | 'right' 'top' Position of the tooltip
tooltipDelay number 0 Delay before showing tooltip (ms)
tooltipDisabled boolean false Whether the tooltip is disabled

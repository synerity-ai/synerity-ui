Drawer

Slide-out panels that can be positioned on any side of the screen with customizable content and behavior.

Basic Drawer

<sui-drawer
  [visible]="leftDrawerVisible"
  [position]="leftDrawerConfig.position"
  [modal]="leftDrawerConfig.modal"
  [closable]="leftDrawerConfig.closable"
  [width]="leftDrawerConfig.width"
  (onShow)="onDrawerShow()"
  (onHide)="onDrawerHide()">
  
  <div class="p-6">
    <h3>Left Drawer Content</h3>
    <p>This is the content of the left drawer.</p>
  </div>
</sui-drawer>

Modal vs Non-Modal

Modal drawer blocks interaction with the background, while non-modal allows interaction with the main content. 

<sui-drawer
  [visible]="modalDrawerVisible"
  [position]="modalDrawerConfig.position"
  [modal]="modalDrawerConfig.modal"
  [closable]="modalDrawerConfig.closable"
  [width]="modalDrawerConfig.width"
  (onShow)="onDrawerShow()"
  (onHide)="onDrawerHide()">
  
  <div class="p-6">
    <h3>Modal Drawer Content</h3>
    <p>This drawer blocks interaction with the background.</p>
  </div>
</sui-drawer>

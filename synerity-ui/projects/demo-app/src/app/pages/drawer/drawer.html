Drawer

A slide-out panel component that can be positioned on any side of the screen.

Left Drawer

A drawer that slides in from the left side of the screen.

<sui-drawer 
  [visible]="showDrawer" 
  position="left"
  header="Left Drawer"
  (onHide)="showDrawer = false">
  <p>This drawer slides in from the left.</p>
</sui-drawer>

Installation

The Drawer component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Drawer component in your Angular application:

import { Drawer } from '@synerity/ui';

Then use it in your template:

<sui-drawer [visible]="showDrawer" position="left" header="Drawer Title" (onHide)="showDrawer = false"> <p>Drawer content goes here.</p> </sui-drawer>

Different Positions

The drawer can be positioned on any side of the screen:

<!-- Left Drawer --> <sui-drawer [visible]="showDrawer" position="left"></sui-drawer> <!-- Right Drawer --> <sui-drawer [visible]="showDrawer" position="right"></sui-drawer> <!-- Top Drawer --> <sui-drawer [visible]="showDrawer" position="top"></sui-drawer> <!-- Bottom Drawer --> <sui-drawer [visible]="showDrawer" position="bottom"></sui-drawer>

Custom Size

You can customize the size of the drawer:

<sui-drawer [visible]="showDrawer" position="left" width="400px" height="500px" (onHide)="showDrawer = false"> <p>Custom sized drawer.</p> </sui-drawer>

Component Properties

Prop Type Default Description
visible boolean false Controls the visibility of the drawer
position 'left' | 'right' | 'top' | 'bottom' 'left' Position of the drawer
header string '' Header text of the drawer
width string '300px' Width of the drawer (for left/right positions)
height string '300px' Height of the drawer (for top/bottom positions)
closable boolean true Whether the drawer can be closed

Events

Event Description
onShow Emitted when the drawer is shown
onHide Emitted when the drawer is hidden

Overview

The Drawer component provides a slide-out panel that can be positioned on any side of the screen. It's perfect for navigation menus, settings panels, or any secondary content that needs to be accessible but not always visible.

Drawers slide in smoothly from the specified direction and can be closed by clicking the backdrop, the close button, or pressing the ESC key.

InputTextarea

Multi-line text input components with auto-resize, validation states, and character limits.

Basic Textarea

<!-- Basic Textarea -->
<sui-textarea 
  [(ngModel)]="basicTextarea"
  (change)="onTextareaChange($event)"
  [rows]="4"
  placeholder="Enter your text here..."
  inputId="basic-textarea"
  name="basicTextarea">
</sui-textarea>

<!-- With Placeholder -->
<sui-textarea 
  [(ngModel)]="placeholderTextarea"
  [rows]="4"
  placeholder="This is a placeholder text for the textarea field. You can type multiple lines here."
  inputId="placeholder-textarea"
  name="placeholderTextarea">
</sui-textarea>

Auto-resize Textarea

This textarea automatically adjusts its height based on content.

<!-- Auto-resize Textarea -->
<sui-textarea 
  [(ngModel)]="autoResizeValue"
  (change)="onTextareaChange($event)"
  [rows]="3"
  [autosize]="true"
  placeholder="Type here and watch the textarea grow automatically..."
  inputId="auto-resize-textarea"
  name="autoResizeTextarea">
</sui-textarea>

// Component method (optional - sui-textarea handles auto-resize automatically)
onTextareaChange(event: string) {{ '{' }}
  console.log('Textarea value changed:', event);
{{ '}' }}

Installation

The InputTextarea component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the InputTextarea component in your Angular application:

import {{ '{' }} Textarea {{ '}' }} from '@synerity/ui';

{{ '@' }}Component({{ '{' }}
  template: `
    <sui-textarea 
      [(ngModel)]="textareaValue"
      [rows]="4"
      placeholder="Enter your text here..."
      inputId="my-textarea"
      name="myTextarea">
    </sui-textarea>
  `
{{ '}' }})
export class MyComponent {{ '{' }}
  textareaValue = '';
{{ '}' }}

Character Count

Implement character counting and limits:

// In your component
getCharacterCount(): number {{ '{' }}
  return this.textareaValue.length;
{{ '}' }}

isCharacterLimitReached(): boolean {{ '{' }}
  return this.getCharacterCount() >= this.maxLength;
{{ '}' }}

// In your template
<textarea 
  [(ngModel)]="textareaValue"
  [maxlength]="maxLength"
  rows="4"
  class="w-full px-3 py-2 border border-gray-300 rounded-md shadow-sm focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-blue-500 transition-colors resize-vertical"
  placeholder="Enter your text here..."></textarea>

<div class="text-sm text-gray-500 mt-2">
  <span [class.text-red-500]="isCharacterLimitReached()">
    {{ '{{' }} getCharacterCount() {{ '}}' }}/{{ '{{' }} maxLength {{ '}}' }} characters
  </span>
</div>

Props

Prop Type Default Description
disabled boolean false Whether the textarea is disabled
name string '' Name attribute for the textarea
inputId string '' ID attribute for the textarea
placeholder string '' Placeholder text for the textarea
rows number 3 Number of visible text lines
autosize boolean false Whether the textarea should automatically resize based on content

Events

Event Type Description
change EventEmitter<string> Emitted when the textarea value changes

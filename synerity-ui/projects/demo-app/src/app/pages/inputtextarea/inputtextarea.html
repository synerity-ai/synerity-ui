InputTextarea

Multi-line text input components with auto-resize, validation states, and character limits.

Auto-resize Textarea

This textarea automatically adjusts its height based on content.

<!-- Auto-resize Textarea -->
<textarea 
  [(ngModel)]="textareaValue"
  (input)="autoResizeTextarea($event)"
  rows="3"
  class="w-full px-3 py-2 border border-gray-300 rounded-md shadow-sm focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-blue-500 transition-colors resize-none overflow-hidden"
  placeholder="Type here and watch the textarea grow automatically..."></textarea>

// Component method
autoResizeTextarea(event: Event) {{ '{' }}
  const textarea = event.target as HTMLTextAreaElement;
  textarea.style.height = 'auto';
  textarea.style.height = textarea.scrollHeight + 'px';
{{ '}' }}

Installation

The InputTextarea component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the InputTextarea component in your Angular application:

import {{ '{' }} InputTextarea {{ '}' }} from '@synerity/ui';

{{ '@' }}Component({{ '{' }}
  template: `
    <textarea 
      [(ngModel)]="textareaValue"
      rows="4"
      class="w-full px-3 py-2 border border-gray-300 rounded-md shadow-sm focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-blue-500 transition-colors resize-vertical"
      placeholder="Enter your text here..."></textarea>
  `
{{ '}' }})
export class MyComponent {{ '{' }}
  textareaValue = '';
{{ '}' }}

Auto-resize Implementation

To implement auto-resize functionality:

// In your component
autoResizeTextarea(event: Event) {{ '{' }}
  const textarea = event.target as HTMLTextAreaElement;
  textarea.style.height = 'auto';
  textarea.style.height = textarea.scrollHeight + 'px';
{{ '}' }}

// In your template
<textarea 
  [(ngModel)]="value"
  (input)="autoResizeTextarea($event)"
  rows="3"
  class="w-full px-3 py-2 border border-gray-300 rounded-md shadow-sm focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-blue-500 transition-colors resize-none overflow-hidden"
  placeholder="Auto-resizing textarea..."></textarea>

Character Count

Implement character counting and limits:

// In your component
getCharacterCount(): number {{ '{' }}
  return this.textareaValue.length;
{{ '}' }}

isCharacterLimitReached(): boolean {{ '{' }}
  return this.getCharacterCount() >= this.maxLength;
{{ '}' }}

// In your template
<textarea 
  [(ngModel)]="textareaValue"
  [maxlength]="maxLength"
  rows="4"
  class="w-full px-3 py-2 border border-gray-300 rounded-md shadow-sm focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-blue-500 transition-colors resize-vertical"
  placeholder="Enter your text here..."></textarea>

<div class="text-sm text-gray-500 mt-2">
  <span [class.text-red-500]="isCharacterLimitReached()">
    {{ '{{' }} getCharacterCount() {{ '}}' }}/{{ '{{' }} maxLength {{ '}}' }} characters
  </span>
</div>

Props

Prop Type Default Description
rows number 4 Number of visible text lines
cols number 50 Number of visible character widths
placeholder string '' Placeholder text for the textarea
disabled boolean false Whether the textarea is disabled
readonly boolean false Whether the textarea is read-only
required boolean false Whether the textarea is required
maxlength number undefined Maximum number of characters allowed
autoResize boolean false Whether the textarea should auto-resize

Events

Event Type Description
onInput EventEmitter<string> Emitted when the textarea value changes
onFocus EventEmitter<FocusEvent> Emitted when the textarea receives focus
onBlur EventEmitter<FocusEvent> Emitted when the textarea loses focus
onResize EventEmitter<Event> Emitted when the textarea is resized

Mask Patterns

9 - Digits

Accepts only numeric characters (0-9)

A - Letters

Accepts only alphabetic characters (A-Z, a-z)

* - Alphanumeric

Accepts both letters and numbers

Literals

All other characters are displayed as-is

Props

Prop Type Default Description
ngModel string '' Two-way binding for input value
mask string '' Mask pattern (9=digit, A=letter, *=alphanumeric)
maskPlaceholder string '_' Character to show for unfilled positions
placeholder string '' Placeholder text for the input
disabled boolean false Disable the input mask component
name string '' Name attribute for form submission
inputId string '' ID attribute for the input element

Events

Event Parameters Description
change string Callback when input value changes

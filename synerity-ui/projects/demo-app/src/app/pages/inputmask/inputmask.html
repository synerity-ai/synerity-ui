Installation

Import the InputMask component in your Angular module:

import { InputMask } from '@synerity/ui-lib';

@NgModule({
  imports: [
    InputMask
  ]
})

Basic Usage

Use the InputMask component in your template:

<sui-input-mask
  [(ngModel)]="maskedValue"
  mask="999-999-9999"
  placeholder="Enter phone number"
  (change)="onValueChange($event)">
</sui-input-mask>

Form Integration

The InputMask implements ControlValueAccessor and works seamlessly with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-input-mask
    formControlName="phone"
    mask="(999) 999-9999"
    placeholder="Enter phone number">
  </sui-input-mask>
</form>

Component Overview

The InputMask component provides input formatting and validation using predefined patterns. It's perfect for phone numbers, dates, credit cards, and other structured data entry. The component automatically formats user input according to the specified mask pattern and provides real-time validation. It implements ControlValueAccessor for seamless integration with Angular forms.

Props

Property Type Default Description
ngModel string - Two-way binding for the masked value
mask string - Mask pattern (9=digit, A=letter, *=alphanumeric)
placeholder string - Placeholder text for the input
disabled boolean false Disables the input mask interaction
inputId string - Unique identifier for the input element
name string - Name attribute for the input element

Events

Event Type Description
change string Fired when the masked value changes

How to Use InputMask

1. Import the Component

import { SuiInputMask } from 'ui-lib';

2. Add to Component Imports

@Component({
  imports: [SuiInputMask, FormsModule],
  // ...
})

3. Use in Template

<sui-input-mask
  [(ngModel)]="maskedValue"
  mask="999-999-9999"
  placeholder="Enter phone number"
  (change)="onValueChange($event)">
</sui-input-mask>

4. Common Mask Patterns

<!-- Phone Number -->
<sui-input-mask mask="(999) 999-9999"></sui-input-mask>

<!-- Date -->
<sui-input-mask mask="99/99/9999"></sui-input-mask>

<!-- Credit Card -->
<sui-input-mask mask="9999-9999-9999-9999"></sui-input-mask>

<!-- Currency -->
<sui-input-mask mask="999,999.99"></sui-input-mask>

InputMask API Reference

Property Type Default Description
mask string '' Mask pattern (9=digit, A=letter, *=alphanumeric)
placeholder string '' Placeholder text
disabled boolean false Disable the component
inputId string '' ID attribute for the input
name string '' Name attribute for the input

Events

Event Type Description
change EventEmitter<string> Fired when the value changes

Mask Characters

Character Description Example
9 Digit (0-9) 999-999-9999
A Letter (a-z, A-Z) AAA-999
* Alphanumeric ***-***

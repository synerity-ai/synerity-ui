ToggleButton

A toggle button component that supports both single and multiple selections with icon support.

Single Selection

Selected: {{ singleValue || 'None' }}

<sui-toggle-button
  [(ngModel)]="singleValue"
  [options]="singleOptions"
  [multiple]="false"
  (change)="onSingleChange($event)"
></sui-toggle-button>

Multiple Selection

Selected: {{ multipleValue.length > 0 ? multipleValue.join(', ') : 'None' }}

<sui-toggle-button
  [(ngModel)]="multipleValue"
  [options]="multipleOptions"
  [multiple]="true"
  (change)="onMultipleChange($event)"
></sui-toggle-button>

How to Use ToggleButton

Installation

Import the ToggleButton component from the UI library:

import { ToggleButton } from 'ui-lib';

// In your component imports
imports: [ToggleButton]

Basic Usage

Use the component with an array of options:

<sui-toggle-button
  [(ngModel)]="selectedValue"
  [options]="options"
  [multiple]="false"
  (change)="onSelectionChange($event)"
></sui-toggle-button>

Component Properties

  • options: Array of option objects with label, value, and optional icon
  • multiple: Boolean to enable multiple selections
  • disabled: Boolean to disable the entire component
  • ngModel: Two-way binding for the selected value(s)

Events

  • change: Emitted when selection changes

Option Structure

options = [
  { label: 'Option 1', value: 'option1' },
  { label: 'Option 2', value: 'option2', icon: '🎯' },
  { label: 'Option 3', value: 'option3' }
];

Single vs Multiple Selection

  • Single: Set [multiple]="false", value will be a single item
  • Multiple: Set [multiple]="true", value will be an array

API Documentation

Properties

Property Type Default Description
options Array<{label: string, value: any, icon?: string}> [] Array of selectable options
multiple boolean false Enables multiple selection
disabled boolean false Disables the entire component
ngModel any | any[] null Selected value(s) (two-way binding)

Events

Event Type Description
change EventEmitter<any | any[]> Fired when selection changes

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Tab navigation between buttons
  • Focus Management: Clear focus indicators
  • Screen Reader: Descriptive labels for all options

Styling

The component uses CSS classes that can be customized:

  • .toggle-button-container: Main container
  • .toggle-button-option: Individual button styling
  • .toggle-button-option-selected: Selected button styling
  • .toggle-button-option-disabled: Disabled button styling

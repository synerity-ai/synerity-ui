ToggleButton

A toggle button component that supports both single and multiple selections with icon support.

Single Selection

Selected: {{ singleValue || 'None' }}

  [options]="singleOptions"
  [multiple]="false"
  (change)="onSingleChange($event)"
></sui-toggle-button>

Installation

The ToggleButton component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the ToggleButton component in your Angular application:

import { ToggleButton } from '@synerity/ui';

{{ '@' }}Component({{ '{' }}
  template: `
    <sui-toggle-button
      [(ngModel)]="selectedValue"
      [options]="options"
      [multiple]="false"
      (change)="onSelectionChange($event)">
    </sui-toggle-button>
  `
{{ '}' }})
export class MyComponent {{ '{' }}
  selectedValue: any = null;
  options = [
    { label: 'Option 1', value: 'option1' },
    { label: 'Option 2', value: 'option2' },
    { label: 'Option 3', value: 'option3' }
  ];
  
  onSelectionChange(value: any) {{ '{' }}
    console.log('Selection changed:', value);
  {{ '}' }}
{{ '}' }}

Component Overview

The ToggleButton component provides a group of toggle buttons that allow users to select from multiple options. It supports both single and multiple selection modes, making it ideal for forms where users need to choose one or more options from a predefined set.

The component features modern styling with gradient backgrounds, smooth animations, and comprehensive customization options including color variants, size options, and different visual styles.

Props

Property Type Default Description
options Array<{label: string, value: any, icon?: string}> [] Array of selectable options
multiple boolean false Enables multiple selection
disabled boolean false Disables the entire component
ngModel any | any[] null Selected value(s) (two-way binding)
size 'sm' | 'md' | 'lg' 'md' Controls the size of the toggle buttons
variant 'default' | 'success' | 'warning' | 'danger' | 'purple' 'default' Controls the color scheme of selected buttons
style 'default' | 'outline' | 'ghost' 'default' Controls the visual style of the toggle buttons
shape 'default' | 'rounded' | 'pill' 'default' Controls the border radius of the toggle buttons

Events

Event Type Description
change EventEmitter<any | any[]> Fired when selection changes

InputOTP

A one-time password input component for verification codes with automatic focus management.

Basic InputOTP

Enter the 6-digit verification code:

<!-- Basic InputOTP -->
<sui-input-otp
  [(ngModel)]="otpValue"
  [length]="6"
  (change)="onOtpChange($event)"
  (complete)="onOtpComplete($event)"
></sui-input-otp>

Installation

The InputOTP component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Key Features

InputOTP provides several useful features for verification workflows:

auto-focus Automatically moves to next input on digit entry
backspace Moves to previous input on backspace
paste Supports pasting multi-digit codes
validation Filters input based on numericOnly setting
keyboard Full keyboard navigation support

Props

Prop Type Default Description
length number 6 Number of input fields
disabled boolean false Disable the entire component
numericOnly boolean true Restrict input to numbers only
inputType 'text' | 'tel' 'tel' HTML input type attribute
name string '' Name attribute for form control

Events

Event Type Description
change EventEmitter<string> Emitted when any digit changes
complete EventEmitter<string> Emitted when all digits are filled

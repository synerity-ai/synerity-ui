Features

Individual Input Fields

Each character has its own input field

Auto Focus

Automatic focus management between fields

Paste Support

Support for pasting complete codes

Validation

Built-in numeric/alphanumeric validation

Props

Prop Type Default Description
ngModel string '' Two-way binding for OTP value
length number 6 Number of input fields
disabled boolean false Disable the OTP input component
name string '' Name attribute for form submission
inputType 'text' | 'tel' 'tel' Input type for each field
numericOnly boolean true Restrict input to numeric characters only

Events

Event Parameters Description
change string Callback when OTP value changes
complete string Callback when all fields are filled

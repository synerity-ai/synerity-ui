InputOTP

A one-time password input component for verification codes with automatic focus management.

Basic OTP (6 digits)

Enter the 6-digit verification code:

Current value: {{ basicOtpValue || 'Empty' }}

<sui-input-otp
  [(ngModel)]="basicOtpValue"
  [length]="6"
  (change)="onBasicOtpChange($event)"
  (complete)="onBasicOtpComplete($event)"
></sui-input-otp>

How to Use InputOTP

Installation

Import the InputOTP component from the UI library:

import { InputOtp } from 'ui-lib';

// In your component imports
imports: [InputOtp]

Basic Usage

Use the component for verification codes:

<sui-input-otp
  [(ngModel)]="otpValue"
  [length]="6"
  (change)="onOtpChange($event)"
  (complete)="onOtpComplete($event)"
></sui-input-otp>

Component Properties

  • length: Number of input fields (default: 6)
  • disabled: Boolean to disable the entire component
  • numericOnly: Boolean to allow only numeric input (default: true)
  • inputType: HTML input type ('text' or 'tel', default: 'tel')
  • name: String for form control name
  • ngModel: Two-way binding for the OTP value

Events

  • change: Emitted when any digit changes
  • complete: Emitted when all digits are filled

Features

  • Auto-focus: Automatically moves to next input on digit entry
  • Backspace handling: Moves to previous input on backspace
  • Paste support: Supports pasting multi-digit codes
  • Input validation: Filters input based on numericOnly setting
  • Keyboard navigation: Full keyboard support

Common Use Cases

  • SMS Verification: 6-digit codes sent via SMS
  • Email Verification: Alphanumeric codes sent via email
  • Two-Factor Authentication: TOTP codes from authenticator apps
  • Account Recovery: Verification codes for password reset

API Documentation

Properties

Property Type Default Description
length number 6 Number of input fields
disabled boolean false Disables the entire component
numericOnly boolean true Restricts input to numbers only
inputType 'text' | 'tel' 'tel' HTML input type attribute
name string '' Name attribute for form control
ngModel string '' OTP value (two-way binding)

Events

Event Type Description
change EventEmitter<string> Fired when any digit changes
complete EventEmitter<string> Fired when all digits are filled

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Arrow keys, Tab, and Backspace support
  • Focus Management: Automatic focus handling between inputs
  • Screen Reader: Descriptive labels and instructions

Styling

The component uses CSS classes that can be customized:

  • .input-otp-container: Main container
  • .input-otp-field: Individual input field
  • .input-otp-field-disabled: Disabled input field
  • .input-otp-field-focused: Focused input field

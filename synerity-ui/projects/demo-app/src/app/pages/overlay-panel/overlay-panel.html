Overlay Panel

A panel component that displays content in an overlay layer.

Basic Usage

Basic Overlay Panel

This is a basic overlay panel with default styling.

 
(click)="toggleBasicOverlayPanel()"
<sui-overlay-panel 
  [visible]="showBasicOverlayPanel"
  (onShow)="onShow()"
  (onHide)="onHide()"
  (visibleChange)="showBasicOverlayPanel = $event"
>
  <div class="overlay-content">
    <h3>Basic Overlay Panel</h3>
    <p>This is a basic overlay panel with default styling.</p>
  </div>
</sui-overlay-panel>

Positioned Overlay Panel

Positioned Overlay Panel

This overlay panel is positioned above the trigger button.

 
<sui-overlay-panel 
  [visible]="showPositionedOverlayPanel"
  [styleClass]="'sui-overlay-panel-top'"
  (visibleChange)="showPositionedOverlayPanel = $event"
>
  <div class="overlay-content">
    <h3>Positioned Overlay Panel</h3>
    <p>This overlay panel is positioned above the trigger button.</p>
  </div>
</sui-overlay-panel>

Modal Overlay Panel

Modal Overlay Panel

This is a modal overlay panel that covers the entire screen.

 
<sui-overlay-panel 
  [visible]="showModalOverlayPanel"
  [modal]="true"
  [styleClass]="'sui-overlay-panel-large'"
  (visibleChange)="showModalOverlayPanel = $event"
>
  <div class="overlay-content">
    <h3>Modal Overlay Panel</h3>
    <p>This is a modal overlay panel that covers the entire screen.</p>
    <button class="demo-button" (click)="showModalOverlayPanel = false">
      Close Modal
    </button>
  </div>
</sui-overlay-panel>

Custom Styled Overlay Panel

Custom Styled Panel

This overlay panel has custom styling applied.

 
<sui-overlay-panel 
  [visible]="showCustomOverlayPanel"
  [style]="{
    backgroundColor: '#f0f9ff',
    borderColor: '#0ea5e9',
    color: '#0c4a6e'
  }"
  [styleClass]="'sui-overlay-panel-small'"
  (visibleChange)="showCustomOverlayPanel = $event"
>
  <div class="overlay-content">
    <h3>Custom Styled Panel</h3>
    <p>This overlay panel has custom styling applied.</p>
  </div>
</sui-overlay-panel>

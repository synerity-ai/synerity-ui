IftLabel Component

Simple conditional label component that displays different labels based on a boolean condition.

Basic IftLabel

Current condition: {{ basicCondition }} 

<sui-ift-label 
  [condition]="true"
  [trueLabel]="'Enabled'"
  [falseLabel]="'Disabled'"
></sui-ift-label>

IftLabel Variants

Simple true/false label switching based on condition. 

<sui-ift-label 
  [condition]="false"
  [trueLabel]="'Public'"
  [falseLabel]="'Private'"
></sui-ift-label>

IftLabel States

Custom styling with styleClass for different visual states. 

<sui-ift-label 
  [condition]="true"
  [trueLabel]="'Available'"
  [falseLabel]="'Unavailable'"
  styleClass="custom-status-label"
></sui-ift-label>

Installation

npm install @synerity/ui

Basic Usage

The IftLabel component displays different labels based on a boolean condition. It supports two types of labels:

  • trueLabel: Displayed when condition is true
  • falseLabel: Displayed when condition is false
<sui-ift-label 
  [condition]="isEnabled"
  [trueLabel]="'Enabled'"
  [falseLabel]="'Disabled'"
></sui-ift-label>

Styling

You can customize the appearance using styleClass and style properties: 

<sui-ift-label 
  [condition]="status"
  [trueLabel]="'Online'"
  [falseLabel]="'Offline'"
  styleClass="status-badge"
  [style]="{'font-weight': 'bold'}"
></sui-ift-label>

Props

Property Type Default Description
condition boolean true Condition that determines which label to display
trueLabel string '' Label displayed when condition is true
falseLabel string '' Label displayed when condition is false
styleClass string '' Custom CSS class for styling
style any {} Inline styles object

Accessibility

  • Semantic HTML with proper span elements
  • Screen reader friendly with clear label content
  • Supports custom styling for high contrast modes
  • No keyboard interaction required (display-only component)

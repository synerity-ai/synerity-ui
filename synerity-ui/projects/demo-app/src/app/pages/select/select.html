Select

A dropdown selection component for choosing single or multiple options from a list.

Basic Usage

The sui-select component provides a powerful dropdown selection with various features:

1. Basic Select

Use sui-select with ngModel for single option selection.

2. Multi Select

Set [multiple]="true" to allow selection of multiple options.

3. Searchable Select

Set [searchable]="true" to enable search functionality.

4. Clearable Select

Set [clearable]="true" to show a clear button.

Component Integration

To use Select in your Angular component:

1. Import FormsModule

Import FormsModule from @angular/forms to enable two-way data binding.

2. Define Options Data

Create arrays of option objects with value and label properties.

3. Handle Selection Changes

Create event handlers to respond to selection changes.

API Reference

sui-select Component

Property Type Description
ngModel unknown | unknown[] Two-way binding for the selected value(s)
options Array<{{ '{' }}label: string, value: unknown, disabled?: boolean{{ '}' }}> Array of options to display
multiple boolean Enables multiple selection
searchable boolean Enables search functionality
clearable boolean Shows a clear button
disabled boolean Disables the select component
placeholder string | null Placeholder text for the select
inputId string ID attribute for the select element
name string Name attribute for the select element
selectAll boolean Shows select all/clear all buttons (multi-select only)
maxDisplayItems number Maximum items to show in display (with "+X more")
showSelectedCount boolean Shows count of selected items in display
searchPlaceholder string Custom placeholder for search input
noResultsText string Text shown when no search results found
loading boolean Shows loading state for async options

Events

Event Type Description
change EventEmitter<unknown> Emitted when the selection changes

Option Element

Attribute Type Description
value string The value of the option
disabled boolean Disables the option
selected boolean Pre-selects the option

Events

Event Type Description
change Event Fired when selection changes
focus Event Fired when select gains focus
blur Event Fired when select loses focus

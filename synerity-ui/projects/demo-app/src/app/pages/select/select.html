Select

Dropdown selection components with support for search, clear, and various styling options.

Basic Usage

Selected: {{ selectedValue || 'None' }}

Selected: {{ selectedCountry || 'None' }}

<!-- Basic Select -->
<sui-select 
  [options]="basicOptions" 
  [(ngModel)]="selectedValue"
  placeholder="Choose an option"
  name="basicSelect">
</sui-select>

<!-- With Placeholder -->
<sui-select 
  [options]="countryOptions" 
  [(ngModel)]="selectedCountry"
  placeholder="Select a country"
  name="countrySelect">
</sui-select>

<!-- Required Field -->

  [options]="basicOptions" 
  [(ngModel)]="selectedValue"
  placeholder="Please select"
  name="requiredSelect">

Props

Prop Type Default Description
options Array<{{ '{' }}label: string, value: unknown, disabled?: boolean{{ '}' }}> [] Array of options for the select dropdown
placeholder string | null null Placeholder text when no option is selected
disabled boolean false Whether the select is disabled
searchable boolean false Whether the select supports searching
clearable boolean false Whether the select can be cleared
name string '' Name attribute for form handling
inputId string '' ID attribute for the select element

Events

Event Type Description
change EventEmitter<unknown> Emitted when the selected value changes

Accessibility

  • • Proper ARIA attributes for dropdown functionality
  • • Keyboard navigation support (Arrow keys, Enter, Escape)
  • • Screen reader announcements for option changes
  • • Focus management and visual focus indicators
  • • Support for disabled options within the dropdown

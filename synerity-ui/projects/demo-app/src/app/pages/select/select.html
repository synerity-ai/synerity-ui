Select

Installation

Import the Select component in your module or standalone component:

import { Select } from 'ui-lib'; // In your component imports imports: [Select, FormsModule]

Basic Usage

Use the Select component with ngModel for two-way data binding:

<sui-select
  [(ngModel)]="selectedValue"
  [options]="optionsList"
  placeholder="Choose an option..."
  (change)="onSelectionChange($event)"
  inputId="my-select"
  name="mySelect">
</sui-select>

Form Integration

Integrate with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-select
    formControlName="selectedOption"
    [options]="options"
    [multiple]="true"
    [searchable]="true"
    [clearable]="true"
    placeholder="Select options..."
    inputId="form-select"
    name="formSelect">
  </sui-select>
</form>

Select Component

The Select component provides a powerful dropdown selection with support for single/multiple selection, search functionality, and customizable options.

Props

Prop Type Default Description
ngModel unknown | unknown[] - Two-way binding for the selected value(s)
options Array<{{ '{' }}label: string, value: unknown, disabled?: boolean{{ '}' }}> [] Array of options to display
multiple boolean false Enables multiple selection
searchable boolean false Enables search functionality
clearable boolean false Shows a clear button
disabled boolean false Disables the select component
placeholder string - Placeholder text for the select
inputId string - ID attribute for the select element
name string - Name attribute for the select element

Events

Event Type Description
change EventEmitter<unknown> Emitted when the selection changes

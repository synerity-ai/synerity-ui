Card

Flexible container components for displaying content with headers, footers, and various styling options.

Basic Cards

Simple Card

This is a basic card with simple content.

This card has a header and content area.

This card includes both header and subheader.

 
<!-- Simple Card -->
<sui-card>
  <h3 class="text-lg font-semibold text-gray-900 mb-2">Simple Card</h3>
  <p class="text-gray-600">This is a basic card with simple content.</p>
</sui-card>

<!-- Card with Header -->
<sui-card header="Card with Header">
  <p class="text-gray-600">This card has a header and content area.</p>
</sui-card>

<!-- Complete Card -->
<sui-card header="Complete Card" subheader="With subheader">
  <p class="text-gray-600 mb-4">This card includes both header and subheader.</p>
  <button class="px-4 py-2 bg-primary-600 text-white rounded-md hover:bg-primary-700">Action</button>
</sui-card>

Card with elevated shadow for emphasis.

Outlined

Card States

Normal Card

This is a normal, interactive card.

Loading Card

This card is in a loading state.

Disabled Card

This card is disabled and non-interactive.

 
<!-- Normal Card -->
<sui-card>
  <h3 class="text-lg font-semibold text-gray-900 mb-2">Normal Card</h3>
  <p class="text-gray-600">This is a normal, interactive card.</p>
</sui-card>

<!-- Loading Card -->
<sui-card [loading]="true">
  <h3 class="text-lg font-semibold text-gray-900 mb-2">Loading Card</h3>
  <p class="text-gray-600">This card is in a loading state.</p>
</sui-card>

<!-- Disabled Card -->
<sui-card [disabled]="true">
  <h3 class="text-lg font-semibold text-gray-900 mb-2">Disabled Card</h3>
  <p class="text-gray-600">This card is disabled and non-interactive.</p>
</sui-card>

Clickable Cards

Clickable Card

Click this card to see the action.

Hover Effect

This card has hover effects when clickable.

Interactive

Fully interactive card with click handling.

 
<!-- Basic Clickable Card -->
<sui-card [clickable]="true" (click)="onCardClick('Card 1')">
  <h3 class="text-lg font-semibold text-gray-900 mb-2">Clickable Card</h3>
  <p class="text-gray-600">Click this card to see the action.</p>
</sui-card>

<!-- Clickable Card with Hover Effect -->
<sui-card [clickable]="true" [hoverable]="true" (click)="onCardClick('Card 2')">
  <h3 class="text-lg font-semibold text-gray-900 mb-2">Hover Effect</h3>
  <p class="text-gray-600">This card has hover effects when clickable.</p>
</sui-card>

<!-- Interactive Card -->
<sui-card [clickable]="true" (click)="onCardClick('Card 3')">
  <h3 class="text-lg font-semibold text-gray-900 mb-2">Interactive</h3>
  <p class="text-gray-600">Fully interactive card with click handling.</p>
</sui-card>

Installation

npm install @synerity/ui

Basic Usage

<sui-card>
  <div class="p-6">
    <h3>Card Title</h3>
    <p>Card content goes here.</p>
  </div>
</sui-card>

Import

import {{ '{' }} Card {{ '}' }} from '@synerity/ui';

Examples

Card with Header

<sui-card header="My Card">
  <div class="p-6">Content here</div>
</sui-card>

Clickable Card

<sui-card [clickable]="true" (click)="onClick()">
  <div class="p-6">Clickable content</div>
</sui-card>

Card with Footer

<sui-card header="Card Title">
  <div class="p-6">Content here</div>
  <div slot="footer" class="p-6 pt-0">
    <button>Action</button>
  </div>
</sui-card>

Props

Prop Type Default Description
variant 'default' | 'elevated' | 'outlined' 'default' Visual style variant of the card
header string '' Header text for the card
subheader string '' Subheader text for the card
clickable boolean false Whether the card is clickable
disabled boolean false Whether the card is disabled
loading boolean false Whether the card shows loading state

Events

Event Type Description
click MouseEvent Emitted when the card is clicked (if clickable)
headerClick MouseEvent Emitted when the card header is clicked

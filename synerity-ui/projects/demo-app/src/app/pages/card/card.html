Installation

The Card component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Card component in your Angular application:

import {{ '{' }} Card {{ '}' }} from '@synerity/ui';

{{ '@' }}Component({{ '{' }}

Card Structure

Cards can have three main sections:

1

Header

Optional header section with title and actions

2

Body

Main content area for text, images, and other components

3

Footer

Optional footer section with additional actions or information

Styling with Tailwind CSS

Cards use Tailwind CSS classes for styling. Here are the key classes:

bg-white White background
border border-gray-200 Border styling
rounded-lg Border radius
shadow-sm Subtle shadow
overflow-hidden Hide overflow content

Props

Prop Type Default Description
title string '' Card header title
content string '' Card body content
footer string '' Card footer content
elevated boolean false Whether the card has enhanced shadow
bordered boolean true Whether the card has a border
rounded boolean true Whether the card has rounded corners

Events

Event Type Description
onClick EventEmitter<MouseEvent> Emitted when the card is clicked
onHeaderClick EventEmitter<MouseEvent> Emitted when the card header is clicked
onFooterClick EventEmitter<MouseEvent> Emitted when the card footer is clicked

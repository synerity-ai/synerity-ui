Installation

npm install @synerity/ui

Basic Usage

<sui-card>
  <div class="p-6">
    <h3>Card Title</h3>
    <p>Card content goes here.</p>
  </div>
</sui-card>

Import

import {{ '{' }} Card {{ '}' }} from '@synerity/ui';

Examples

Card with Header

<sui-card header="My Card">
  <div class="p-6">Content here</div>
</sui-card>

Clickable Card

<sui-card [clickable]="true" (click)="onClick()">
  <div class="p-6">Clickable content</div>
</sui-card>

Card with Footer

<sui-card header="Card Title">
  <div class="p-6">Content here</div>
  <div slot="footer" class="p-6 pt-0">
    <button>Action</button>
  </div>
</sui-card>

Props

Prop Type Default Description
variant 'default' | 'elevated' | 'outlined' 'default' Visual style variant of the card
header string '' Header text for the card
subheader string '' Subheader text for the card
clickable boolean false Whether the card is clickable
disabled boolean false Whether the card is disabled
loading boolean false Whether the card shows loading state

Events

Event Type Description
click MouseEvent Emitted when the card is clicked (if clickable)
headerClick MouseEvent Emitted when the card header is clicked

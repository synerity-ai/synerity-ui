Card

A flexible container component that displays content in a structured layout with header, body, and footer sections.

Basic Card

{{ cardData.content }} 
<sui-card 
  [header]="cardData.title"
  [subheader]="cardData.subtitle"
  [showFooter]="true"
>
  {{ cardData.content }}
</sui-card>

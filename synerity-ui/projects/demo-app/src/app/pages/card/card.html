Enhanced visual appeal

Cards can include icons and enhanced visual elements.

 
<!-- Basic Card with Header, Content, and Footer -->
<sui-card 
  header="Card Title"
  subheader="Card subtitle"
  [showFooter]="true">
  <p class="text-gray-600">This is the card content.</p>
  
  <ng-template #footer>
    <div class="flex justify-between items-center">
      <span class="text-sm text-gray-500">Footer content</span>
      <button class="px-3 py-1 bg-blue-500 text-white text-sm rounded">
        Action
      </button>
    </div>
  </ng-template>
</sui-card>

Interactive Cards

This card is clickable and responds to hover and click events.

Click to interact

Card with multiple action buttons in the footer.

 
<!-- Clickable Card -->
<sui-card 
  header="Clickable Card"
  subheader="Hover and click interactions"
  [clickable]="true"
  (click)="onCardSelect($event)">
  <p class="text-gray-600">This card is clickable and responds to hover and click events.</p>
</sui-card>

<!-- Card with Actions -->
<sui-card 
  header="Card with Actions"
  subheader="Multiple action buttons">
  <p class="text-gray-600">Card with multiple action buttons.</p>
  
  <ng-template #footer>
    <div class="flex gap-2">
      <button class="flex-1 px-3 py-2 bg-gray-100 text-gray-700 text-sm rounded">
        Cancel
      </button>
      <button class="flex-1 px-3 py-2 bg-blue-500 text-white text-sm rounded">
        Save
      </button>
    </div>
  </ng-template>
</sui-card>

Installation

The Card component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Card component in your Angular application:

Styling with Tailwind CSS

Card uses Tailwind CSS classes for styling. Here are the key classes:

rounded-lg Border radius for card corners
shadow-md Card shadow effect
border Border styling
bg-white Background color
p-6 Internal padding

Props

Prop Type Default Description
header string '' Header text of the card
subheader string '' Subheader text of the card
showFooter boolean false Whether to show the footer section
variant 'default' | 'outlined' | 'elevated' 'default' Visual style variant of the card
clickable boolean false Whether the card can be clicked
hoverable boolean false Whether the card has hover effects

Events

Event Type Description
click EventEmitter<MouseEvent> Emitted when the card is clicked

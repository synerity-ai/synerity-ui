  size="md"

  header="Card Header"
  size="md"

</sui-card>

<!-- Card with Header and Footer -->
<sui-card 
  header="Card with Footer"
  size="md"
  variant="default"
  [showFooter]="true">
  <p class="text-gray-600">This card has both header and footer.</p>
</sui-card>

Installation

The Card component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Card Structure

Cards can have three main sections:

1

Header

Optional header section with title and actions

2

Body

Main content area for text, images, and other components

3

Footer

Optional footer section with additional actions or information

Styling with Tailwind CSS

Cards use Tailwind CSS classes for styling. Here are the key classes:

bg-white White background
border border-gray-200 Border styling
rounded-lg Border radius
shadow-sm Subtle shadow
overflow-hidden Hide overflow content

Props

Prop Type Default Description
variant 'default' | 'outlined' | 'elevated' | 'filled' 'default' Visual style variant of the card
size 'sm' | 'md' | 'lg' 'md' Size of the card padding
clickable boolean false Whether the card is clickable
disabled boolean false Whether the card is disabled
header string '' Card header title text
subheader string '' Card header subtitle text
showHeader boolean true Whether to show the card header
showFooter boolean false Whether to show the card footer
loading boolean false Whether the card is in loading state
hoverable boolean false Whether the card has hover effects
borderless boolean false Whether the card has no border

Events

Event Type Description
click EventEmitter<MouseEvent> Emitted when the card is clicked
headerClick EventEmitter<MouseEvent> Emitted when the card header is clicked

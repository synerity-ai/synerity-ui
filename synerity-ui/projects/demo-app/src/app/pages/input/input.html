Password with Toggle

<sui-input-text type="password" showPasswordToggle="true"></sui-input-text>

Props

Prop Type Default Description
label string '' Label text for the input field
type 'text' | 'email' | 'password' | 'tel' | 'url' 'text' HTML input type
variant 'default' | 'filled' | 'outlined' 'default' Visual style variant
disabled boolean false Whether the input is disabled
showPasswordToggle boolean false Show password visibility toggle for password inputs

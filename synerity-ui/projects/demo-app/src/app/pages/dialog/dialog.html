Dialog

Modal dialog component with draggable, resizable, and customizable features.

Basic Dialog

This is a basic dialog with a header and close button.

 
<sui-dialog 
  [visible]="dialogVisible"
  header="Basic Dialog"
  [closable]="true"
  (onHide)="closeDialog()">
  <div class="p-4">
    <p>Dialog content goes here</p>
  </div>
</sui-dialog>

Modal Dialog

This is a modal dialog that blocks interaction with the background.

 
<sui-dialog 
  [visible]="modalVisible"
  header="Modal Dialog"
  [modal]="true"
  [closable]="true"
  (onHide)="closeModal()">
  <div class="p-4">
    <p>Modal content goes here</p>
  </div>
</sui-dialog>

Installation

The Dialog component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Dialog Types

Basic Dialog

Simple dialog without modal behavior

Modal Dialog

Dialog that blocks interaction with background content

Resizable Dialog

Dialog that can be resized by the user

Draggable Dialog

Dialog that can be moved around the screen

Props

Prop Type Default Description
visible boolean false Whether the dialog is visible
modal boolean true Whether the dialog is modal
closable boolean true Whether the dialog can be closed
draggable boolean true Whether the dialog can be dragged
resizable boolean false Whether the dialog can be resized
header string '' Dialog header text

Events

Event Type Description
onShow EventEmitter<void> Emitted when the dialog is shown
onHide EventEmitter<void> Emitted when the dialog is hidden

Dialog

Modal dialogs with customizable headers, content, and interactive features like dragging and resizing.

Basic Dialog

<button (click)="dialogVisible = true">Show Dialog</button>

<sui-dialog 
  [visible]="dialogVisible"
  header="Dialog Title"
  (onHide)="dialogVisible = false">
  <p>This is the dialog content.</p>
</sui-dialog>

Modal Dialog

<sui-dialog 
  [visible]="modalVisible"
  [modal]="true"
  header="Modal Dialog"
  (onHide)="modalVisible = false">
  <p>This is a modal dialog with backdrop.</p>
</sui-dialog>

Resizable Dialog

<sui-dialog 
  [visible]="resizableVisible"
  [resizable]="true"
  [width]="'500px'"
  [height]="'400px'"
  header="Resizable Dialog"
  (onHide)="resizableVisible = false">
  <p>Drag the bottom-right corner to resize this dialog.</p>
</sui-dialog>

Installation

npm install @synerity/ui

import { Dialog } from '@synerity/ui';

@Component({
  imports: [Dialog],
  // ...
})

Basic Usage

<sui-dialog 
  [visible]="dialogVisible"
  header="Dialog Title"
  (onHide)="dialogVisible = false">
  <p>Dialog content goes here.</p>
</sui-dialog>

Features

Modal Support

Block interaction with the rest of the page

Draggable

Drag the dialog by its header

Resizable

Resize the dialog by dragging corners

Maximizable

Maximize to full screen

Props

Prop Type Default Description
visible boolean false Specifies the visibility of the dialog
modal boolean true Defines if background should be blocked
header string '' Title text of the dialog
draggable boolean true Enables dragging to reposition the dialog
resizable boolean false Enables resizing of the dialog
maximizable boolean false Shows the maximize icon
width string 'auto' Width of the dialog
height string 'auto' Height of the dialog
position string 'center' Position of the dialog
closable boolean true Adds a close icon to the header

Events

Event Parameters Description
onShow void Callback to invoke when dialog becomes visible
onHide void Callback to invoke when dialog becomes hidden
onMaximize void Callback to invoke when dialog is maximized
onUnmaximize void Callback to invoke when dialog is unmaximized

Accessibility

  • • Focus management when dialog opens/closes
  • • ESC key support for closing dialog
  • • ARIA roles and labels for screen readers
  • • Keyboard navigation support
  • • Focus trap within modal dialogs

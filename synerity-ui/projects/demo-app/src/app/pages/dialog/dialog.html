Dialog

A dialog component for displaying modal content with various features like resizing, maximizing, and positioning.

Basic Dialog

A simple modal dialog with header and content.

<sui-dialog 
  [visible]="showDialog" 
  header="Basic Dialog"
  (onHide)="showDialog = false">
  <p>This is a basic dialog with header and content.</p>
  <div class="flex justify-end space-x-3">
    <button (click)="showDialog = false">Cancel</button>
    <button (click)="showDialog = false">OK</button>
  </div>
</sui-dialog>

Draggable Dialog

A dialog that can be dragged around the screen by clicking and dragging the header.

<sui-dialog 
  [visible]="showDialog" 
  header="Draggable Dialog"
  [draggable]="true"
  width="400px"
  height="300px"
  (onHide)="showDialog = false">
  <p>This dialog can be dragged around the screen.</p>
</sui-dialog>

Resizable Dialog

A dialog that can be resized by dragging the bottom-right corner.

<sui-dialog 
  [visible]="showDialog" 
  header="Resizable Dialog"
  [resizable]="true"
  width="500px"
  height="400px"
  (onHide)="showDialog = false">
  <p>This dialog can be resized by dragging the corner.</p>
</sui-dialog>

Maximizable Dialog

A dialog that can be maximized to full screen by clicking the maximize button.

<sui-dialog 
  [visible]="showDialog" 
  header="Maximizable Dialog"
  [maximizable]="true"
  width="600px"
  height="400px"
  (onHide)="showDialog = false">
  <p>This dialog can be maximized to full screen.</p>
</sui-dialog>

Positioned Dialog

A dialog positioned at the top-right of the screen instead of the center.

<sui-dialog 
  [visible]="showDialog" 
  header="Positioned Dialog"
  position="top-right"
  width="350px"
  height="250px"
  (onHide)="showDialog = false">
  <p>This dialog is positioned at the top-right.</p>
</sui-dialog>

Installation

The Dialog component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Dialog component in your Angular application:

import { Dialog } from '@synerity/ui';

Then use it in your template:

<sui-dialog [visible]="showDialog" header="Dialog Title" (onHide)="showDialog = false"> <p>Dialog content goes here.</p> </sui-dialog>

Advanced Features

Draggable Dialog

Enable dragging by setting the draggable property to true:

<sui-dialog [visible]="showDialog" header="Draggable Dialog" [draggable]="true" (onHide)="showDialog = false"> <p>This dialog can be dragged around.</p> </sui-dialog>

Resizable Dialog

Enable resizing by setting the resizable property to true:

<sui-dialog [visible]="showDialog" header="Resizable Dialog" [resizable]="true" width="400px" height="300px" (onHide)="showDialog = false"> <p>This dialog can be resized.</p> </sui-dialog>

Maximizable Dialog

Enable maximizing by setting the maximizable property to true:

<sui-dialog [visible]="showDialog" header="Maximizable Dialog" [maximizable]="true" (onHide)="showDialog = false"> <p>This dialog can be maximized.</p> </sui-dialog>

Positioned Dialog

Set custom position using the position property:

<sui-dialog [visible]="showDialog" header="Positioned Dialog" position="top-right" width="350px" height="250px" (onHide)="showDialog = false"> <p>This dialog is positioned at the top-right.</p> </sui-dialog>

Component Properties

Prop Type Default Description
visible boolean false Controls the visibility of the dialog
header string '' Header text of the dialog
modal boolean true Whether the dialog is modal
closable boolean true Whether the dialog can be closed
draggable boolean true Whether the dialog can be dragged
resizable boolean false Whether the dialog can be resized
maximizable boolean false Whether the dialog can be maximized
width string 'auto' Width of the dialog
height string 'auto' Height of the dialog
position string 'center' Position of the dialog (center, top, bottom, left, right, top-left, top-right, bottom-left, bottom-right)

Events

Event Description
onShow Emitted when the dialog is shown
onHide Emitted when the dialog is hidden
onMaximize Emitted when the dialog is maximized
onUnmaximize Emitted when the dialog is unmaximized

Overview

The Dialog component provides a modal dialog interface for displaying content. It supports various features including dragging, resizing, maximizing, and custom positioning.

Dialogs are perfect for displaying forms, confirmations, detailed information, or any content that requires user attention without navigating away from the current page.

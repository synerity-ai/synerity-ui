Installation

Import the Slider component in your Angular module:

import { Slider } from '@synerity/ui-lib';

@NgModule({
  imports: [
    Slider
  ]
})

Basic Usage

Use the Slider component in your template:

<sui-slider
  [(ngModel)]="sliderValue"
  (change)="onSliderChange($event)"
  [min]="0"
  [max]="100"
  [step]="1"
  inputId="slider"
  name="slider">
</sui-slider>

Form Integration

The Slider implements ControlValueAccessor and works seamlessly with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-slider
    formControlName="slider"
    [min]="0"
    [max]="100"
    [step]="1">
  </sui-slider>
</form>

Component Overview

The Slider component provides an intuitive way to select numeric values within a specified range. It supports customizable min/max values, step increments, and various interactive states. The component implements ControlValueAccessor for seamless integration with Angular forms.

Props

Property Type Default Description
ngModel number 0 Two-way binding for the slider value
min number 0 Minimum value of the slider
max number 100 Maximum value of the slider
step number 1 Step size for value changes
disabled boolean false Disables the slider interaction
inputId string - Unique identifier for the slider element
name string - Name attribute for the slider element

Events

Event Type Description
change number Fired when the slider value changes

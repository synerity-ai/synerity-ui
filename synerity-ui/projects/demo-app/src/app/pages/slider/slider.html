Slider

Basic Usage

The Slider component provides an intuitive way to select numeric values within a specified range.

1. Import the Component

import { Slider as SuiSlider } from '@synerity/ui';

2. Add to Component Imports

@Component({
  imports: [CommonModule, FormsModule, SuiSlider],
  // ...
})

3. Use in Template

<sui-slider
  [(ngModel)]="sliderValue"
  (change)="onSliderChange($event)"
  [min]="0"
  [max]="100"
  [step]="1"
  inputId="my-slider"
  name="sliderName">
</sui-slider>

API Reference

Properties

Property Type Default Description
min number 0 Minimum value of the slider
max number 100 Maximum value of the slider
step number 1 Step size for value changes
disabled boolean false Whether the slider is disabled
name string '' Name attribute for the input
inputId string '' ID attribute for the input

Events

Event Type Description
change EventEmitter<number> Emitted when the slider value changes

Features

Customizable Range

Set custom min, max, and step values

Form Integration

Full Angular forms support with validation

Accessibility

Keyboard navigation and screen reader support

Responsive

Works seamlessly across all device sizes

Props

Prop Type Default Description
ngModel number 0 Two-way binding for slider value
min number 0 Minimum allowed value
max number 100 Maximum allowed value
step number 1 Step increment value
disabled boolean false Disable the slider
name string '' Name attribute for form submission
inputId string '' ID attribute for the input

Events

Event Parameters Description
change number Callback to invoke when value changes

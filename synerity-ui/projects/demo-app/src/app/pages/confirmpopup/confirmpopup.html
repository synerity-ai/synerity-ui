ConfirmPopup

Contextual popup component for confirming user actions with customizable positioning and styling.

Basic ConfirmPopup

Last action: {{ lastBasicAction }} 
<sui-confirm-popup
  message="Are you sure you want to delete this item?"
  acceptLabel="Delete"
  rejectLabel="Cancel"
  (onAccept)="onBasicAccept()"
  (onReject)="onBasicReject()">
  <button class="px-4 py-2 text-sm bg-red-600 text-white hover:bg-red-700 rounded-md">
    Delete Item
  </button>
</sui-confirm-popup>

Installation

The ConfirmPopup component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
visible boolean false Whether the popup is visible
message string 'Are you sure?' Message text of the popup
position 'top' | 'bottom' | 'left' | 'right' 'top' Position of the popup relative to the target

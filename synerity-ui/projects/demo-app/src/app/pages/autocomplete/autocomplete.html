AutoComplete

An input component that provides real-time suggestions as users type, with customizable filtering and selection options.

Basic AutoComplete

{{ basicValue || 'None' }} 
<sui-auto-complete
  [(ngModel)]="basicValue"
  [options]="countries"
  placeholder="Type to search countries..."
  (change)="onBasicChange($event)"
  inputId="basic-autocomplete"
  name="basicAutocomplete">
</sui-auto-complete>

Installation

Import the AutoComplete component in your module or standalone component:

import { AutoComplete } from 'ui-lib'; // In your component imports imports: [AutoComplete]

Basic Usage

Use the AutoComplete component with ngModel for two-way data binding:

<sui-auto-complete
  [(ngModel)]="selectedValue"
  [options]="optionsList"
  placeholder="Type to search..."
  (change)="onValueChange($event)"
  inputId="my-autocomplete"
  name="myAutocomplete">
</sui-auto-complete>

Form Integration

Integrate with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-auto-complete
    formControlName="selectedCountry"
    [options]="countries"
    placeholder="Select a country..."
    inputId="country-autocomplete"
    name="countryAutocomplete">
  </sui-auto-complete>
</form>

AutoComplete Component

The AutoComplete component provides an input field with dropdown suggestions that appear as the user types. It supports customizable filtering and selection options.

Props

Prop Type Default Description
ngModel string - Two-way binding for the selected value
options string[] [] Array of options to display in the dropdown
placeholder string - Placeholder text for the input field
disabled boolean false Disable the input field
inputId string - Unique identifier for the input element
name string - Name attribute for the input element

Events

Event Type Description
change string Emitted when the selected value changes

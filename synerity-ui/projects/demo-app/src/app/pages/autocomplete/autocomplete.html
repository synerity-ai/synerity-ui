AutoComplete

An input component that provides real-time suggestions as users type, with customizable filtering and selection options.

Basic AutoComplete

{{ basicValue || 'None' }} 
<sui-auto-complete
  [(ngModel)]="basicValue"
  [options]="countries"
  placeholder="Type to search countries..."
  (change)="onBasicChange($event)"
  inputId="basic-autocomplete"
  name="basicAutocomplete">
</sui-auto-complete>

Installation

Import the AutoComplete component in your module or standalone component:

import { AutoComplete } from 'ui-lib';

// In your component imports
imports: [AutoComplete]

Basic Usage

Use the AutoComplete component with ngModel for two-way data binding:

<sui-auto-complete
  [(ngModel)]="selectedValue"
  [options]="optionsList"
  placeholder="Type to search..."
  (ngModelChange)="onValueChange($event)"
  inputId="my-autocomplete"
  name="myAutocomplete">
</sui-auto-complete>

Form Integration

Integrate with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-auto-complete
    formControlName="selectedCountry"
    [options]="countries"
    placeholder="Select a country..."
    inputId="country-autocomplete"
    name="countryAutocomplete">
  </sui-auto-complete>
</form>

AutoComplete Component

The AutoComplete component provides an input field with dropdown suggestions that appear as the user types. It supports single and multiple selection modes with customizable filtering.

Props

Property Type Default Description
ngModel string | string[] - Two-way binding for the selected value(s)
options string[] [] Array of options to display in the dropdown
placeholder string - Placeholder text for the input field
multiple boolean false Enable multiple selection mode
disabled boolean false Disable the input field
filterFunction function - Custom filter function for options
inputId string - Unique identifier for the input element
name string - Name attribute for the input element

Events

Event Type Description
ngModelChange string | string[] Emitted when the selected value changes
onFocus Event Emitted when the input receives focus
onBlur Event Emitted when the input loses focus

Examples

Basic AutoComplete

<sui-auto-complete
  [(ngModel)]="selectedCountry"
  [options]="countries"
  placeholder="Select a country..."
  inputId="country-select"
  name="countrySelect">
</sui-auto-complete>

Multiple Selection

<sui-auto-complete
  [(ngModel)]="selectedCities"
  [options]="cities"
  [multiple]="true"
  placeholder="Select cities..."
  inputId="cities-select"
  name="citiesSelect">
</sui-auto-complete>

Custom Filter Function

<sui-auto-complete
  [(ngModel)]="selectedItem"
  [options]="items"
  [filterFunction]="customFilter"
  placeholder="Search items..."
  inputId="item-search"
  name="itemSearch">
</sui-auto-complete>

Form Integration

<form [formGroup]="myForm" (ngSubmit)="onSubmit()">
  <sui-auto-complete
    formControlName="selectedOption"
    [options]="options"
    placeholder="Choose an option..."
    inputId="form-autocomplete"
    name="formAutocomplete">
  </sui-auto-complete>
  
  <button type="submit" class="btn btn-primary">Submit</button>
</form>

How to Use AutoComplete

1. Import the Component

import { SuiAutoComplete } from 'ui-lib';

2. Add to Component Imports

@Component({
  imports: [SuiAutoComplete, FormsModule],
  // ...
})

3. Use in Template

<sui-auto-complete
  [(ngModel)]="selectedValue"
  [options]="suggestionList"
  placeholder="Type to search"
  (change)="onValueChange($event)">
</sui-auto-complete>

4. With Custom Options

<sui-auto-complete
  [(ngModel)]="selectedValue"
  [options]="productList"
  placeholder="Search products"
  [minLength]="2">
</sui-auto-complete>

AutoComplete API Reference

Property Type Default Description
options Array<string | object> [] Array of options to display
minLength number 1 Minimum length to trigger suggestions
inputId string '' ID attribute for the input
disabled boolean false Disable the component
placeholder string '' Placeholder text
name string '' Name attribute for the input

Event Type Description
change EventEmitter<any> Fired when the value changes

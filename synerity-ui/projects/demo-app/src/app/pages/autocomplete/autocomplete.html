Features

Smart Filtering

Real-time filtering with customizable minimum length

Keyboard Navigation

Arrow keys, Enter, and Escape support

Flexible Options

Support for string arrays or object arrays

Form Integration

Full Angular forms support with validation

Props

Prop Type Default Description
ngModel unknown '' Two-way binding for selected value
options Array<string | {{ '{' }}label: string, value: unknown{{ '}' }}> [] Array of options to search from
placeholder string '' Placeholder text for the input
minLength number 1 Minimum characters to trigger suggestions
disabled boolean false Disable the autocomplete component
name string '' Name attribute for form submission
inputId string '' ID attribute for the input element

Events

Event Parameters Description
change unknown Callback to invoke when value changes

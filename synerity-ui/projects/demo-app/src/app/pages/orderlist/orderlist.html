OrderList

Reorderable list component with selection, filtering, and drag-and-drop capabilities.

Basic OrderList

<sui-order-list 
  [value]="items"
  header="Task List"
  [filter]="true"
  filterPlaceholder="Search tasks..."
  (onReorder)="onReorder($event)"
  (onSelectionChange)="onSelectionChange($event)">
</sui-order-list>

Installation

The OrderList component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
value any[] [] Array of items to display
header string '' Header text for the list
filter boolean false Enable search/filter functionality

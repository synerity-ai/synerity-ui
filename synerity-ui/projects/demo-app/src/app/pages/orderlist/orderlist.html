How to Use OrderList

Installation

Import the OrderList component from the UI library:

import { OrderList } from 'ui-lib';

// In your component imports
imports: [OrderList]

Basic Usage

Use the component with an array of items:

<sui-order-list 
  [value]="items"
  header="My Items"
  (onReorder)="onReorder($event)"
  (onSelectionChange)="onSelectionChange($event)"
></sui-order-list>

Component Properties

  • value: Array of items to display and reorder
  • header: Optional header text for the list
  • listStyle: Custom styling for the list container
  • dragdrop: Boolean to enable drag & drop functionality
  • filter: Boolean to enable filtering
  • filterPlaceholder: Placeholder text for the filter input

Events

  • onReorder: Emitted when items are reordered
  • onSelectionChange: Emitted when item selection changes

Features

  • Manual Reordering: Use up/down buttons to reorder items
  • Drag & Drop: Enable drag & drop for intuitive reordering
  • Filtering: Search and filter items in the list
  • Selection: Select multiple items for bulk operations
  • Custom Styling: Flexible styling options

Data Structure

Items can be any objects, but typically include:

items = [
  { id: 1, name: 'Item 1', description: 'Description' },
  { id: 2, name: 'Item 2', description: 'Description' }
];

API Documentation

Properties

Property Type Default Description
value any[] [] Array of items to display
header string '' Header text for the list
listStyle any {} Custom styling for the list
dragdrop boolean false Enable drag & drop functionality
filter boolean false Enable filtering
filterPlaceholder string 'Search' Filter input placeholder

Events

Event Type Description
onReorder EventEmitter<{items: any[], direction: string, index: number}> Fired when items are reordered
onSelectionChange EventEmitter<any[]> Fired when selection changes

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Full keyboard support for reordering
  • Focus Management: Proper focus handling and visual indicators
  • Screen Reader: Descriptive labels for all actions

Styling

The component uses CSS classes that can be customized:

  • .sui-order-list: Main container
  • .sui-order-list-header: Header styling
  • .sui-order-list-filter: Filter input styling
  • .sui-order-list-items: Items container
  • .sui-order-list-item: Individual item styling
  • .sui-order-list-item-selected: Selected item styling
  • .sui-order-list-controls: Reorder controls styling

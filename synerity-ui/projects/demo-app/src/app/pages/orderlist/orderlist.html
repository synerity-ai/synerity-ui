OrderList

Order list components for managing and reordering items with drag-and-drop functionality and filtering capabilities.

Basic OrderList

<sui-order-list 
  [value]="basicItems"
  header="Task List"
  (onReorder)="onReorder($event)"
  (onSelectionChange)="onSelectionChange($event)">
</sui-order-list>

With Filter

<sui-order-list 
  [value]="filterItems"
  header="Product List"
  [filter]="true"
  filterPlaceholder="Search products..."
  (onReorder)="onReorder($event)"
  (onSelectionChange)="onSelectionChange($event)">
</sui-order-list>

Custom Styling

<sui-order-list 
  [value]="styledItems"
  header="Priority Tasks"
  [listStyle]="customListStyle"
  (onReorder)="onReorder($event)"
  (onSelectionChange)="onSelectionChange($event)">
</sui-order-list>

Installation

npm install @synerity/ui-lib

Import

import { OrderList } from '@synerity/ui-lib';

@Component({
  imports: [OrderList],
  // ...
})

Basic Usage

<sui-order-list [value]="items"></sui-order-list>

Features

  • ✓ Item reordering with up/down controls
  • ✓ Multiple item selection
  • ✓ Built-in filtering functionality
  • ✓ Customizable header and styling
  • ✓ Event handling for reorder and selection

Props

Name Type Default Description
value any[] [] Array of items to display
header string '' Header text for the list
filter boolean false Enable filtering functionality

Events

Name Parameters Description
onReorder { items: any[], direction: string, index: number } Fired when items are reordered
onSelectionChange any[] Fired when selection changes

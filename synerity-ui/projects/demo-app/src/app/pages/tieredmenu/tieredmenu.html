TieredMenu

Multi-level hierarchical menu component with nested submenus.

Basic TieredMenu

Interactive: Click on menu items to expand submenus. Hover over items with arrows to see nested options. 

<sui-tiered-menu 
  [model]="tieredMenuItems"
  (onItemClick)="onTieredMenuItemClick($event)">
</sui-tiered-menu>

// Component data
tieredMenuItems = [
  {
    label: 'File',
    icon: '📁',
    items: [
      { label: 'New', icon: '➕' },
      { label: 'Open', icon: '📂' },
      { label: 'Save', icon: '💾' }
    ]
  }
];

Installation

The TieredMenu component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the TieredMenu component in your Angular application:

import { TieredMenu } from '@synerity/ui';

@Component({
  template: `
    <sui-tiered-menu [model]="menuItems"></sui-tiered-menu>
  `,
  data: {
    menuItems: [
      { label: 'Home', icon: '🏠' },
      {
        label: 'Products',
        icon: '📦',
        items: [
          { label: 'Electronics', icon: '📱' }
        ]
      }
    ]
  }
})
export class MyComponent {}

Menu Item Structure

Each menu item can have the following properties:

label Optional. Display text for the menu item
icon Optional. Icon to display before the label
command Optional. Function to execute when clicked
disabled Optional. Boolean to disable the menu item
items Optional. Array of submenu items

Event Handling

Handle menu item clicks using the onItemClick event:

<sui-tiered-menu 
  [model]="menuItems"
  (onItemClick)="onMenuItemClick($event)">
</sui-tiered-menu>

onMenuItemClick(item: TieredMenuItem) {
  console.log('Menu item clicked:', item.label);
  
  // Execute custom command
  if (item.command) {
    item.command();
  }
}

Popup Mode

Use popup mode to show the menu as an overlay:

<sui-tiered-menu 
  [model]="menuItems"
  [popup]="true"
  (onItemClick)="onMenuItemClick($event)">
</sui-tiered-menu>

Props

Prop Type Default Description
model Array<TieredMenuItem> [] Array of menu items to display
popup boolean false Show menu as popup overlay
style any {} Inline styles for the menu
styleClass string '' CSS class name for custom styling

Events

Event Type Description
onItemClick EventEmitter<TieredMenuItem> Emitted when a menu item is clicked

TieredMenuItem Interface

interface TieredMenuItem {
  label?: string;                    // Display text
  icon?: string;                     // Icon to display
  command?: () => void;              // Click handler function
  disabled?: boolean;                // Disabled state
  items?: TieredMenuItem[];          // Submenu items
}

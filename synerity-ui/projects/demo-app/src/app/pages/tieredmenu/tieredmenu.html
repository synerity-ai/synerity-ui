Basic Usage

Import the TieredMenu component and use it in your template:

// Component imports import {{ '{' }} TieredMenu as SuiTieredMenu {{ '}' }} from 'ui-lib'; @Component({{ '{' }} imports: [SuiTieredMenu], template: \` <sui-tiered-menu [model]="menuItems" (onItemClick)="onItemClick($event)"> </sui-tiered-menu> \` {{ '}' }}) export class MyComponent {{ '{' }} menuItems = [ {{ '{' }} label: 'Home', icon: '🏠', command: () => this.goHome() {{ '}' }}, {{ '{' }} label: 'About', icon: 'ℹ️', command: () => this.goAbout() {{ '}' }} ]; onItemClick(item: any): void {{ '{' }} console.log('Item clicked:', item); {{ '}' }} {{ '}' }}

Menu Items

Define your menu items in your component:

// Component data menuItems = [ { label: 'Products', icon: '📦', items: [ { label: 'Electronics', command: () => this.showElectronics() }, { label: 'Clothing', command: () => this.showClothing() }, { label: 'Books', command: () => this.showBooks() } ] }, { label: 'Services', icon: '🔧', command: () => this.showServices() } ];

Nested Submenus

Create multi-level nested submenus:

nestedItems = [ { label: 'Products', items: [ { label: 'Electronics', items: [ { label: 'Phones', command: () => this.showPhones() }, { label: 'Laptops', command: () => this.showLaptops() } ] } ] } ];

Overview

The TieredMenu component provides a horizontal navigation menu with multi-level submenu support. It's designed for top navigation bars and complex menu structures where users need to access hierarchical content through hover interactions.

TieredMenu supports nested submenus, icons, disabled items, and smooth hover transitions to create intuitive navigation experiences.

TieredMenuItem Interface

interface TieredMenuItem { label?: string; // Display text for the menu item icon?: string; // Icon to display (emoji or icon class) command?: () => void; // Function to execute when clicked disabled?: boolean; // Whether the item is disabled separator?: boolean; // Whether this is a separator line items?: TieredMenuItem[]; // Child menu items }

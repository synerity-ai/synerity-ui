Installation

The Tiered Menu component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Tiered Menu component in your Angular application:

import { TieredMenu } from '@synerity/ui';

@Component({
  template: `
    <sui-tieredmenu 
      [model]="tieredMenuItems"
      (onItemClick)="onItemClick($event)">
    </sui-tieredmenu>
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Tiered Menu uses Tailwind CSS classes for styling. Here are the key classes:

sui-tieredmenu Base class for the tiered menu container
sui-tieredmenu-item Class for individual menu items
sui-tieredmenu-submenu Class for submenu containers

Props

Prop Type Default Description
model Array<MenuItem> [] Array of menu items to display
style any {} Inline styles for the tiered menu container
styleClass string '' Style class for the tiered menu container

Events

Event Type Description
onItemClick EventEmitter<MenuItem> Emitted when a menu item is clicked

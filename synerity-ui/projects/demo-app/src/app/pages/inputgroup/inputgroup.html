How to Use InputGroup

Installation

Import the InputGroup component from the UI library:

import { InputGroup } from 'ui-lib';

// In your component imports
imports: [InputGroup]

Basic Usage

Use the component to group related input elements:

<sui-input-group [size]="'normal'" [disabled]="false">
  <div class="flex">
    <span class="inline-flex items-center px-3 text-sm text-gray-900 bg-gray-200 border border-r-0 border-gray-300 rounded-l-md">
      @
    </span>
    <input
      type="text"
      class="rounded-none rounded-r-md border-gray-300 focus:ring-blue-500 focus:border-blue-500 block flex-1 min-w-0 w-full text-sm"
      placeholder="Enter your username"
    />
  </div>
</sui-input-group>

Component Properties

  • size: Size variant ('small', 'normal', 'large')
  • disabled: Boolean to disable the entire group
  • styleClass: Custom CSS classes
  • style: Inline styles object

Common Patterns

  • Prefix/Suffix Icons: Add icons or text before/after input
  • Button Groups: Group buttons with inputs
  • Multiple Elements: Combine inputs, selects, and buttons
  • Form Layouts: Create consistent form styling

Styling Guidelines

  • Use consistent sizing across all elements in the group
  • Maintain proper border radius (rounded-l for first, rounded-r for last)
  • Ensure proper focus states and accessibility
  • Use semantic colors and icons for better UX

API Documentation

Properties

Property Type Default Description
size 'small' | 'normal' | 'large' 'normal' Size variant for the input group
disabled boolean false Disables the entire input group
styleClass string '' Custom CSS classes
style any {} Inline styles object

Methods

Method Return Type Description
getInputGroupClass() string Returns computed CSS classes
getInputGroupStyle() any Returns computed inline styles
getSize() string Returns current size
isDisabled() boolean Returns disabled state
isSmall() boolean Checks if size is small
isNormal() boolean Checks if size is normal
isLarge() boolean Checks if size is large

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Full keyboard support
  • Focus Management: Proper focus handling within groups
  • Screen Reader: Descriptive labels and instructions

Styling

The component uses CSS classes that can be customized:

  • .sui-input-group: Main container
  • .sui-input-group-small: Small size variant
  • .sui-input-group-normal: Normal size variant
  • .sui-input-group-large: Large size variant
  • .sui-input-group-disabled: Disabled state

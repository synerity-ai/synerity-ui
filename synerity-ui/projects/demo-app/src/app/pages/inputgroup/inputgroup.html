InputGroup

A container component for grouping related input elements with consistent styling and behavior.

Installation

The InputGroup component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the InputGroup component in your Angular application:

import {{ '{' }} InputGroup {{ '}' }} from '@synerity/ui';

{{ '@' }}Component({{ '{' }}
  template: `
    <sui-input-group [size]="'normal'">
      <div class="flex">

Styling with Tailwind CSS

InputGroup uses Tailwind CSS classes for styling. Here are the key classes:

flex Flexbox layout for grouping elements
rounded-l-md Left border radius for first element
rounded-r-md Right border radius for last element
border-r-0 Remove right border for seamless connection
focus:ring-blue-500 Focus state styling

Props

Prop Type Default Description
size 'small' | 'normal' | 'large' 'normal' Size variant for the input group
disabled boolean false Disable the entire input group
styleClass string '' Custom CSS classes
style any {{ '{}' }} Inline styles object

Events

Event Type Description
onFocus EventEmitter<FocusEvent> Emitted when any element in the group receives focus
onBlur EventEmitter<FocusEvent> Emitted when any element in the group loses focus

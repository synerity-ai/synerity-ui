File Upload

A file upload component with drag and drop support, file validation, and progress tracking.

Basic File Upload

<sui-upload 
  [multiple]="true"
  [accept]="allowedFileTypes.join(',')"
  [maxFileSize]="maxFileSize"
  (onUpload)="onFileUpload($event)"
  (onSelect)="onFileSelect($event)"
  (onError)="onFileError($event)"
></sui-upload>

Installation

The File Upload component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the File Upload component in your Angular application:

import { FileUpload } from '@synerity/ui';

@Component({
  template: `
    <div class="p-4 bg-gray-100 rounded-lg" 
      [maxFileSize]="1000000"
      [allowedFileTypes]="['image/*', 'application/pdf']"
      (onUpload)="onUpload($event)"
      (onSelect)="onSelect($event)"
      (onError)="onError($event)">
    </div class="p-4 bg-gray-100 rounded-lg">
  `
})
export class MyComponent {}

Styling with Tailwind CSS

File Upload uses Tailwind CSS classes for styling. Here are the key classes:

div class="p-4 bg-gray-100 rounded-lg" Base class for the file upload container
div class="p-4 bg-gray-100 rounded-lg"-content Class for the upload content area
div class="p-4 bg-gray-100 rounded-lg"-button Class for the upload button

Props

Prop Type Default Description
maxFileSize number 0 Maximum file size in bytes
allowedFileTypes string[] [] Array of allowed file types (MIME types)
multiple boolean false Whether to allow multiple file selection
auto boolean true Whether to automatically upload files when selected
style any {} Inline styles for the file upload container
styleClass string '' Style class for the file upload container

Events

Event Type Description
onUpload EventEmitter<any> Emitted when files are uploaded
onSelect EventEmitter<any> Emitted when files are selected
onError EventEmitter<any> Emitted when an error occurs during upload

Chip

Compact elements that represent input, attribute, or action with icons, labels, and removal functionality.

Basic Chips

  label="Angular"
  [removable]="true"
  (onRemove)="onChipRemove($event)">
</sui-chip>

  // ...

  label="Angular"

Features

Labels

Display text content in the chip

Icons

Add icons or emojis to chips

Images

Include user avatars or images

Removable

Allow users to remove chips

Props

Prop Type Default Description
label string '' Text content of the chip
icon string '' Icon or emoji to display
image string '' URL of the image to display
removable boolean false Shows the remove icon
disabled boolean false Disables the chip interaction
style object {} Inline style object
styleClass string '' CSS class name

Events

Event Parameters Description
onRemove void Callback to invoke when chip is removed

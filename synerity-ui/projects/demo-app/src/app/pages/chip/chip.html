Chip

Small, compact elements for tags, labels, and removable items.

  [removable]="true"

</sui-chip>

Installation

The Chip component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
label string '' Chip text content
removable boolean false Whether the chip can be removed
icon string '' Icon to display in the chip
image string '' Image URL to display in the chip
disabled boolean false Whether the chip is disabled
style object {} Custom inline styles
styleClass string '' Additional CSS classes

Events

Event Type Description
onRemove EventEmitter<string> Emitted when the chip is removed

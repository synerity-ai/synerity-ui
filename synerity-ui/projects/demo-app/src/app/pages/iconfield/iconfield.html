Installation

The IconField component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the IconField component to add icons to your form fields:

Component Overview

The IconField component is a wrapper that adds icons to form fields. It provides a simple way to enhance form inputs with visual icons, improving user experience and visual clarity.

The component supports left and right icon positioning, different sizes, and custom styling options. It uses content projection to wrap any form field component.

Props

Property Type Default Description
icon string '' The icon to display (emoji, text, or symbol)
position 'left' | 'right' 'left' Position of the icon relative to the content
size 'small' | 'normal' | 'large' 'normal' Size of the icon field
style any {{ '{' }} {{ '}' }} Inline styles for the icon field
styleClass string '' Custom CSS class for the icon field

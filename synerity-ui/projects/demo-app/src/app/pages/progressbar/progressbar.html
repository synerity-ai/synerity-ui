Progress Bar

A progress bar component for displaying progress, loading states, and completion status.

Basic Progress Bar

A basic progress bar showing current progress with interactive controls.

Progress: {{ progressValue }}% {{ progressValue }}/{{ progressMax }}

Basic: Progress bar with value display and interactive controls. 

<sui-progress-bar 
  [value]="progressValue" 
  [showValue]="true">
</sui-progress-bar>

Indeterminate Progress Bar

An indeterminate progress bar for unknown duration tasks.

Loading...

Indeterminate: Continuous animation for unknown duration tasks. 

<sui-progress-bar 
  [mode]="'indeterminate'">
</sui-progress-bar>

Custom Progress Bar

Custom progress bars with different colors and units.

Custom Color Progress 75%
Custom Unit Progress 2.5 GB / 10 GB

Custom: Progress bars with custom colors and units. 

<sui-progress-bar 
  [value]="75" 
  [showValue]="true"
  color="#10b981">
</sui-progress-bar>

<sui-progress-bar 
  [value]="25" 
  [showValue]="true"
  unit="GB"
  color="#3b82f6">
</sui-progress-bar>

Installation

npm install ui-lib

Basic Usage

Import the ProgressBar component and use it in your template:

import {{ '{' }} ProgressBar {{ '}' }} from 'ui-lib';

// In your component
progressValue = 45;

// In your template
<sui-progress-bar [value]="progressValue"></sui-progress-bar>

Progress Types

Determinate Progress

Shows specific progress with a known maximum value

Indeterminate Progress

Shows continuous animation for unknown duration tasks

Properties

Property Type Default Description
value number 0 Current progress value
showValue boolean true Whether to display the progress value
unit string '%' Unit to display with the value
color string '' Custom color for the progress bar
mode 'determinate' | 'indeterminate' 'determinate' Progress bar mode
style any {} Inline styles for the component
styleClass string '' CSS class for the component

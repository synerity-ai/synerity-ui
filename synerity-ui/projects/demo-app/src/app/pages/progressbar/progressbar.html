Installation

The Progress Bar component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Progress Types

Determinate Progress

Shows specific progress with a known maximum value

Indeterminate Progress

Shows continuous animation for unknown duration tasks

Props

Prop Type Default Description
value number 0 Current progress value
max number 100 Maximum progress value
indeterminate boolean false Whether to show indeterminate animation
showValue boolean false Whether to display the progress value

Progress Bar

Visual indicators that show the progress of a task or operation with customizable colors, modes, and display options.

Basic Progress Bar

<sui-progress-bar 
  [value]="45"
  [showValue]="true">
</sui-progress-bar>

Colored Progress Bar

<sui-progress-bar 
  [value]="75"
  [color]="'#10b981'"
  [showValue]="true">
</sui-progress-bar>

Indeterminate Progress Bar

<sui-progress-bar 
  mode="indeterminate"
  [showValue]="false">
</sui-progress-bar>

Interactive Demo

<sui-progress-bar 
  [value]="interactiveProgress"
  [color]="interactiveColor"
  [showValue]="interactiveShowValue"
  [unit]="interactiveUnit">
</sui-progress-bar>

Installation

npm install @synerity/ui

Import

import { ProgressBar } from '@synerity/ui';

@Component({
  imports: [ProgressBar],
  // ...
})

Basic Usage

<sui-progress-bar 
  [value]="progress"
  [showValue]="true">
</sui-progress-bar>

Features

Determinate Mode

Shows specific progress value

Indeterminate Mode

Shows continuous loading animation

Custom Colors

Set custom progress bar colors

Value Display

Show or hide progress values

Props

Prop Type Default Description
value number 0 Current progress value (0-100)
mode string 'determinate' Progress bar mode: 'determinate' or 'indeterminate'
showValue boolean true Whether to display the progress value
unit string '%' Unit to display with the value
color string '' Custom color for the progress bar
style object {} Inline style object
styleClass string '' CSS class name

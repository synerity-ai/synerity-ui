Table

Data table component with sorting, filtering, selection, and pagination capabilities.

Basic Table

<sui-table 
  [value]="tableData"
  [columns]="tableColumns"
  [emptyMessage]="'No data available'">
</sui-table>

Table with Selection

Selected rows: {{ selectedRows.length }} 
<sui-table 
  [value]="tableData"
  [columns]="tableColumns"
  selectionMode="multiple"
  (onSelectionChange)="onSelectionChange($event)">
</sui-table>

Installation

The Table component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
value any[] [] Array of data objects to display
columns TableColumn[] [] Array of column definitions

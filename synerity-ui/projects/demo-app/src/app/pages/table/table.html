Table

Data table component with sorting, filtering, pagination, and selection capabilities.

Basic Table

<sui-table 
  [value]="data" 
  [columns]="columns"
  (onSort)="onSort($event)"
  (onFilter)="onFilter($event)">
</sui-table>

Sortable Table

Click on column headers to sort the data.

<sui-table 
  [value]="data" 
  [columns]="sortableColumns"
  (onSort)="onSort($event)">
</sui-table>

Installation

The Table component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Data Structure

Define your data array and column configuration:

// Sample data
data = [
  {{ '{' }} id: 1, name: 'John Doe', email: 'john@example.com' {{ '}' }},
  {{ '{' }} id: 2, name: 'Jane Smith', email: 'jane@example.com' {{ '}' }}
];

// Column configuration
columns = [
  {{ '{' }} field: 'id', header: 'ID', sortable: true {{ '}' }},
  {{ '{' }} field: 'name', header: 'Name', sortable: true, filterable: true {{ '}' }},
  {{ '{' }} field: 'email', header: 'Email', sortable: true, filterable: true {{ '}' }}
];

Props

Prop Type Default Description
value any[] [] Array of data objects to display
columns Column[] [] Column configuration array
paginator boolean false Whether to show pagination
rows number 10 Number of rows per page
selectionMode 'single' | 'multiple' | null null Row selection mode

Events

Event Type Description
onSort EventEmitter<any> Emitted when a column is sorted
onFilter EventEmitter<any> Emitted when data is filtered
onRowSelect EventEmitter<any> Emitted when a row is selected

Table

Data tables with sorting, filtering, pagination, and selection capabilities.

Basic Table

<sui-table 
  [value]="tableData"
  [columns]="tableColumns"
  (onSort)="onSort($event)"
  (onFilter)="onFilter($event)">
</sui-table>

Sortable Table

<sui-table 
  [value]="tableData"
  [columns]="sortableColumns"
  sortMode="multiple"
  (onSort)="onSort($event)">
</sui-table>

Selectable Table

Selected rows: {{ selectedRows.length }} 

<sui-table 
  [value]="tableData"
  [columns]="tableColumns"
  [selectionMode]="'multiple'"
  (onRowSelect)="onRowSelect($event)"
  (onRowUnselect)="onRowUnselect($event)"
  (onSelectionChange)="onSelectionChange($event)">
</sui-table>

Installation

npm install @synerity/ui

Import

import { Table } from '@synerity/ui';

@Component({
  imports: [Table],
  // ...
})

Basic Usage

<sui-table 
  [value]="data"
  [columns]="columns">
</sui-table>

Component Properties

// In your component
data = [
  { id: 1, name: 'John Doe', email: 'john@example.com' },
  { id: 2, name: 'Jane Smith', email: 'jane@example.com' }
];

columns = [
  { field: 'name', header: 'Name' },
  { field: 'email', header: 'Email' }
];

Features

Sorting

Enable column sorting with sortable property

Filtering

Add filtering capabilities to columns

Selection

Single or multiple row selection

Pagination

Built-in pagination support

Props

Prop Type Default Description
value Array [] Array of objects to display
columns Array [] Column definitions
sortMode string 'single' Sorting mode: 'single' or 'multiple'
selectionMode string null Selection mode: 'single' or 'multiple'
selection any null Selected row(s)
paginator boolean false Whether to show paginator
rows number 0 Number of rows per page

Events

Event Parameters Description
onSort SortEvent Callback to invoke when a column is sorted
onFilter FilterEvent Callback to invoke when data is filtered
onRowSelect RowSelectEvent Callback to invoke when a row is selected
onRowUnselect RowUnselectEvent Callback to invoke when a row is unselected
onSelectionChange SelectionChangeEvent Callback to invoke when selection changes

Accessibility

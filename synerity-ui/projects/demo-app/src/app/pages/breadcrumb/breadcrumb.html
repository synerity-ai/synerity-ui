Breadcrumb

A navigation component that displays the current page's location within a hierarchical structure, allowing users to navigate back to previous levels.

Basic Breadcrumb

<sui-breadcrumb 
  [model]="basicModel"
  [home]="homeItem"
  (onItemClick)="onBreadcrumbClick($event)"
  (onHomeClick)="onHomeClick()"
></sui-breadcrumb>

Breadcrumb Variants

Installation

Import the Breadcrumb component in your module or standalone component:

import { Breadcrumb } from 'ui-lib';

Basic Usage

Use the Breadcrumb component to show navigation hierarchy:

<sui-breadcrumb 
  [model]="breadcrumbModel"
  [home]="homeItem"
  (onItemClick)="onBreadcrumbClick($event)"
  (onHomeClick)="onHomeClick()"
></sui-breadcrumb>

Styling Guidelines

  • Use clear, descriptive labels for each breadcrumb item
  • Include icons for better visual hierarchy and recognition
  • Ensure proper spacing and separators between items
  • Consider truncation for very long breadcrumb paths

Props

Property Type Default Description
model Array<BreadcrumbItem> [] Array of breadcrumb items
home BreadcrumbItem - Home item configuration
styleClass string - Custom CSS class for styling

Events

Event Type Description
onItemClick BreadcrumbItem Emitted when a breadcrumb item is clicked
onHomeClick void Emitted when the home item is clicked

Accessibility

  • Breadcrumb items are keyboard accessible with proper ARIA attributes
  • Screen readers announce the navigation hierarchy
  • Focus management ensures proper keyboard navigation
  • ARIA labels provide context for assistive technologies

Breadcrumb

A navigation component that shows the current page's location in a hierarchy.

Basic Breadcrumb

<sui-breadcrumb 
  [model]="simpleBreadcrumbItems"
  (onItemClick)="onBreadcrumbItemClick($event)">
</sui-breadcrumb>

Icon Breadcrumb

Breadcrumb with icons for better visual hierarchy.

<sui-breadcrumb 
  [model]="breadcrumbItems"
  (onItemClick)="onBreadcrumbItemClick($event)">
</sui-breadcrumb>

Installation

The Breadcrumb component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Breadcrumb component in your Angular application:

import { Breadcrumb } from '@synerity/ui';

@Component({
  template: `
    <sui-breadcrumb 
      [model]="breadcrumbItems"
      (onItemClick)="onItemClick($event)">
    </sui-breadcrumb>
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Breadcrumb uses Tailwind CSS classes for styling. Here are the key classes:

sui-breadcrumb Base class for the breadcrumb container
sui-breadcrumb-item Class for individual breadcrumb items
sui-breadcrumb-separator Class for the separator between items

Props

Prop Type Default Description
model Array<MenuItem> [] Array of breadcrumb items to display
style any {} Inline styles for the breadcrumb container
styleClass string '' Style class for the breadcrumb container

Events

Event Type Description
onItemClick EventEmitter<MenuItem> Emitted when a breadcrumb item is clicked

Breadcrumb

A navigation component that shows the current page's location within a hierarchical structure.

Basic Breadcrumb

A simple breadcrumb navigation with home icon and separators.

<sui-breadcrumb 
  [model]="breadcrumbItems"
  [home]="homeItem"
  (onItemClick)="onBreadcrumbClick($event)">
</sui-breadcrumb>

Breadcrumb with Icons

Breadcrumb items with custom icons for better visual hierarchy.

// Component data
iconBreadcrumb = [
  { label: 'Dashboard', icon: 'pi pi-home' },
  { label: 'Products', icon: 'pi pi-box' },
  { label: 'Electronics', icon: 'pi pi-mobile' }
];

<sui-breadcrumb 
  [model]="iconBreadcrumb"
  [home]="homeItem"
  (onItemClick)="onBreadcrumbClick($event)">
</sui-breadcrumb>

Breadcrumb with Disabled Items

Some breadcrumb items can be disabled to prevent navigation.

// Component data
disabledBreadcrumb = [
  { label: 'Dashboard', disabled: false },
  { label: 'Products', disabled: true },
  { label: 'Electronics', disabled: false }
];

<sui-breadcrumb 
  [model]="disabledBreadcrumb"
  [home]="homeItem"
  (onItemClick)="onBreadcrumbClick($event)">
</sui-breadcrumb>

Installation

The Breadcrumb component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Breadcrumb component in your Angular application:

import { Breadcrumb } from '@synerity/ui';

Then use it in your template:

<sui-breadcrumb [model]="breadcrumbItems" [home]="homeItem" (onItemClick)="onBreadcrumbClick($event)"> </sui-breadcrumb>

Data Structure

Define your breadcrumb items in your component:

// Component data breadcrumbItems = [ { label: 'Dashboard' }, { label: 'Products' }, { label: 'Electronics' } ]; homeItem = { label: 'Home', icon: 'pi pi-home', command: () => this.navigateToHome() };

Component Properties

Prop Type Default Description
model MenuItem[] [] Array of menu items for the breadcrumb
home MenuItem null Home menu item
style object null Inline style of the component
styleClass string null Style class of the component

Events

Event Description
onItemClick Emitted when a breadcrumb item is clicked

Overview

The Breadcrumb component provides a navigation aid that shows the current page's location within a hierarchical structure. It helps users understand where they are in the application and provides quick navigation to parent pages.

Breadcrumbs are essential for improving user experience in complex applications with deep navigation hierarchies.

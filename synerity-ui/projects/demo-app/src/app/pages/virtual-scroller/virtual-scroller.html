Installation

The VirtualScroller component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
value any[] [] Array of items to display
itemSize number 50 Height/width of each item in pixels
orientation 'vertical' | 'horizontal' 'vertical' Scroll orientation

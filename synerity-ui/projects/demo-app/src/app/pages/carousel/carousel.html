Carousel

A carousel component for displaying multiple images or content in a rotating slideshow.

Basic Carousel

A simple carousel with navigation and indicators.

<sui-carousel
  [value]="basicCarouselItems"
  [showIndicators]="true"
  [showNavigators]="true"
  [circular]="true"
  (onPageChange)="onCarouselPageChange($event)">
</sui-carousel>

Autoplay Carousel

Carousel with automatic rotation every 3 seconds.

<sui-carousel
  [value]="autoplayCarouselItems"
  [autoplayInterval]="3000"
  [showIndicators]="true"
  [showNavigators]="true"
  [circular]="true"
  (onPageChange)="onCarouselPageChange($event)">
</sui-carousel>

No Indicators Carousel

Carousel without page indicators.

<sui-carousel
  [value]="noIndicatorsCarouselItems"
  [showIndicators]="false"
  [showNavigators]="true"
  [circular]="true"
  (onPageChange)="onCarouselPageChange($event)">
</sui-carousel>

No Navigators Carousel

Carousel without navigation arrows.

<sui-carousel
  [value]="noNavigatorsCarouselItems"
  [showIndicators]="true"
  [showNavigators]="false"
  [circular]="true"
  (onPageChange)="onCarouselPageChange($event)">
</sui-carousel>

Getting Started

Import the Carousel component and add it to your template:

import {{ '{' }} Carousel {{ '}' }} from 'ui-lib';

<sui-carousel [value]="carouselItems"></sui-carousel>

Define your carousel items:

carouselItems = [
  {{ '{' }} src: 'image1.jpg', alt: 'Image 1', title: 'First Image' {{ '}' }},
  {{ '{' }} src: 'image2.jpg', alt: 'Image 2', title: 'Second Image' {{ '}' }}
];

API Reference

Properties

Property Type Default Description
value CarouselItem[] [] Array of carousel items
activeIndex number 0 Index of the active item
circular boolean true Whether to loop infinitely
autoplayInterval number 0 Autoplay interval in milliseconds
showIndicators boolean true Whether to show page indicators
showNavigators boolean true Whether to show navigation arrows

Events

Event Type Description
onPageChange number Emitted when the active page changes

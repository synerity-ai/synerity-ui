Carousel

A slideshow component for cycling through elements with navigation controls.

Basic Carousel

{{ item.header }}

{{ item.content }}

 
<sui-carousel 
  [value]="carouselItems"
  (onChange)="onCarouselChange($event)">
  <ng-template let-item pTemplate="item">
    <div class="text-center p-6">
      <img [src]="item.image" [alt]="item.header">
      <h3>Item Header</h3>
      <p>Item Content</p>
    </div>
  </ng-template>
</sui-carousel>

Installation

The Carousel component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Carousel component in your Angular application:

import { Carousel } from '@synerity/ui';

@Component({
  template: `
    <sui-carousel 
      [value]="carouselItems"
      (onChange)="onChange($event)">
      <ng-template let-item pTemplate="item">
        <div>Item Content</div>
      </ng-template>
    </sui-carousel>
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Carousel uses Tailwind CSS classes for styling. Here are the key classes:

sui-carousel Base class for the carousel container
sui-carousel-item Class for individual carousel items
sui-carousel-indicators Class for the carousel indicators

Props

Prop Type Default Description
value Array<any> [] Array of items to display in the carousel
autoplayInterval number 0 Time in milliseconds for automatic slide transitions
style any {} Inline styles for the carousel container
styleClass string '' Style class for the carousel container

Events

Event Type Description
onChange EventEmitter<any> Emitted when the active slide changes

Carousel

Image carousel component with navigation controls, indicators, and autoplay functionality.

Basic Carousel

Interactive: Use the navigation arrows or click the indicators to navigate between slides. Features captions and titles. 

<sui-carousel 
  [value]="carouselItems"
  [showIndicators]="true"
  [showNavigators]="true"
  [circular]="true"
  (onPageChange)="onCarouselChange($event)">
</sui-carousel>

// Component data
carouselItems = [
  {
    src: 'image1.jpg',
    alt: 'Mountain landscape',
    title: 'Mountain Adventure',
    caption: 'Explore breathtaking mountain landscapes'
  }
];

Simple Carousel

Minimal: Simple image carousel with indicators only. No navigation arrows or captions. 

<sui-carousel 
  [value]="simpleCarouselItems"
  [showIndicators]="true"
  [showNavigators]="false"
  [circular]="false">
</sui-carousel>

// Simple image data
simpleCarouselItems = [
  { src: 'image1.jpg', alt: 'Mountain view' },
  { src: 'image2.jpg', alt: 'Forest path' },
  { src: 'image3.jpg', alt: 'Ocean waves' }
];

Autoplay Carousel

Autoplay: Automatically advances slides every 3 seconds. Hover to pause autoplay. 

<sui-carousel 
  [value]="autoplayCarouselItems"
  [autoplayInterval]="3000"
  [showIndicators]="true"
  [showNavigators]="true"
  [circular]="true">
</sui-carousel>

// Autoplay every 3 seconds (3000ms)
autoplayInterval="3000"

Installation

The Carousel component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Carousel component in your Angular application:

import { Carousel } from '@synerity/ui';

@Component({
  template: `
    <sui-carousel [value]="carouselItems"></sui-carousel>
  `,
  data: {
    carouselItems: [
      { src: 'image1.jpg', alt: 'Image 1' },
      { src: 'image2.jpg', alt: 'Image 2' }
    ]
  }
})
export class MyComponent {}

Carousel Item Structure

Each carousel item can have the following properties:

src Required. Image source URL
alt Optional. Alternative text for accessibility
title Optional. Title displayed in caption
caption Optional. Caption text displayed below title

Configuration Options

Navigation Controls

<sui-carousel [showNavigators]="true">

Show/hide previous and next navigation arrows.

Indicators

<sui-carousel [showIndicators]="true">

Show/hide dot indicators for direct slide navigation.

Circular Navigation

<sui-carousel [circular]="true">

Enable continuous looping through slides.

Autoplay

<sui-carousel [autoplayInterval]="3000">

Auto-advance slides every specified milliseconds.

Event Handling

Handle carousel page changes using the onPageChange event:

<sui-carousel 
  [value]="carouselItems"
  (onPageChange)="onPageChange($event)">
</sui-carousel>

onPageChange(activeIndex: number) {
  console.log('Current slide:', activeIndex);
  console.log('Current item:', this.carouselItems[activeIndex]);
}

Props

Prop Type Default Description
value Array<CarouselItem> [] Array of carousel items to display
activeIndex number 0 Index of the currently active slide
circular boolean true Enable circular navigation
autoplayInterval number 0 Autoplay interval in milliseconds (0 to disable)
showIndicators boolean true Show slide indicators
showNavigators boolean true Show navigation arrows
style any {} Inline styles for the carousel
styleClass string '' CSS class name for custom styling

Events

Event Type Description
onPageChange EventEmitter<number> Emitted when the active slide changes

CarouselItem Interface

interface CarouselItem {
  src: string;          // Image source URL (required)
  alt?: string;         // Alternative text for accessibility
  title?: string;       // Title displayed in caption
  caption?: string;     // Caption text displayed below title
}

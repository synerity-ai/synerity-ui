How to Use TreeTable

Installation

Import the TreeTable component from the UI library:

import { TreeTable } from 'ui-lib';

// In your component imports
imports: [TreeTable]

Basic Usage

Use the component with hierarchical table data:

<sui-tree-table 
  [value]="treeTableData"
  [columns]="columns"
  (onNodeExpand)="onNodeExpand($event)"
  (onNodeCollapse)="onNodeCollapse($event)"
  (onSort)="onSort($event)"
></sui-tree-table>

Component Properties

  • value: Array of hierarchical node objects
  • columns: Array of column definitions
  • selectionMode: Selection mode ('single', 'multiple', 'checkbox', null)
  • metaKeySelection: Boolean for meta key selection behavior
  • propagateSelectionUp: Boolean to propagate selection to parent nodes
  • propagateSelectionDown: Boolean to propagate selection to child nodes
  • sortField: Current sort field
  • sortOrder: Sort order ('asc' or 'desc')
  • emptyMessage: Message to display when no records found

Events

  • onNodeSelect: Emitted when a node is selected
  • onNodeUnselect: Emitted when a node is unselected
  • onNodeExpand: Emitted when a node is expanded
  • onNodeCollapse: Emitted when a node is collapsed
  • onSort: Emitted when column sorting changes
  • onSelectionChange: Emitted when selection changes

columns = [

Features

  • Hierarchical Display: Tree structure with expandable rows
  • Column Sorting: Sort by any column (if enabled)
  • Selection Modes: Single, multiple, or checkbox selection
  • Responsive Design: Horizontal scrolling for wide tables
  • Custom Styling: Flexible styling options

API Documentation

Properties

Property Type Default Description
value Array<TreeNode> [] Array of tree nodes
columns Array<Column> [] Column definitions
selectionMode 'single' | 'multiple' | 'checkbox' | null null Selection mode
metaKeySelection boolean true Meta key selection behavior
propagateSelectionUp boolean true Propagate selection to parent nodes
propagateSelectionDown boolean true Propagate selection to child nodes
sortField string '' Current sort field
sortOrder 'asc' | 'desc' 'asc' Sort order
emptyMessage string 'No records found' Empty state message

Events

Event Type Description
onNodeSelect EventEmitter<TreeNode> Fired when a node is selected
onNodeUnselect EventEmitter<TreeNode> Fired when a node is unselected
onNodeExpand EventEmitter<TreeNode> Fired when a node is expanded
onNodeCollapse EventEmitter<TreeNode> Fired when a node is collapsed
onSort EventEmitter<{field: string, order: string}> Fired when column sorting changes
onSelectionChange EventEmitter<TreeNode[]> Fired when selection changes

Interfaces

TreeNode

interface TreeNode {
  key: string;
  data: any;
  children?: TreeNode[];
  expanded?: boolean;
  selectable?: boolean;
}

Column

interface Column {
  field: string;
  header: string;
  sortable?: boolean;
  filterable?: boolean;
}

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Arrow keys for tree navigation, Tab for table navigation
  • Focus Management: Proper focus handling and visual indicators
  • Screen Reader: Descriptive labels for all cells and headers

Styling

The component uses CSS classes that can be customized:

  • .sui-tree-table: Main container
  • .sui-tree-table-header: Table header
  • .sui-tree-table-body: Table body
  • .sui-tree-table-row: Table row
  • .sui-tree-table-cell: Table cell
  • .sui-tree-table-node-selected: Selected node styling
  • .sui-tree-table-node-expanded: Expanded node styling

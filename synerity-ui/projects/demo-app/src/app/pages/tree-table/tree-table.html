TreeTable

Hierarchical table component that combines tree structure with tabular data display and sorting capabilities.

Basic TreeTable

<sui-tree-table 
  [value]="treeTableData"
  [columns]="columns"
  [emptyMessage]="'No data available'"
  (onNodeSelect)="onNodeSelect($event)"
  (onNodeExpand)="onNodeExpand($event)"
  (onSort)="onSort($event)">
</sui-tree-table>

TreeTable with Checkbox Selection

Selected nodes: {{ selectedNodes.length }} 
<sui-tree-table 
  [value]="treeTableData"
  [columns]="columns"
  selectionMode="checkbox"
  (onNodeSelect)="onNodeSelect($event)"
  (onSelectionChange)="onSelectionChange($event)">
</sui-tree-table>

Installation

The TreeTable component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
value TreeNode[] [] Array of tree table nodes
columns Array<{{ '{' }} field: string; header: string; sortable?: boolean {{ '}' }}> [] Array of column definitions
selectionMode 'single' | 'multiple' | 'checkbox' | null null Selection mode for nodes

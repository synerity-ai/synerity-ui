Dynamic Dialog

A dialog component that can be created and managed programmatically with dynamic content.

Basic Dynamic Dialog

A dynamic dialog that can be created programmatically with custom content.

// Component method
openDynamicDialog(): void {
  this.dynamicDialog.open(MyComponent, {
    header: 'Dynamic Dialog',
    width: '400px',
    height: '300px'
  });
}

Draggable Dynamic Dialog

A dynamic dialog that can be dragged around the screen.

// Component method
openDraggableDialog(): void {
  this.dynamicDialog.open(MyComponent, {
    header: 'Draggable Dialog',
    draggable: true,
    width: '500px',
    height: '400px'
  });
}

Custom Size Dynamic Dialog

A dynamic dialog with custom dimensions and positioning.

// Component method
openCustomSizeDialog(): void {
  this.dynamicDialog.open(MyComponent, {
    header: 'Custom Size Dialog',
    width: '600px',
    height: '500px',
    position: 'top-right',
    resizable: true
  });
}

Installation

The DynamicDialog component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the DynamicDialog component in your Angular application:

import { DynamicDialog } from '@synerity/ui';

Then use it in your template:

// Component method openDynamicDialog(): void { this.dynamicDialog.open(MyComponent, { header: 'Dynamic Dialog', width: '400px', height: '300px' }); }

Advanced Features

Draggable Dialog

Enable dragging by setting the draggable property to true:

this.dynamicDialog.open(MyComponent, { header: 'Draggable Dialog', draggable: true, width: '500px', height: '400px' });

Custom Size

Set custom dimensions and positioning:

this.dynamicDialog.open(MyComponent, { header: 'Custom Size Dialog', width: '600px', height: '500px', position: 'top-right', resizable: true });

Component Properties

Prop Type Default Description
component Type<any> - Component to display in the dialog
header string '' Header text of the dialog
width string 'auto' Width of the dialog
height string 'auto' Height of the dialog
draggable boolean true Whether the dialog can be dragged
resizable boolean false Whether the dialog can be resized

Events

Event Description
onShow Emitted when the dialog is shown
onHide Emitted when the dialog is hidden

Overview

The DynamicDialog component allows you to create and manage dialogs programmatically. Unlike regular dialogs, dynamic dialogs can be created at runtime with any component as content, making them perfect for complex workflows and dynamic content.

Dynamic dialogs are managed through a service that handles their lifecycle, positioning, and interaction with the underlying content components.

Dynamic Dialog

Programmatically created dialogs with dynamic content, dragging capabilities, and customizable behavior.

Basic Dynamic Dialog

<sui-dynamic-dialog
  [visible]="basicDialogVisible"
  [header]="basicDialogConfig.header"
  [modal]="basicDialogConfig.modal"
  [closable]="basicDialogConfig.closable"
  [draggable]="basicDialogConfig.draggable"
  [width]="basicDialogConfig.width"
  [height]="basicDialogConfig.height"
  (onShow)="onDialogShow()"
  (onHide)="onDialogHide()">
  
  <div class="p-6">
    <h3>Dynamic Dialog Content</h3>
    <p>This dialog was created dynamically.</p>
  </div>
</sui-dynamic-dialog>

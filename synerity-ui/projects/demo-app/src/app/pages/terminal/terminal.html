Terminal

A terminal emulator component for displaying command-line interfaces with interactive command execution.

Basic Terminal

Basic terminal with default settings and welcome message.

Interactive: Type commands like 'help', 'date', 'echo hello', or 'clear' to see the terminal in action. 

<sui-terminal 
  [welcomeMessage]="'Welcome to Synerity Terminal Demo'"
  [prompt]="'$'"
  (onCommand)="onTerminalCommand($event)"
></sui-terminal>

Custom Styled Terminal

Terminal with custom styling and appearance.

Custom Styling: Apply custom colors, fonts, and styling to match your application theme. 

<sui-terminal 
  welcomeMessage="Custom Terminal Interface"
  prompt=">"
  styleClass="custom-terminal"
  [style]="{'background-color': '#1a1a1a', 'color': '#00ff00', 'border': '2px solid #333'}"
  (onCommand)="onTerminalCommand($event)"
></sui-terminal>

Installation

npm install ui-lib

Basic Usage

Import the Terminal component and use it in your template:

import { Terminal } from 'ui-lib';

// In your template
<sui-terminal 
  [welcomeMessage]="'Welcome to Terminal'"
  [prompt]="'$'"
  (onCommand)="onTerminalCommand($event)"
></sui-terminal>

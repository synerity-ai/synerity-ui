Getting Started

Import the Terminal component and add it to your template:

import { Terminal } from 'ui-lib';

<sui-terminal [welcomeMessage]="Welcome to Terminal" [prompt]="$">
</sui-terminal>

Basic terminal usage:

<sui-terminal
  [welcomeMessage]="Welcome to My Terminal"
  [prompt]="user@host:~$"
  [style]="{ backgroundColor: '#1a1a1a', color: '#00ff00' }"
  [styleClass]="custom-terminal"
  (onCommand)="onCommand($event)">
</sui-terminal>

API Reference

Properties

Property Type Default Description
welcomeMessage string 'Welcome to Terminal' Welcome message displayed when terminal starts
prompt string '$' Command prompt symbol
style object {} Inline styles for the terminal
styleClass string '' CSS class for the terminal

Events

Event Type Description
onCommand EventEmitter<string> Emitted when a command is executed

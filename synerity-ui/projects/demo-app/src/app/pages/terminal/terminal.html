Terminal

A terminal component for command-line interface interactions.

Basic Usage

<sui-terminal 
  [welcomeMessage]="terminalWelcomeMessage"
  [prompt]="terminalPrompt"
  (onCommand)="onTerminalCommand($event)"
></sui-terminal>

Custom Styled Terminal

<sui-terminal 
  [welcomeMessage]="'Custom Terminal'"
  [prompt]="'❯'"
  [style]="{
    backgroundColor: '#1a1a1a',
    color: '#00ff00',
    fontFamily: 'Courier New, monospace',
    height: '300px'
  }"
  (onCommand)="onTerminalCommand($event)"
></sui-terminal>

Interactive Terminal

<sui-terminal 
  [welcomeMessage]="'Interactive Terminal - Try typing commands!'"
  [prompt]="'user@demo:~$'"
  [style]="{
    height: '350px'
  }"
  (onCommand)="onTerminalCommand($event)"
></sui-terminal>

Block UI

Component that blocks user interaction with a loading overlay, perfect for preventing user actions during async operations.

Basic Block UI

Click the button below to toggle the block UI overlay.

Content Area

This content will be blocked when the overlay is active.

Interactive: The overlay blocks all user interaction with the content area. 

<sui-block-ui [blocked]="isBlocked">
  <div class="p-6 bg-gray-50 rounded-lg">
    <h4 class="text-lg font-semibold mb-3">Content Area</h4>
    <p class="text-gray-600 mb-4">This content will be blocked when the overlay is active.</p>
    <button (click)="toggleBlock()">{{ isBlocked ? 'Unblock' : 'Block' }} UI</button>
  </div>
</sui-block-ui>

Custom Message

Click the button to simulate a loading operation with a custom message.

Form Area

Custom Message: Display a custom loading message to inform users about the operation. 

<sui-block-ui [blocked]="isBlocked" [message]="customMessage">
  <div class="p-6 bg-gray-50 rounded-lg">
    <h4 class="text-lg font-semibold mb-3">Form Area</h4>
    <input type="text" placeholder="Name" class="w-full px-3 py-2 border border-gray-300 rounded-md">
    <button (click)="simulateLoading()">Submit Form</button>
  </div>
</sui-block-ui>

Installation

The Block UI component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Block UI component in your Angular application:

import { BlockUi } from '@synerity/ui';

@Component({
  selector: 'app-example',
  imports: [BlockUi],
  template: `
    <sui-block-ui [blocked]="isLoading">
      <div>Your content here</div>
    </sui-block-ui>
  `
})
export class ExampleComponent {
  isLoading = false;
}

Props

Property Type Default Description
blocked boolean false Whether the UI should be blocked
message string 'Loading...' Message to display in the overlay
showSpinner boolean true Whether to show the loading spinner

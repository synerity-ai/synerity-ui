BlockUI

A component that blocks user interaction with an overlay and optional loading message.

Getting Started

Import the BlockUI component and add it to your template:

import { BlockUi } from 'ui-lib';

<sui-block-ui [blocked]="isBlocked" [message]="'Loading...'">
  <div>Content to block</div>
</sui-block-ui>

Basic block UI usage:

<sui-block-ui
  [blocked]="isBlocked"
  [message]="'Loading...'"
  [showSpinner]="true">
  <div class="p-6">
    <h4>Content Area</h4>
    <p>This content will be blocked when the overlay is active.</p>
    <button>Click me</button>
  </div>
</sui-block-ui>

API Reference

Properties

Property Type Default Description
blocked boolean false Whether to block the UI
message string 'Loading...' Message to display in the overlay
showSpinner boolean true Whether to show the loading spinner
overlayStyle any {} Custom styles for the overlay
overlayClass string '' Custom CSS classes for the overlay

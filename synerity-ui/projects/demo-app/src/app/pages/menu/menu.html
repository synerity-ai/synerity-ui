Installation

The Menu component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Menu component in your Angular application:

import { Menu } from '@synerity/ui';

@Component({
  template: `
    <sui-menu [model]="menuItems"></sui-menu>
  `
})
export class MyComponent {}

Menu Features

Hierarchical Structure

Support for nested menu items and submenus

Icons and Labels

Rich menu items with icons and descriptive labels

Separators

Visual separators to group related menu items

Keyboard Navigation

Full keyboard support for accessibility

Props

Prop Type Default Description
model MenuItem[] [] Array of menu items
popup boolean false Whether the menu is a popup
style any {} Inline styles for the menu
styleClass string '' CSS class for the menu

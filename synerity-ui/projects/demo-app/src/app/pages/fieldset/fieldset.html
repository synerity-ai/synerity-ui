Fieldset

Form field grouping component with collapsible functionality.

Basic Fieldset

 
<sui-fieldset legend="Personal Information">
  <div class="space-y-4">
    <div>
      <label class="block text-sm font-medium text-gray-700 mb-1">First Name</label>
      <input type="text" class="w-full px-3 py-2 border border-gray-300 rounded-md">
    </div>
    <div>
      <label class="block text-sm font-medium text-gray-700 mb-1">Last Name</label>
      <input type="text" class="w-full px-3 py-2 border border-gray-300 rounded-md">
    </div>
  </div>
</sui-fieldset>

Collapsible Fieldset

 
<sui-fieldset 
  legend="Contact Information"
  [toggleable]="true"
  [collapsed]="fieldsetCollapsed"
  (onToggle)="toggleFieldset()">
  <div class="space-y-4">
    <!-- Form fields -->
  </div>
</sui-fieldset>

Installation

The Fieldset component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Fieldset component in your Angular application:

import { Fieldset } from '@synerity/ui';

@Component({
  template: `
    <sui-fieldset legend="Form Section">
      <!-- Form fields go here -->
    </sui-fieldset>
  `
})
export class MyComponent {}

Fieldset Features

Legend

Descriptive title for the fieldset group

Collapsible

Toggle fieldset visibility with smooth animations

Form Grouping

Organize related form fields logically

Accessibility

Proper ARIA attributes and keyboard navigation

Props

Prop Type Default Description
legend string '' Fieldset legend text
toggleable boolean false Whether the fieldset can be collapsed
collapsed boolean false Whether the fieldset is collapsed
style any {} Inline styles for the fieldset
styleClass string '' CSS class for the fieldset

Events

Event Type Description
onToggle EventEmitter<any> Emitted when fieldset is toggled

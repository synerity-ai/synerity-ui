Fieldset Component

Form field grouping component that organizes related form elements with optional collapsible functionality.

Basic Fieldset

A basic fieldset that groups related form elements with a legend. 

<sui-fieldset [legend]="basicLegend">
  <div class="space-y-4">
    <div>
      <label class="block text-sm font-medium text-gray-700 mb-1">Name</label>
      <input type="text" class="w-full px-3 py-2 border border-gray-300 rounded-md">
    </div>
    <!-- More form fields -->
  </div>
</sui-fieldset>

Toggleable Fieldset

A toggleable fieldset that can be collapsed and expanded by clicking the legend. 

<sui-fieldset 
  [legend]="toggleableLegend"
  [toggleable]="true"
  [collapsed]="false"
  (onToggle)="onFieldsetToggle($event)"
>
  <div class="space-y-4">
    <!-- Form fields -->
  </div>
</sui-fieldset>

Custom Styled Fieldset

A fieldset with custom styling using styleClass and inline styles. 

<sui-fieldset 
  [legend]="customLegend"
  [toggleable]="true"
  styleClass="custom-fieldset"
  [style]="{ borderColor: '#3b82f6', borderRadius: '8px' }"
>
  <div class="space-y-4">
    <!-- Form fields -->
  </div>
</sui-fieldset>

Installation

npm install @synerity/ui

Basic Usage

The Fieldset component groups related form elements with a legend and optional collapsible functionality. 

<sui-fieldset [legend]="'Personal Information'">
  <div class="space-y-4">
    <!-- Form fields go here -->
  </div>
</sui-fieldset>

Toggleable Fieldset

Make the fieldset collapsible by setting the toggleable property to true: 

<sui-fieldset 
  [legend]="'Advanced Options'"
  [toggleable]="true"
  [collapsed]="false"
  (onToggle)="onFieldsetToggle($event)"
>
  <!-- Collapsible content -->
</sui-fieldset>

Use Cases

  • Form Organization: Group related form fields logically
  • Settings Panels: Organize configuration options
  • User Profiles: Separate personal and professional information
  • Advanced Options: Collapsible sections for optional settings
  • Multi-step Forms: Organize form steps into sections

Props

Property Type Default Description
legend string '' Legend text for the fieldset
toggleable boolean false Whether the fieldset can be collapsed/expanded
collapsed boolean false Initial collapsed state
style any {{ '{}' }} Inline styles for the fieldset
styleClass string '' CSS class for custom styling

Events

Event Type Description
onToggle any Emitted when the fieldset is toggled (collapsed/expanded)

Accessibility

  • Semantic HTML with proper fieldset and legend elements
  • Keyboard navigation support for toggleable fieldsets
  • Screen reader friendly with appropriate ARIA attributes
  • Focus management for collapsible content

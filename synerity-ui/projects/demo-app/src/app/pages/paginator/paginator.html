Paginator

Navigation component for paginating through large datasets with customizable page sizes and controls.

Basic Paginator

<sui-paginator 
  [rows]="10"
  [totalRecords]="100"
  [showCurrentPageReport]="true"
  (onPageChange)="onPageChange($event)">
</sui-paginator>

Paginator with Custom Options

<sui-paginator 
  [rows]="5"
  [totalRecords]="50"
  [rowsPerPageOptions]="[5, 10, 25, 50]"
  [showCurrentPageReport]="true"
  [showFirstLastIcon]="true"
  [showPageLinks]="true"
  [showRowsPerPageOptions]="true"
  (onPageChange)="onPageChange($event)">
</sui-paginator>

Installation

The Paginator component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
rows number 10 Number of rows per page
totalRecords number 0 Total number of records
showCurrentPageReport boolean false Show current page information

Paginator

Pagination components for navigating through large datasets with customizable page sizes and navigation controls.

Basic Paginator

Sample Data ({{ totalRecords }} total records)

{{ item.name }}
{{ item.category }} - ${{ item.price }}
<sui-paginator 
  [totalRecords]="totalRecords"
  [rows]="rows"
  [rowsPerPageOptions]="rowsPerPageOptions"
  (onPageChange)="onPageChange($event)">
</sui-paginator>

Custom Options

<sui-paginator 
  [totalRecords]="totalRecords"
  [rows]="25"
  [rowsPerPageOptions]="customRowsPerPageOptions"
  [pageLinkSize]="3"
  (onPageChange)="onCustomPageChange($event)">
</sui-paginator>

Installation

npm install @synerity/ui-lib

Import

import { Paginator } from '@synerity/ui-lib';

@Component({
  imports: [Paginator],
  // ...
})

Basic Usage

<sui-paginator 
  [totalRecords]="1000"
  [rows]="10"
  (onPageChange)="onPageChange($event)">
</sui-paginator>

Props

Name Type Default Description
totalRecords number 0 Total number of records
rows number 10 Number of rows per page
rowsPerPageOptions number[] [5, 10, 20, 30] Array of page size options

Events

Name Parameters Description
onPageChange { first: number, rows: number, page: number, pageCount: number } Fired when page changes

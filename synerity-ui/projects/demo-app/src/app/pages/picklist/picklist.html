PickList

A dual list component for transferring items between two lists with filtering and drag & drop support.

Basic PickList

<sui-pick-list 
  [source]="sourceItems"
  [target]="targetItems"
  sourceHeader="Available Items"
  targetHeader="Selected Items"
  (onMoveToTarget)="onMoveToTarget($event)"
  (onMoveToSource)="onMoveToSource($event)"
  (onMoveAllToTarget)="onMoveAllToTarget($event)"
  (onMoveAllToSource)="onMoveAllToSource($event)"
></sui-pick-list>

PickList with Drag & Drop

<sui-pick-list 
  [source]="dragDropSourceItems"
  [target]="dragDropTargetItems"
  sourceHeader="Available Items"
  targetHeader="Selected Items"
  [dragdrop]="true"
  (onMoveToTarget)="onMoveToTarget($event)"
  (onMoveToSource)="onMoveToSource($event)"
  (onMoveAllToTarget)="onMoveAllToTarget($event)"
  (onMoveAllToSource)="onMoveAllToSource($event)"
></sui-pick-list>

Custom PickList

<sui-pick-list 
  [source]="customSourceItems"
  [target]="customTargetItems"
  sourceHeader="Available Products"
  targetHeader="Selected Products"
  filterPlaceholder="Search products..."
  (onMoveToTarget)="onMoveToTarget($event)"

></sui-pick-list>

How to Use PickList

Installation

Import the PickList component from the UI library:

import { PickList } from 'ui-lib';

// In your component imports
imports: [PickList]

Basic Usage

Use the component with source and target arrays:

<sui-pick-list 
  [source]="availableItems"
  [target]="selectedItems"
  sourceHeader="Available"
  targetHeader="Selected"
  (onMoveToTarget)="onMoveToTarget($event)"
  (onMoveToSource)="onMoveToSource($event)"
></sui-pick-list>

Component Properties

  • source: Array of items in the source list
  • target: Array of items in the target list
  • sourceHeader: Header text for the source list
  • targetHeader: Header text for the target list
  • filter: Boolean to enable filtering (default: true)
  • filterPlaceholder: Placeholder text for filter inputs
  • dragdrop: Boolean to enable drag & drop functionality
  • showSourceControls: Boolean to show/hide source controls
  • showTargetControls: Boolean to show/hide target controls

Events

  • onMoveToTarget: Emitted when items are moved to target
  • onMoveToSource: Emitted when items are moved to source
  • onMoveAllToTarget: Emitted when all items are moved to target
  • onMoveAllToSource: Emitted when all items are moved to source

Features

  • Dual List Transfer: Move items between two lists
  • Bulk Operations: Move all items at once
  • Filtering: Search and filter items in both lists
  • Drag & Drop: Intuitive drag and drop functionality
  • Selection: Select multiple items for transfer
  • Custom Headers: Customizable list headers

Data Structure

Items can be any objects:

sourceItems = [
  { id: 1, name: 'Item 1', category: 'Category A' },
  { id: 2, name: 'Item 2', category: 'Category B' }
];

targetItems = [
  { id: 3, name: 'Selected Item', category: 'Selected' }
];

API Documentation

Properties

Property Type Default Description
source any[] [] Array of source items
target any[] [] Array of target items
sourceHeader string 'Available' Header text for source list
targetHeader string 'Selected' Header text for target list
filter boolean true Enable filtering
filterPlaceholder string 'Search' Filter input placeholder
dragdrop boolean false Enable drag & drop
showSourceControls boolean true Show source list controls
showTargetControls boolean true Show target list controls

Events

Event Type Description
onMoveToTarget EventEmitter<{items: any[]}> Fired when items are moved to target
onMoveToSource EventEmitter<{items: any[]}> Fired when items are moved to source
onMoveAllToTarget EventEmitter<{items: any[]}> Fired when all items are moved to target
onMoveAllToSource EventEmitter<{items: any[]}> Fired when all items are moved to source

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Full keyboard support for item selection and transfer
  • Focus Management: Proper focus handling and visual indicators
  • Screen Reader: Descriptive labels for all actions and states

Styling

The component uses CSS classes that can be customized:

  • .sui-pick-list: Main container
  • .sui-pick-list-source: Source list container
  • .sui-pick-list-target: Target list container
  • .sui-pick-list-header: List header styling
  • .sui-pick-list-filter: Filter input styling
  • .sui-pick-list-items: Items container
  • .sui-pick-list-item: Individual item styling
  • .sui-pick-list-item-selected: Selected item styling
  • .sui-pick-list-controls: Transfer controls styling

PickList

Dual list component for transferring items between source and target lists with filtering and selection capabilities.

Basic PickList

<sui-pick-list 
  [source]="sourceItems"
  [target]="targetItems"
  sourceHeader="Available Items"
  targetHeader="Selected Items"
  [filter]="true"
  filterPlaceholder="Search items..."
  (onMoveToTarget)="onMoveToTarget($event)"
  (onMoveToSource)="onMoveToSource($event)">
</sui-pick-list>

PickList with Custom Headers

<sui-pick-list 
  [source]="sourceItems"
  [target]="targetItems"
  sourceHeader="All Permissions"
  targetHeader="User Permissions"
  [filter]="true"
  filterPlaceholder="Search permissions..."
  (onMoveToTarget)="onMoveToTarget($event)"
  (onMoveToSource)="onMoveToSource($event)">
</sui-pick-list>

Installation

The PickList component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
source any[] [] Array of source items
target any[] [] Array of target items
sourceHeader string 'Available' Header text for source list
targetHeader string 'Selected' Header text for target list
filter boolean true Enable search/filter functionality

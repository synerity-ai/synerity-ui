StyleClass

A utility component for dynamically adding, removing, and toggling CSS classes on elements.

Basic Class Toggle

Toggle CSS classes on elements using the StyleClass component.

{{ isActive ? 'Active State' : 'Inactive State' }}

Toggle: Click the button to toggle the active state and see the class changes. 

<button (click)="toggleActive()">Toggle Active</button>
<sui-style-class 
  selector=".demo-element-1"
  [toggleClass]="isActive ? 'active-state' : ''"
></sui-style-class>
<div 
  class="demo-element-1 w-16 h-16 bg-gray-200 rounded-lg transition-all duration-300"
></div>

Multiple Class Toggle

Toggle multiple CSS classes simultaneously on different elements.

{{ isHighlighted ? 'Highlighted' : 'Normal' }}

Multiple Classes: Toggle multiple elements with the same class changes. 

<button (click)="toggleHighlight()">Toggle Highlight</button>
<sui-style-class 
  selector=".demo-element-2"
  [toggleClass]="isHighlighted ? 'highlighted-state' : ''"
></sui-style-class>
<div 
  class="demo-element-2 w-32 h-8 bg-gray-200 rounded transition-all duration-300"
></div>
<div 
  class="demo-element-2 w-24 h-8 bg-gray-200 rounded transition-all duration-300"
></div>

Installation

npm install ui-lib

Basic Usage

Import the StyleClass component and use it in your template:

import { StyleClass } from 'ui-lib';

// In your template
<sui-style-class 
  selector=".my-element"
  toggleClass="active"
  (click)="toggle()"
></sui-style-class>

Common Use Cases

Dynamic Styling

Add or remove CSS classes based on user interactions or application state

Animation Control

Toggle animation classes for smooth transitions and effects

State Management

Manage visual states like active, disabled, or selected

Theme Switching

Switch between different themes or color schemes

Properties

Property Type Default Description
selector string 'body' CSS selector for the target element
enterClass string '' CSS class to add on enter
leaveClass string '' CSS class to add on leave
toggleClass string '' CSS class to toggle
target ElementRef | string 'body' Target element reference or selector
style any {} Inline styles for the component
styleClass string '' CSS class for the component

Methods

addClass(className: string)

Add a CSS class to the target element

removeClass(className: string)

Remove a CSS class from the target element

toggleClassMethod(className: string)

Toggle a CSS class on the target element

enter()

Apply the enter class to the target element

leave()

Apply the leave class to the target element

toggle()

Toggle the toggle class on the target element

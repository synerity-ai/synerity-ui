StyleClass

A utility component for managing CSS classes on DOM elements.

Basic StyleClass

Basic style class management with enter/leave animations.

Active: {{ basicActive ? 'Yes' : 'No' }}

<sui-style-class
  [target]="basicTarget"
  [enterClass]="basicEnterClass"
  [leaveClass]="basicLeaveClass"
  [toggleClass]="basicToggleClass">
</sui-style-class>

Getting Started

Import the StyleClass component and add it to your template:

import { StyleClass } from 'ui-lib';

<div #target class="my-element"></div>
<sui-style-class [target]="target" [toggleClass]="active">
</sui-style-class>

Basic style class usage:

<div #target class="w-20 h-20 bg-blue-500 rounded-lg"></div>
<sui-style-class
  [target]="target"
  [enterClass]="fade-in"
  [leaveClass]="fade-out"
  [toggleClass]="active"
  [style]="{ transition: 'all 0.3s' }">
</sui-style-class>

API Reference

Properties

Property Type Default Description
selector string 'body' CSS selector for the target element
enterClass string '' CSS class to add on enter
leaveClass string '' CSS class to add on leave
toggleClass string '' CSS class to toggle
target ElementRef | string 'body' Target element reference or selector
style object {} Inline styles for the component
styleClass string '' CSS class for the component

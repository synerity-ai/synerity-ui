MeterGroup

A component that displays progress meters with customizable values, colors, and sizes.

Basic MeterGroup

A basic meter group showing progress value.

Value: {{ basicValue[0].value }}/{{ basicMax }} 
<sui-meter-group
  [value]="basicValue"
  [min]="basicMin"
  [max]="basicMax"
  label="Basic Meter">
</sui-meter-group>

Custom MeterGroup

Custom meter group with different styling and configuration.

Value: {{ customValue[0].value }}/{{ customMax }} 
<sui-meter-group
  [value]="customValue"
  [min]="customMin"
  [max]="customMax"
  [label]="customLabel"
  color="#10b981"
  size="large">
</sui-meter-group>

Getting Started

Import the MeterGroup component and add it to your template:

import { MeterGroup } from 'ui-lib';

<sui-meter-group [value]="75" [min]="0" [max]="100" label="Progress">
</sui-meter-group>

Basic meter group usage:

<sui-meter-group
  [value]="progressValue"
  [min]="0"
  [max]="100"
  label="Task Progress"
  color="#3b82f6"
  size="medium">
</sui-meter-group>

API Reference

Properties

Property Type Default Description
value number 0 Current value of the meter
min number 0 Minimum value of the meter
max number 100 Maximum value of the meter
label string '' Label text for the meter
color string '#3b82f6' Color of the meter bar
size string 'medium' Size of the meter (small, medium, large)

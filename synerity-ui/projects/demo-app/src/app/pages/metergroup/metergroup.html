Meter Group

A meter group component for displaying multiple meter values in a horizontal or vertical layout.

Basic Meter Group

A basic meter group showing system resource usage.

Basic: Horizontal meter group with labels and values displayed. 

<sui-meter-group 
  [value]="basicMeterData" 
  [showValue]="true" 
  [showLabel]="true"
  orientation="horizontal">
</sui-meter-group>

Horizontal Meter Group

Horizontal meter group showing department performance metrics.

Horizontal: Meter group displayed in horizontal orientation with custom colors. 

<sui-meter-group 
  [value]="horizontalMeterData" 
  [showValue]="true" 
  [showLabel]="true"
  orientation="horizontal">
</sui-meter-group>

Vertical Meter Group

Vertical meter group showing quarterly performance data.

Vertical: Meter group displayed in vertical orientation for compact layouts. 

<div class="h-64">
  <sui-meter-group 
    [value]="verticalMeterData" 
    [showValue]="true" 
    [showLabel]="true"
    orientation="vertical">
  </sui-meter-group>
</div>

Installation

npm install ui-lib

Basic Usage

Import the MeterGroup component and use it in your template:

import {{ '{' }} MeterGroup {{ '}' }} from 'ui-lib';

// In your component
meterData = [
  {{ '{' }} label: 'CPU', value: 75, color: '#3b82f6' {{ '}' }},
  {{ '{' }} label: 'Memory', value: 60, color: '#10b981' {{ '}' }}
];

// In your template
<sui-meter-group [value]="meterData"></sui-meter-group>

Properties

Property Type Default Description
value MeterItem[] [] Array of meter items to display
min number 0 Minimum value for the meter
max number 100 Maximum value for the meter
showValue boolean true Whether to show the value text
showLabel boolean true Whether to show the label text
orientation 'horizontal' | 'vertical' 'horizontal' Orientation of the meter group
style any {} Inline styles for the component
styleClass string '' CSS class for the component

MeterItem Interface

Each meter item should have the following structure:

interface MeterItem {{ '{' }}
  label?: string;    // Optional label for the meter
  value: number;     // Required value for the meter
  color?: string;    // Optional custom color
  icon?: string;     // Optional icon
{{ '}' }}

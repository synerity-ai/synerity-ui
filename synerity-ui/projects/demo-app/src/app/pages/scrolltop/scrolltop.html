How to Use Scroll Top

Installation

Import the ScrollTop component from the UI library:

import { ScrollTop } from 'ui-lib';

// In your component imports
imports: [ScrollTop]

Basic Usage

Add the component to your template:

<sui-scroll-top></sui-scroll-top>

Component Properties

  • threshold: Scroll position in pixels when button becomes visible
  • icon: Icon or text to display on the button
  • behavior: Scroll behavior ('smooth' or 'auto')
  • style: Custom CSS styles
  • styleClass: Custom CSS class

Common Use Cases

  • Long Pages: Help users quickly return to top of long content
  • Blog Posts: Provide easy navigation for long articles
  • Product Pages: Allow quick return to product details
  • Documentation: Improve navigation in long documentation
  • E-commerce: Help users return to top of product listings

Best Practices

  • Appropriate Threshold: Set threshold based on content length (200-500px typical)
  • Clear Icon: Use intuitive icons like ‚Üë, ‚¨ÜÔ∏è, or üîù
  • Smooth Scrolling: Use 'smooth' behavior for better UX
  • Positioning: Place button in bottom-right corner
  • Accessibility: Ensure button is accessible via keyboard

Positioning

The component is positioned fixed to the bottom-right of the viewport by default:

.sui-scroll-top {
  position: fixed;
  bottom: 20px;
  right: 20px;
  z-index: 1000;
}

API Documentation

Properties

Property Type Default Description
threshold number 400 Scroll position when button becomes visible
icon string '‚Üë' Icon or text displayed on button
behavior 'smooth' | 'auto' 'smooth' Scroll animation behavior
style any {} Custom CSS styles
styleClass string '' Custom CSS class

Methods

  • scrollToTop(): Scrolls to the top of the page
  • getScrollTopClass(): Returns CSS classes for the button
  • getScrollTopStyle(): Returns inline styles for the button
  • isVisible(): Checks if button should be visible
  • getIcon(): Returns the icon to display

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Accessible via Tab key
  • Screen Reader: Descriptive labels and announcements
  • Focus Management: Proper focus handling

Styling

The component uses CSS classes that can be customized:

  • .sui-scroll-top: Main button container
  • .sui-scroll-top-visible: Visible state styling
  • .sui-scroll-top-hidden: Hidden state styling

OrgChart

Hierarchical organization chart components for displaying company structures, family trees, and other tree-like data relationships.

Basic OrgChart

<sui-org-chart 
  [value]="basicOrgData"
  (onNodeSelect)="onNodeSelect($event)"
  (onNodeExpand)="onNodeExpand($event)"
  (onNodeCollapse)="onNodeCollapse($event)">
</sui-org-chart>

Selection Modes

Single Selection

Multiple Selection

<!-- Single Selection -->
<sui-org-chart 
  [value]="basicOrgData"
  selectionMode="single"
  (onNodeSelect)="onNodeSelect($event)">
</sui-org-chart>

<!-- Multiple Selection -->
<sui-org-chart 
  [value]="basicOrgData"
  selectionMode="multiple"
  (onNodeSelect)="onNodeSelect($event)"
  (onNodeUnselect)="onNodeUnselect($event)">
</sui-org-chart>

Orientation

Vertical (Default)

Horizontal

<!-- Vertical -->
<sui-org-chart 
  [value]="basicOrgData"
  orientation="vertical">
</sui-org-chart>

<!-- Horizontal -->
<sui-org-chart 
  [value]="basicOrgData"
  orientation="horizontal">
</sui-org-chart>

Complex Data Structure

<sui-org-chart 
  [value]="complexOrgData"
  selectionMode="single"
  (onNodeSelect)="onNodeSelect($event)">
</sui-org-chart>

Data Structure

interface OrgChartNode {{ '{' }}
  key: string;
  label: string;
  data?: any;
  children?: OrgChartNode[];
  expanded?: boolean;
  type?: string;
{{ '}' }}

const orgData: OrgChartNode[] = [
  {{ '{' }}
    key: 'ceo',
    label: 'CEO',
    data: {{ '{' }} department: 'Executive', role: 'Chief Executive Officer' {{ '}' }},
    children: [
      {{ '{' }}
        key: 'cto',
        label: 'CTO',
        data: {{ '{' }} department: 'Technology', role: 'Chief Technology Officer' {{ '}' }},
        children: [
          {{ '{' }} key: 'dev1', label: 'Senior Developer', data: {{ '{' }} role: 'Developer' {{ '}' }} {{ '}' }},
          {{ '{' }} key: 'dev2', label: 'Junior Developer', data: {{ '{' }} role: 'Developer' {{ '}' }} {{ '}' }}
        ]
      {{ '}' }}
    ]
  {{ '}' }}
];

Features

  • ✓ Hierarchical data display with parent-child relationships
  • ✓ Single and multiple selection modes
  • ✓ Vertical and horizontal orientations
  • ✓ Collapsible nodes with expand/collapse functionality
  • ✓ Custom node data display
  • ✓ Event handling for node interactions

Props

Name Type Default Description
value OrgChartNode[] [] Array of organization chart nodes
selectionMode 'single' | 'multiple' | null null Selection mode for nodes
collapsible boolean true Whether nodes can be collapsed/expanded
orientation 'vertical' | 'horizontal' 'vertical' Chart orientation

Events

Name Parameters Description
onNodeSelect node: OrgChartNode Fired when a node is selected
onNodeUnselect node: OrgChartNode Fired when a node is unselected
onNodeExpand node: OrgChartNode Fired when a node is expanded
onNodeCollapse node: OrgChartNode Fired when a node is collapsed

Accessibility

  • ℹ Keyboard navigation support for node selection
  • ℹ ARIA labels for screen readers
  • ℹ Focus management for interactive elements

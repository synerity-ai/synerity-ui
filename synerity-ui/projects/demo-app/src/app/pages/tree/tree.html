Tree

Hierarchical tree component for displaying structured data with expand/collapse and selection capabilities.

Basic Tree

<sui-tree 
  [value]="treeData"
  [filter]="true"
  filterPlaceholder="Search nodes..."
  (onNodeSelect)="onNodeSelect($event)"
  (onNodeExpand)="onNodeExpand($event)">
</sui-tree>

Tree with Checkbox Selection

Selected nodes: {{ selectedNodes.length }} 
<sui-tree 
  [value]="treeData"
  selectionMode="checkbox"
  [filter]="true"
  filterPlaceholder="Search nodes..."
  (onNodeSelect)="onNodeSelect($event)"
  (onSelectionChange)="onSelectionChange($event)">
</sui-tree>

Installation

The Tree component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
value TreeNode[] [] Array of tree nodes
selectionMode 'single' | 'multiple' | 'checkbox' | null null Selection mode for nodes
filter boolean false Enable search/filter functionality

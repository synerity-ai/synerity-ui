Upload

File upload component with drag & drop, progress tracking, and validation.

Basic Upload

Selected Files:

{{ file.name }} 
<sui-upload 
  [multiple]="false"
  [accept]="'.jpg,.jpeg,.png,.pdf'"
  (onUpload)="onUpload($event)"
  (onSelect)="onFileSelect($event)">
</sui-upload>

Drag & Drop Upload

<sui-upload 
  [multiple]="true"
  [dragDrop]="true"
  [accept]="'.jpg,.jpeg,.png,.gif,.pdf,.doc,.docx'"
  (onUpload)="onUpload($event)"
  (onSelect)="onFileSelect($event)">
</sui-upload>

Upload Features

Drag & Drop

Support for dragging files directly onto the upload area

Multiple Files

Upload multiple files at once with progress tracking

File Validation

Validate file types, sizes, and other constraints

Progress Tracking

Real-time upload progress and status updates

Props

Prop Type Default Description
multiple boolean false Whether to allow multiple file selection
accept string '' Accepted file types
dragDrop boolean false Whether to enable drag and drop
maxFileSize number 0 Maximum file size in bytes

Events

Event Type Description
onSelect EventEmitter<FileList> Emitted when files are selected
onUpload EventEmitter<any> Emitted when upload starts
onProgress EventEmitter<number> Emitted during upload progress
onComplete EventEmitter<any> Emitted when upload completes

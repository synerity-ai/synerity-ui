Upload

A file upload component with drag-and-drop support, file validation, and progress tracking.

Basic Upload

A simple file upload component with drag and drop support.

<sui-upload 
  [multiple]="true"
  accept=".jpg,.jpeg,.png,.pdf,.doc,.docx"
  [maxFileSize]="5 * 1024 * 1024"
  (onSelect)="onFileSelect($event)"
  (onUpload)="onUpload($event)"
  (onError)="onError($event)">
</sui-upload>

Single File Upload

Upload only one file at a time with image file restrictions.

<sui-upload 
  [multiple]="false"
  accept="image/*"
  [maxFileSize]="2 * 1024 * 1024"
  (onSelect)="onFileSelect($event)"
  (onUpload)="onUpload($event)"
  (onError)="onError($event)">
</sui-upload>

Auto Upload

Files are automatically uploaded when selected.

              <sui-upload 
  [multiple]="true"
  [auto]="true"
  accept="*/*"
  [maxFileSize]="10 * 1024 * 1024"
  (onSelect)="onFileSelect($event)"
  (onUpload)="onUpload($event)"
  (onError)="onError($event)">
</sui-upload>

Custom Upload Handler

Upload component with custom upload logic and progress tracking.

// Component method
handleCustomUpload(event: any): void {
  const { files } = event;
  // Implement your custom upload logic
  files.forEach(file => {
    // Upload file to your server
  });
}

<sui-upload 
  [multiple]="true"
  accept=".txt,.md,.json"
  [maxFileSize]="1 * 1024 * 1024"
  (onSelect)="onFileSelect($event)"
  (onUpload)="onCustomUpload($event)"
  (onError)="onError($event)">
</sui-upload>

Installation

The Upload component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Upload component in your Angular application:

import { Upload } from '@synerity/ui';

Then use it in your template:

<sui-upload [multiple]="true" accept=".jpg,.jpeg,.png" [maxFileSize]="5 * 1024 * 1024" (onSelect)="onFileSelect($event)" (onUpload)="onUpload($event)"> </sui-upload>

File Validation

Configure file type and size restrictions:

<sui-upload accept=".pdf,.doc,.docx" [maxFileSize]="10 * 1024 * 1024" [multiple]="false"> </sui-upload>

Auto Upload

Enable automatic upload when files are selected:

<sui-upload [auto]="true" [multiple]="true" (onUpload)="onUpload($event)"> </sui-upload>

Component Properties

Prop Type Default Description
multiple boolean false Whether to allow multiple file selection
accept string '*/*' File types to accept (e.g., '.jpg,.png' or 'image/*')
maxFileSize number 0 Maximum file size in bytes
auto boolean false Whether to automatically upload files when selected
disabled boolean false Whether the upload component is disabled

Events

Event Description
onSelect Emitted when files are selected
onUpload Emitted when files are uploaded
onError Emitted when an error occurs
onProgress Emitted during upload progress

Overview

The Upload component provides a user-friendly interface for file uploads with drag-and-drop support, file validation, and progress tracking. It's perfect for forms that require file attachments, document uploads, or media file handling.

The component supports both single and multiple file uploads with configurable file type restrictions and size limits.

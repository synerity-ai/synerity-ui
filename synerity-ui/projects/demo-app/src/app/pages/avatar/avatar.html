Avatar

User profile images with fallback support for names and initials.

Basic Avatar

  [image]="avatarImage"
  [label]="avatarName">
</sui-avatar>

<!-- Initials Avatar -->

</sui-avatar>

Avatar Shapes

Shapes: Choose between circle (default) and square shapes for different design needs. 

<sui-avatar [label]="initials" shape="circle"></sui-avatar>
<sui-avatar [label]="initials" shape="square"></sui-avatar>

Installation

The Avatar component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Avatar Types

Image Avatar

Display user profile images with fallback support

Initials Avatar

Show user initials when no image is available

Icon Avatar

Display icons or emojis as avatar content

Props

Prop Type Default Description
image string '' Image URL for the avatar
label string '' Text to display (usually initials)
icon string '' Icon or emoji to display
size 'small' | 'normal' | 'large' | 'xlarge' 'normal' Avatar size
shape 'circle' | 'square' 'circle' Avatar shape
style object {} Custom inline styles
styleClass string '' Additional CSS classes

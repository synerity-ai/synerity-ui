Avatar

User profile images with fallback support for names and initials.

  [alt]="avatarName">

  [label]="avatarInitials"

Installation

The Avatar component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Avatar Types

Image Avatar

Display user profile images with fallback support

Initials Avatar

Show user initials when no image is available

Icon Avatar

Display icons or emojis as avatar content

Props

Prop Type Default Description
image string '' Image URL for the avatar
label string '' Text to display (usually initials)
icon string '' Icon or emoji to display
size 'sm' | 'md' | 'lg' | 'xl' 'md' Avatar size
alt string '' Alt text for accessibility

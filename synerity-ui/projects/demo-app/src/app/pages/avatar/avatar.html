Installation

npm install @synerity/ui

Basic Usage

<sui-avatar [label]="'John Doe'"></sui-avatar>

Import

import { Avatar } from '@synerity/ui';

Examples

Image Avatar

<sui-avatar [image]="'path/to/image.jpg'"></sui-avatar>

Label Avatar

<sui-avatar [label]="'John Doe'"></sui-avatar>

  size="large">
</sui-avatar>

Props

Prop Type Default Description
image string '' URL of the avatar image
label string '' Text to display as initials or label
icon string '' Icon to display in the avatar
size 'small' | 'normal' | 'large' | 'xlarge' 'normal' Size of the avatar
shape 'square' | 'circle' 'circle' Shape of the avatar
style any {} Inline styles for the avatar
styleClass string '' CSS class name for custom styling

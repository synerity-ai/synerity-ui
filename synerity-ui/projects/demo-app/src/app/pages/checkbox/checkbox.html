Checkbox

Installation

The Checkbox component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Checkbox component in your Angular application:

import {{ '{' }} Checkbox {{ '}' }} from '@synerity/ui';

{{ '@' }}Component({{ '{' }}
  template: `
    <div class="flex items-center space-x-3">
      <input 
        type="checkbox" 
        id="my-checkbox"
        [(ngModel)]="isChecked"
        class="w-4 h-4 text-blue-600 bg-gray-100 border-gray-300 rounded focus:ring-blue-500 focus:ring-2">
      <label for="my-checkbox" class="text-sm font-medium text-gray-900">
        Check this option
      </label>
    </div>
  `
{{ '}' }})
export class MyComponent {{ '{' }}
  isChecked = false;
{{ '}' }}

Form Integration

Integrate checkboxes with Angular reactive forms:

import {{ '{' }} FormBuilder, FormGroup {{ '}' }} from '@angular/forms';

export class MyComponent {{ '{' }}
  form: FormGroup;
  
  constructor(private fb: FormBuilder) {{ '{' }}
    this.form = this.fb.group({{ '{' }}
      acceptTerms: [false],
      newsletter: [true],
      notifications: [false]
    {{ '}' }});
  {{ '}' }}
  
  onSubmit() {{ '{' }}
    console.log(this.form.value);
  {{ '}' }}
{{ '}' }}

// Template
<form [formGroup]="form" (ngSubmit)="onSubmit()">
  <div class="space-y-4">
    <div class="flex items-center space-x-3">
      <input 
        type="checkbox" 
        formControlName="acceptTerms"
        class="w-4 h-4 text-blue-600 bg-gray-100 border-gray-300 rounded focus:ring-blue-500 focus:ring-2">
      <label class="text-sm font-medium text-gray-900">Accept Terms & Conditions</label>
    </div>
    
    <button type="submit" class="px-4 py-2 bg-blue-600 text-white rounded-md">
      Submit
    </button>
  </div>
</form>

Styling with Tailwind CSS

Checkboxes use Tailwind CSS classes for styling. Here are the key classes:

w-4 h-4 Checkbox size (width and height)
text-blue-600 Checked state color
bg-gray-100 Background color
border-gray-300 Border color
focus:ring-blue-500 Focus ring color

Props

Prop Type Default Description
checked boolean false Whether the checkbox is checked
disabled boolean false Whether the checkbox is disabled
indeterminate boolean false Whether the checkbox is in indeterminate state
required boolean false Whether the checkbox is required
name string '' Name attribute for form submission
value string 'on' Value attribute for form submission

Events

Event Type Description
onChange EventEmitter<boolean> Emitted when the checkbox state changes
onFocus EventEmitter<FocusEvent> Emitted when the checkbox receives focus
onBlur EventEmitter<FocusEvent> Emitted when the checkbox loses focus

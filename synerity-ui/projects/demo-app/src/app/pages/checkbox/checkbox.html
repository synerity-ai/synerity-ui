Checkbox

A form control component that allows users to select one or more options from a set of choices.

Basic Checkboxes

Checked: {{ basicChecked }}
Checked: {{ disabledChecked }}
Checked: {{ indeterminateChecked }} 
<!-- Basic Checkbox -->
<sui-checkbox 
  [(ngModel)]="isChecked"
  (ngModelChange)="onChange($event)"
  name="basic">
</sui-checkbox>

<!-- Disabled Checkbox -->
<sui-checkbox 
  [(ngModel)]="isChecked"
  [disabled]="true"
  name="disabled">
</sui-checkbox>

<!-- Another Checkbox -->
<sui-checkbox 
  [(ngModel)]="isChecked"
  name="another">
</sui-checkbox>

Installation

Import the Checkbox component in your module or standalone component:

import { Checkbox } from 'ui-lib';

// In your component imports
imports: [Checkbox]

Basic Usage

Use the Checkbox component with ngModel for two-way data binding:

<sui-checkbox
  [(ngModel)]="isChecked"
  (ngModelChange)="onChange($event)"
  name="myCheckbox">
</sui-checkbox>

Form Integration

Integrate with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-checkbox
    formControlName="agreeToTerms"
    name="agreeToTerms">
  </sui-checkbox>
</form>

Checkbox Groups

Create checkbox groups with select all functionality:

<!-- Select All Checkbox -->
<sui-checkbox
  [(ngModel)]="allSelected"
  (ngModelChange)="onSelectAllChange($event)"
  name="selectAll">
</sui-checkbox>

<!-- Individual Checkboxes -->
<div *ngFor="let option of options">
  <sui-checkbox
    [(ngModel)]="option.checked"
    (ngModelChange)="onOptionChange(option, $event)"
    [name]="option.id">
  </sui-checkbox>
</div>

Checkbox Component

The Checkbox component provides a form control for selecting one or more options from a set of choices. It supports various states including checked, unchecked, disabled, and indeterminate.

Props

Property Type Default Description
ngModel boolean false Two-way binding for the checkbox state
disabled boolean false Disable the checkbox
name string - Name attribute for the checkbox
inputId string - Unique identifier for the checkbox element

Events

Event Type Description
ngModelChange boolean Emitted when the checkbox state changes
onFocus Event Emitted when the checkbox receives focus
onBlur Event Emitted when the checkbox loses focus

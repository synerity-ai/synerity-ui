How to Use Checkbox

Installation

Import the Checkbox component from the UI library:

import { Checkbox } from 'ui-lib';

// In your component imports
imports: [Checkbox]

Basic Usage

Use the Checkbox component with two-way data binding:

<sui-checkbox
  [(ngModel)]="isChecked"
  (ngModelChange)="onChange($event)"
  name="myCheckbox"
>Checkbox Label</sui-checkbox>

Component Properties

  • ngModel: Two-way binding for the checked state
  • name: Name attribute for the input
  • disabled: Disable the checkbox
  • indeterminate: Set indeterminate state
  • inputId: Custom ID for the input element

Events

  • ngModelChange: Emitted when the checkbox state changes

Common Use Cases

  • Form Controls: Basic form input for boolean values
  • Settings: Toggle options and preferences
  • Lists: Select multiple items from a list
  • Agreements: Terms and conditions acceptance
  • Filters: Filter options in search interfaces

Best Practices

  • Clear Labels: Use descriptive labels for checkboxes
  • Grouping: Group related checkboxes logically
  • Indeterminate State: Use for "select all" scenarios
  • Accessibility: Ensure proper labeling and keyboard navigation
  • Validation: Provide clear feedback for required checkboxes

Form Integration

The component works seamlessly with Angular forms:

<form [formGroup]="myForm">
  <sui-checkbox
    formControlName="agreeToTerms"
    name="agreeToTerms"
  >I agree to the terms and conditions</sui-checkbox>
</form>

Styling Options

  • checkbox-enhanced: Enhanced appearance with custom checkmark
  • checkbox-animated: Smooth hover and click animations
  • checkbox-glow: Glow effect on focus
  • checkbox-ripple: Ripple effect on interaction

API Documentation

Properties

Property Type Default Description
checked boolean false Checked state of the checkbox
disabled boolean false Disable the checkbox
indeterminate boolean false Set indeterminate state
name string '' Name attribute for the input
inputId string '' ID attribute for the input

Events

  • ngModelChange: Emitted when the checkbox state changes (boolean)

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Full keyboard support (Space, Tab)
  • Focus Management: Visible focus indicators
  • Screen Reader: Compatible with screen reader announcements

Styling Classes

  • .checkbox-enhanced: Enhanced appearance with custom checkmark
  • .checkbox-animated: Smooth animations on interaction
  • .checkbox-glow: Glow effect on focus
  • .checkbox-ripple: Ripple effect on click
  • .checkbox-border-animated: Animated border effect

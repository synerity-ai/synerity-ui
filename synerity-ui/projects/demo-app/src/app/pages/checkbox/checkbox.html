Checkbox

Checkbox component for binary choices with various states, sizes, and styling options.

Basic Usage

Basic checked: {{ basicChecked }}

<div class="space-y-4">
  <div class="flex items-center space-x-3">
    <sui-checkbox 
      [(ngModel)]="basicChecked"
      (change)="onCheckboxChange($event)" />
    <label class="text-sm font-medium text-gray-700">Basic checkbox</label>
  </div>
  
  <div class="flex items-center space-x-3">
    <sui-checkbox 
      [(ngModel)]="labeledChecked"
      name="labeled" />
    <label class="text-sm font-medium text-gray-700">Checkbox with label</label>
  </div>
  
  <p class="text-sm text-gray-500 mt-2">Basic checked: {{ basicChecked }}</p>
</div>

Checkbox Group

Select your preferences:

Selected: {{ selectedOptions.join(', ') || 'None' }}

  <h4 class="text-lg font-medium text-gray-900">Select your preferences:</h4>

      <sui-checkbox 
        [value]="{{ "{{" }} option.value {{ "}}" }}"

    </div>

Properties

Property Type Default Description
label string null Label text for the checkbox
disabled boolean false Disables the checkbox
readonly boolean false Makes the checkbox read-only
required boolean false Makes the checkbox required
size 'sm' | 'md' | 'lg' 'md' Size of the checkbox

Events

Event Type Description
change Event Fired when the checkbox value changes
focus FocusEvent Fired when the checkbox receives focus
blur FocusEvent Fired when the checkbox loses focus

Usage Example

<sui-checkbox
  [(ngModel)]="checked"
  label="Accept terms and conditions"
  [required]="true"
  [disabled]="false"
  size="md"
  (change)="onCheckboxChange($event)"
  (focus)="onFocus($event)"
  (blur)="onBlur($event)">
</sui-checkbox>

Date Constraints

Only dates before next month

Weekends (Saturday & Sunday) are disabled

  [(ngModel)]="date"
  [minDate]="new Date()"
  [maxDate]="nextMonth">
</sui-date-picker>

<!-- Disable specific days of week -->

Props

Prop Type Default Description
value Date | null null Two-way binding for selected date
size 'small' | 'medium' | 'large' 'medium' Size variant of the date picker
theme 'light' | 'dark' 'light' Theme variant
variant 'default' | 'compact' | 'inline' 'default' Display variant
showTime boolean false Enable time selection
timeFormat '12' | '24' '24' Time format when showTime is enabled
minDate Date | null null Minimum selectable date
maxDate Date | null null Maximum selectable date
disabledDays number[] [] Days of week to disable (0=Sunday)
disabled boolean false Disable the date picker
readonly boolean false Make the date picker readonly

Events

Event Parameters Description
onChange Date | null Callback when value changes
onSelect Date | null Callback when date is selected
onOpen void Callback when calendar opens
onClose void Callback when calendar closes

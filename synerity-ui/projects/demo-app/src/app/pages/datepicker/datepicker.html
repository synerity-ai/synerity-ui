Installation

Import the DatePicker component in your module or standalone component:

import { DatePicker } from 'ui-lib'; // In your component imports imports: [DatePicker, FormsModule]

Basic Usage

Use the DatePicker component with ngModel for two-way data binding:

<sui-date-picker
  [(ngModel)]="selectedDate"
  (onChange)="onDateChange($event)"
  placeholder="Select a date"
  size="medium">
</sui-date-picker>

Form Integration

Integrate with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-date-picker
    formControlName="selectedDate"
    [minDate]="new Date()"
    [maxDate]="new Date(2025, 11, 31)"
    [showTime]="true"
    placeholder="Select date and time"
    size="medium">
  </sui-date-picker>
</form>

DatePicker Component

The DatePicker component provides a comprehensive date selection interface with calendar popup, time selection, and advanced customization options.

Props

Prop Type Default Description
ngModel Date | null - Two-way binding for the selected date
placeholder string - Placeholder text for the input
size string medium Size of the date picker (small, medium, large)
minDate Date - Minimum selectable date
maxDate Date - Maximum selectable date
showTime boolean false Show time selection
disabled boolean false Disables the date picker

Events

Event Type Description
onChange Date | null Emitted when the date value changes

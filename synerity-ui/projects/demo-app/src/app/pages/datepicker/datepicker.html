How to Use DatePicker

Installation

Import the DatePicker component in your Angular module:

import {{ '{' }} DatePicker {{ '}' }} from '@synerity/ui-lib';

@NgModule({{ '{' }}
  imports: [
    DatePicker
  ]
{{ '}' }})
export class YourModule {{ '{' }} {{ '}' }}

Basic Usage

Use the DatePicker component in your template:

<sui-date-picker
  [(ngModel)]="selectedDate"
  placeholder="Select a date"
  variant="dropdown"
  size="normal"
  (onChange)="onDateChange($event)">
</sui-date-picker>

Form Integration

The DatePicker implements ControlValueAccessor and works seamlessly with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-date-picker
    formControlName="date"
    placeholder="Select a date"
    variant="dropdown"
    size="normal">
  </sui-date-picker>
</form>

DatePicker Documentation

Component Overview

The DatePicker component provides a comprehensive date selection interface with calendar popup, keyboard navigation, and accessibility support. It supports both dropdown and inline variants with customizable sizing options.

Props

Property Type Default Description
value Date | null null Selected date value
placeholder string 'Select date' Placeholder text for the input
variant string 'dropdown' Visual variant (dropdown, inline)
size string 'normal' Size of the calendar (normal, compact)
disabled boolean false Whether the date picker is disabled
readonly boolean false Whether the date picker is readonly

Events

Event Type Description
onChange EventEmitter<Date | null> Emitted when the selected date changes
onSelect EventEmitter<Date | null> Emitted when a date is selected
onOpen EventEmitter<void> Emitted when the calendar opens
onClose EventEmitter<void> Emitted when the calendar closes

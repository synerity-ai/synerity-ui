Props

Prop Type Default Description
value Date | null null Two-way binding for selected date
placeholder string 'Select date' Placeholder text to display
disabled boolean false Disable the date picker
readonly boolean false Make the date picker readonly
minDate Date | null null Minimum selectable date
maxDate Date | null null Maximum selectable date
dateFormat string 'MM/dd/yyyy' Date format for display
showIcon boolean true Show calendar icon
style object {{ '{' }} {{ '}' }} Inline style object
styleClass string '' CSS class name

Events

Event Parameters Description
onChange Date | null Callback to invoke when value changes
onSelect Date | null Callback to invoke when date is selected

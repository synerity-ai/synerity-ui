API Reference

Inputs

Property Type Default Description
value Date | null null Selected date value
placeholder string 'Select date' Placeholder text
disabled boolean false Disables the date picker
readonly boolean false Makes the date picker read-only
required boolean false Marks the field as required
size 'small' | 'medium' | 'large' 'medium' Size of the date picker
variant 'default' | 'compact' | 'inline' 'default' Visual variant
minDate Date | null null Minimum selectable date
maxDate Date | null null Maximum selectable date
showTime boolean false Enable time selection
timeFormat '12' | '24' '24' Time format when showTime is true

Outputs

Event Type Description
onChange EventEmitter<Date | null> Emitted when the date value changes
onSelect EventEmitter<Date | null> Emitted when a date is selected
onOpen EventEmitter<void> Emitted when the calendar opens
onClose EventEmitter<void> Emitted when the calendar closes

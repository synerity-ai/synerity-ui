Menubar

Horizontal navigation menu with dropdown submenus for application navigation.

Basic Menubar

Interactive: Hover over menu items to see dropdown submenus. Click on items to trigger events. 

<sui-menubar 
  [model]="menubarItems"
  (onItemClick)="onMenubarItemClick($event)">
</sui-menubar>

Custom Styled Menubar

Custom Styling: Use styleClass prop to apply custom CSS classes for theming. 

<sui-menubar 
  [model]="customMenubarItems"
  styleClass="custom-menubar"
  (onItemClick)="onMenubarItemClick($event)">
</sui-menubar>

// Custom CSS
.custom-menubar {
  @apply bg-gradient-to-r from-blue-500 to-purple-600;
}

.custom-menubar .sui-menubar-item-content {
  @apply text-white hover:bg-white hover:bg-opacity-20;
}

Installation

The Menubar component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Menubar component in your Angular application:

import { Menubar } from '@synerity/ui';

@Component({
  template: `
    <sui-menubar [model]="menuItems"></sui-menubar>

export class MyComponent {}

Menu Structure

The menubar expects an array of menu items with the following structure:

label Required. Display text for the menu item
icon Optional. Icon to display before the label
items Optional. Array of submenu items
command Optional. Function to execute when clicked
disabled Optional. Boolean to disable the menu item

Event Handling

Handle menu item clicks using the onItemClick event:

<sui-menubar 
  [model]="menuItems"
  (onItemClick)="onMenuItemClick($event)">
</sui-menubar>

onMenuItemClick(item: any) {
  console.log('Menu item clicked:', item);
  
  // Handle different menu actions
  switch(item.label) {
    case 'New':
      this.createNew();
      break;
    case 'Open':
      this.openFile();
      break;
  }
}

Props

Prop Type Default Description
model Array<MenuItem> [] Array of menu items to display
style any {} Inline styles for the menubar
styleClass string '' CSS class name for custom styling

Events

Event Type Description
onItemClick EventEmitter<MenuItem> Emitted when a menu item is clicked

MenuItem Interface

interface MenuItem {
  label: string;           // Display text
  icon?: string;           // Optional icon
  command?: () => void;    // Optional click handler
  disabled?: boolean;      // Optional disabled state
  items?: MenuItem[];      // Optional submenu items
}

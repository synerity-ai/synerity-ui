Menubar

A horizontal menu bar with dropdown menus for navigation.

Basic Menubar

<sui-menubar 
  [model]="menubarItems"
  (onItemClick)="onMenubarItemClick($event)">
</sui-menubar>

Nested Menubar

Menubar with nested menu items and separators.

<sui-menubar 
  [model]="menubarItems"
  (onItemClick)="onMenubarItemClick($event)">
</sui-menubar>

Installation

The Menubar component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Menubar component in your Angular application:

import { Menubar } from '@synerity/ui';

@Component({
  template: `
    <sui-menubar 
      [model]="menubarItems"
      (onItemClick)="onItemClick($event)">
    </sui-menubar>
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Menubar uses Tailwind CSS classes for styling. Here are the key classes:

sui-menubar Base class for the menubar container
sui-menubar-item Class for individual menu items
sui-menubar-item-active Class for the active menu item

Props

Prop Type Default Description
model Array<MenuItem> [] Array of menu items to display
style any {} Inline styles for the menubar container
styleClass string '' Style class for the menubar container

Events

Event Type Description
onItemClick EventEmitter<MenuItem> Emitted when a menu item is clicked

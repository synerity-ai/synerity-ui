Menubar

A horizontal navigation menubar component for displaying a list of options with support for submenus and icons.

Basic Menubar

A simple horizontal menubar with basic menu items and click handlers.

<sui-menubar 
  [model]="menubarItems" 
  (onItemClick)="onMenubarItemClick($event)">
</sui-menubar>

Menubar with Submenus

Horizontal menubar with nested submenus for hierarchical navigation.

// Component data
menubarWithSubmenus = [
  { 
    label: 'File',
    icon: 'pi pi-file',
    items: [
      { label: 'New', command: () => this.newFile() },
      { label: 'Open', command: () => this.openFile() },
      { label: 'Save', command: () => this.saveFile() }
    ]
  },
  { 
    label: 'Edit',
    icon: 'pi pi-pencil',
    items: [
      { label: 'Cut', command: () => this.cut() },
      { label: 'Copy', command: () => this.copy() },
      { label: 'Paste', command: () => this.paste() }
    ]
  }
];

<sui-menubar 
  [model]="menubarWithSubmenus" 
  (onItemClick)="onMenubarItemClick($event)">
</sui-menubar>

Menubar with Disabled Items

Menubar items with disabled states and separators.

// Component data
menubarWithDisabledItems = [
  { 
    label: 'Enabled Item', 
    icon: 'pi pi-check',
    command: () => this.enabledAction() 
  },
  { 
    label: 'Disabled Item', 
    icon: 'pi pi-times',
    disabled: true
  },
  { 
    label: '',
    separator: true
  },
  { 
    label: 'Last Item', 
    icon: 'pi pi-flag',
    command: () => this.lastAction() 
  }
];

<sui-menubar 
  [model]="menubarWithDisabledItems" 
  (onItemClick)="onMenubarItemClick($event)">
</sui-menubar>

Installation

The Menubar component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Menubar component in your Angular application:

import { Menubar } from '@synerity/ui';

Then use it in your template:

<sui-menubar [model]="menubarItems" (onItemClick)="onMenubarItemClick($event)"> </sui-menubar>

Data Structure

Define your menubar items in your component:

// Component data menubarItems = [ { label: 'Home', icon: 'pi pi-home', command: () => this.navigateToHome() }, { label: 'About', icon: 'pi pi-info-circle', command: () => this.navigateToAbout() }, { label: 'Contact', icon: 'pi pi-envelope', command: () => this.navigateToContact() } ];

Submenus

Create nested submenus for hierarchical navigation:

submenuItems = [ { label: 'File', items: [ { label: 'New', command: () => this.newFile() }, { label: 'Open', command: () => this.openFile() }, { label: 'Save', command: () => this.saveFile() } ] } ];

Component Properties

Prop Type Default Description
model MenuItem[] [] Array of menubar items
style object null Inline style of the component
styleClass string null Style class of the component

Events

Event Description
onItemClick Emitted when a menubar item is clicked

Overview

The Menubar component provides a horizontal navigation bar for displaying a list of options. It supports hierarchical navigation with submenus, icons, and custom styling to create rich navigation experiences.

Menubars are essential for application navigation, providing users with organized access to functionality in a horizontal layout.

Splitter

Splitter components for creating resizable panel layouts with customizable gutters and size constraints.

Installation

npm install @synerity/ui-lib

Import

import { Splitter } from '@synerity/ui-lib';

@Component({
  imports: [Splitter],
  // ...
})

Basic Usage

<sui-splitter layout="horizontal">
  <div slot="panel1">Left Panel</div>
  <div slot="panel2">Right Panel</div>
</sui-splitter>

Features

  • ✓ Horizontal and vertical layouts
  • ✓ Resizable panels with drag handles
  • ✓ Customizable gutter size
  • ✓ Minimum and maximum size constraints
  • ✓ Resize event handling

Props

Name Type Default Description
layout 'horizontal' | 'vertical' 'horizontal' Layout orientation of the splitter
gutterSize number 4 Size of the gutter in pixels
minSizes number[] [0, 0] Minimum sizes for each panel (percentage)
maxSizes number[] [100, 100] Maximum sizes for each panel (percentage)
style any {} Inline styles for the splitter
styleClass string '' CSS class for the splitter

Events

Name Parameters Description
onResizeEnd { sizes: number[] } Fired when resizing is complete

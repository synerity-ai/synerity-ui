Splitter

Resizable panel component that allows users to adjust the size of adjacent panels by dragging the gutter.

<sui-splitter 
  layout="horizontal"
  [minSizes]="[20, 20]"
  [maxSizes]="[80, 80]"
  (onResizeEnd)="onResizeEnd($event)">
  
  <div slot="panel1" class="p-4 h-full">
    <h4 class="font-medium text-gray-900 mb-3">Left Panel</h4>
    <div class="space-y-2">
      <div class="p-2 bg-blue-100 rounded text-sm text-blue-800">Content 1</div>
    </div>
  </div>

    <div class="space-y-2">
      <div class="p-2 bg-green-100 rounded text-sm text-green-800">Content A</div>
    </div>
  </div>
</sui-splitter>

Installation

The Splitter component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Props

Prop Type Default Description
layout 'horizontal' | 'vertical' 'horizontal' Orientation of the splitter
gutterSize number 4 Size of the gutter in pixels
minSizes number[] [0, 0] Minimum sizes for panels (in percentage)

Splitter

Resizable panel splitter component for creating flexible layouts.

Basic Splitter

Left Panel

This is the left panel content. You can resize it by dragging the splitter.

Right Panel

This is the right panel content. The splitter allows you to adjust the panel sizes.

 
<sui-splitter 
  layout="horizontal"
  (onResizeEnd)="onResize($event)"
  style="height: 300px;">
  <ng-template pTemplate="panel">
    <div class="p-4 bg-blue-50">Left Panel</div>
  </ng-template>
  <ng-template pTemplate="panel">
    <div class="p-4 bg-green-50">Right Panel</div>
  </ng-template>
</sui-splitter>

Installation

The Splitter component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Splitter component in your Angular application:

import { Splitter } from '@synerity/ui';

@Component({
  template: `
    <sui-splitter [panelSizes]="[30, 70]">
      <ng-template pTemplate="panel">
        <div>Panel 1</div>
      </ng-template>
      <ng-template pTemplate="panel">
        <div>Panel 2</div>
      </ng-template>
    </sui-splitter>
  `
})
export class MyComponent {}

Props

Prop Type Default Description
panelSizes number[] [50, 50] Array of panel sizes in percentage
layout 'horizontal' | 'vertical' 'horizontal' Splitter orientation
gutterSize number 4 Size of the gutter in pixels
style any {} Inline styles for the splitter
styleClass string '' CSS class for the splitter

Events

Event Type Description
onResize EventEmitter<any> Emitted when panels are resized

Best Practices

  • ‚úì Keep the number of actions limited (3-5 items) for better UX
  • ‚úì Use clear, intuitive icons and labels for actions
  • ‚úì Position the speed dial where it won't obstruct important content
  • ‚úì Consider the direction based on available screen space
  • ‚úì Provide visual feedback when actions are performed

API Reference

Property Type Default Description
items Array<SpeedDialItem> [] Array of speed dial items to display
direction 'up' | 'down' | 'left' | 'right' 'up' Direction in which items expand
buttonIcon string '+' Icon for the main speed dial button
buttonLabel string '' Label for the main speed dial button
disabled boolean false Whether the speed dial is disabled

Events

Event Type Description
itemClick EventEmitter<SpeedDialItem> Emitted when a speed dial item is clicked

SpeedDialItem Interface

Property Type Description
label string (optional) Label text for the item
icon string (optional) Icon for the item
command () => void (optional) Function to execute when item is clicked
disabled boolean (optional) Whether the item is disabled

Accessibility

  • Focus management for better keyboard interaction
  • High contrast support for better visibility

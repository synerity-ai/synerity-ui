MegaMenu

A large dropdown menu component with support for multi-column layouts and extensive navigation options.

Basic MegaMenu

A simple mega menu with basic menu items and click handlers.

<sui-mega-menu 
  [model]="megaMenuItems" 
  (onItemClick)="onMegaMenuItemClick($event)">
</sui-mega-menu>

MegaMenu with Items

Mega menu with regular submenu items in a single column layout.

// Component data
megaMenuWithItems = [
  { 
    label: 'Products',
    icon: 'pi pi-box',
    items: [
      { label: 'Electronics', command: () => this.navigateToElectronics() },
      { label: 'Clothing', command: () => this.navigateToClothing() },
      { label: 'Books', command: () => this.navigateToBooks() }
    ]
  },
  { 
    label: 'Services',
    icon: 'pi pi-cog',
    items: [
      { label: 'Consulting', command: () => this.navigateToConsulting() },
      { label: 'Support', command: () => this.navigateToSupport() }
    ]
  }
];

<sui-mega-menu 
  [model]="megaMenuWithItems" 
  (onItemClick)="onMegaMenuItemClick($event)">
</sui-mega-menu>

MegaMenu with Columns

Mega menu with multi-column layout for organizing large amounts of content.

// Component data
megaMenuWithColumns = [
  { 
    label: 'Categories',
    icon: 'pi pi-th-large',
    columns: [
      [
        { label: 'Electronics', command: () => this.navigateToElectronics() },
        { label: 'Computers', command: () => this.navigateToComputers() }
      ],
      [
        { label: 'Clothing', command: () => this.navigateToClothing() },
        { label: 'Men\'s', command: () => this.navigateToMens() }
      ],
      [
        { label: 'Home', command: () => this.navigateToHome() },
        { label: 'Furniture', command: () => this.navigateToFurniture() }
      ]
    ]
  }
];

<sui-mega-menu 
  [model]="megaMenuWithColumns" 
  (onItemClick)="onMegaMenuItemClick($event)">
</sui-mega-menu>

Installation

The MegaMenu component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the MegaMenu component in your Angular application:

import { MegaMenu } from '@synerity/ui';

Then use it in your template:

<sui-mega-menu [model]="megaMenuItems" (onItemClick)="onMegaMenuItemClick($event)"> </sui-mega-menu>

Data Structure

Define your mega menu items in your component:

// Component data megaMenuItems = [ { label: 'Products', icon: 'pi pi-box', command: () => this.navigateToProducts() }, { label: 'Services', icon: 'pi pi-cog', command: () => this.navigateToServices() } ];

Multi-Column Layout

Create multi-column layouts for organizing large amounts of content:

megaMenuWithColumns = [ { label: 'Categories', columns: [ [ { label: 'Electronics', command: () => this.navigateToElectronics() }, { label: 'Computers', command: () => this.navigateToComputers() } ], [ { label: 'Clothing', command: () => this.navigateToClothing() }, { label: 'Accessories', command: () => this.navigateToAccessories() } ] ] } ];

Component Properties

Prop Type Default Description
model MegaMenuItem[] [] Array of mega menu items
orientation 'horizontal' | 'vertical' 'horizontal' Menu orientation
style object null Inline style of the component
styleClass string null Style class of the component

Events

Event Description
onItemClick Emitted when a mega menu item is clicked

Overview

The MegaMenu component provides a large dropdown menu with support for multi-column layouts and extensive navigation options. It's perfect for organizing large amounts of content in a structured, easy-to-navigate format.

MegaMenus are ideal for e-commerce sites, large websites, and applications that need to display extensive navigation options in an organized manner.

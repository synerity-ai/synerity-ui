Installation

The Tabs component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Tab Configuration

Define your tabs data structure:

// Tab data structure
tabsData = [
  {{ '{' }} id: 'tab1', label: 'Tab 1', content: 'Content 1' {{ '}' }},
  {{ '{' }} id: 'tab2', label: 'Tab 2', content: 'Content 2' {{ '}' }},
  {{ '{' }} id: 'tab3', label: 'Tab 3', content: 'Content 3' {{ '}' }}
];

// Track active tab
activeTabIndex = 0;

Props

Prop Type Default Description
tabs Tab[] [] Array of tab objects
activeIndex number 0 Index of the active tab
orientation 'horizontal' | 'vertical' 'horizontal' Tab orientation
closable boolean false Whether tabs can be closed

Events

Event Type Description
onTabChange EventEmitter<number> Emitted when a tab is selected
onTabClose EventEmitter<number> Emitted when a tab is closed

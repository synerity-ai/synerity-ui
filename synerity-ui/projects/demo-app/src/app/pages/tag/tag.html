Getting Started

Import the Tag component and add it to your template:

import { Tag } from 'ui-lib';

<sui-tag [value]="New" [severity]="success">
</sui-tag>

Basic tag usage:

<sui-tag
  [value]="Important"
  [severity]="danger"
  [icon]="⚠️"
  [removable]="true"
  [disabled]="false"
  (onRemove)="onTagRemove()">
</sui-tag>

API Reference

Properties

Property Type Default Description
value string '' Text content of the tag
severity string 'info' Severity level (info, success, warning, danger, contrast)
icon string '' Icon to display in the tag
removable boolean false Whether the tag can be removed
disabled boolean false Whether the tag is disabled
style object {} Inline styles for the tag
styleClass string '' CSS class for the tag

Events

Event Type Description
onRemove EventEmitter<void> Emitted when the remove button is clicked

Tag

Flexible tag component for categorizing and labeling content with various severity levels and interactive features.

Basic Tags

Severity Levels: Different severity levels provide visual context for different types of information. 

<sui-tag value="Angular" severity="info"></sui-tag>
<sui-tag value="React" severity="success"></sui-tag>
<sui-tag value="Vue" severity="warning"></sui-tag>
<sui-tag value="TypeScript" severity="danger"></sui-tag>
<sui-tag value="JavaScript" severity="contrast"></sui-tag>

Removable Tags

Interactive: Click the √ó button to remove tags. Disabled tags cannot be removed. 

<sui-tag 
  value="Removable Tag" 
  severity="info"
  [removable]="true"
  (onRemove)="onTagRemove($event)">
</sui-tag>

<sui-tag 
  value="Disabled Tag" 
  severity="danger"
  [removable]="true"
  [disabled]="true"
  (onRemove)="onTagRemove($event)">
</sui-tag>

Installation

The Tag component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Tag component in your Angular application:

import { Tag } from '@synerity/ui';

@Component({
  template: `
    <sui-tag value="Angular" severity="info"></sui-tag>
  `
})
export class MyComponent {}

Configuration Options

Severity Levels

<sui-tag severity="info|success|warning|danger|contrast">

Choose from predefined severity levels for different contexts.

Icon Support

<sui-tag icon="üé®" value="Frontend">

Add icons to enhance visual appeal and provide context.

Removable Tags

<sui-tag [removable]="true" (onRemove)="handleRemove()">

Enable removal functionality with clickable √ó button.

Custom Styling

<sui-tag styleClass="small rounded gradient">

Apply custom CSS classes for size, shape, and style variants.

Event Handling

Handle tag events using the provided event emitter:

<sui-tag 
  value="Removable Tag"
  [removable]="true"
  (onRemove)="onTagRemove($event)">
</sui-tag>

onTagRemove() {
  console.log('Tag removed');
  // Handle tag removal logic
}

Available Styling Classes

Size Classes

  • small - Smaller padding and text
  • large - Larger padding and text

Shape Classes

  • rounded - Rounded corners
  • square - Square corners

Style Classes

  • filled - Solid background
  • outlined - Border only
  • gradient - Gradient background

Effect Classes

  • pulse - Pulsing animation
  • loading - Loading state

Props

Prop Type Default Description
value string '' The text content of the tag
severity 'info' | 'success' | 'warning' | 'danger' | 'contrast' 'info' Severity level determining the visual style
icon string '' Icon to display before the tag text
removable boolean false Whether the tag can be removed
disabled boolean false Whether the tag is disabled
style any {} Inline styles for the tag
styleClass string '' CSS class name for custom styling

Events

Event Type Description
onRemove EventEmitter<void> Emitted when the remove button is clicked

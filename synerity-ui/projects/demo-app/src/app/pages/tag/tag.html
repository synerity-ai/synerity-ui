Tag

A small label component for categorizing and organizing content.

Basic Tag

<sui-tag 
  [value]="tagValue"
  (onClick)="onTagClick($event)">
</sui-tag>

Icon Tag

Tag with an icon for better visual representation.

<sui-tag 
  [value]="tagValue"
  [icon]="tagIcon"
  (onClick)="onTagClick($event)">
</sui-tag>

Installation

The Tag component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Tag component in your Angular application:

import { Tag } from '@synerity/ui';

@Component({
  template: `
    <sui-tag 
      [value]="tagValue"
      (onClick)="onClick($event)">
    </sui-tag>
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Tag uses Tailwind CSS classes for styling. Here are the key classes:

sui-tag Base class for the tag container
sui-tag-value Class for the tag text content
sui-tag-icon Class for the tag icon

Props

Prop Type Default Description
value string '' Text content of the tag
icon string '' Icon class for the tag
style any {} Inline styles for the tag container
styleClass string '' Style class for the tag container

Events

Event Type Description
onClick EventEmitter<Event> Emitted when the tag is clicked

Tag

A component for displaying labels, categories, or keywords with different severity levels and optional removal functionality.

Basic Tags

Basic tags with different severity levels.

Severity Levels: info, success, warning, danger, and contrast variants. 

<sui-tag value="Angular" severity="info"></sui-tag>
<sui-tag value="React" severity="success"></sui-tag>
<sui-tag value="Vue" severity="warning"></sui-tag>
<sui-tag value="TypeScript" severity="danger"></sui-tag>
<sui-tag value="JavaScript" severity="contrast"></sui-tag>

Tags with Icons

Tags with icons for better visual representation.

Icons: Add emoji or icon characters to enhance tag meaning. 

<sui-tag value="Frontend" icon="🎨" severity="info"></sui-tag>
<sui-tag value="Backend" icon="⚙️" severity="success"></sui-tag>
<sui-tag value="Database" icon="🗄️" severity="warning"></sui-tag>
<sui-tag value="DevOps" icon="🚀" severity="danger"></sui-tag>
<sui-tag value="Design" icon="🎯" severity="contrast"></sui-tag>

Removable Tags

Tags with remove functionality and disabled state.

Removable: Tags with remove buttons and disabled state support. 

<sui-tag 
  value="Removable Tag" 
  severity="info" 
  [removable]="true"
  (onRemove)="onTagRemove(tag)"
></sui-tag>

<sui-tag 
  value="Disabled Tag" 
  severity="danger" 
  [removable]="true"
  [disabled]="true"
></sui-tag>

Custom Styled Tags

Tags with custom styling and inline styles.

Custom Styling: Apply custom CSS classes and inline styles for unique appearances. 

<sui-tag 
  value="Custom Style" 
  severity="info"
  styleClass="custom-tag"
  [style]="{'background-color': '#8B5CF6', 'color': 'white'}"
></sui-tag>

<sui-tag 
  value="Gradient" 
  severity="success"
  [style]="{'background': 'linear-gradient(45deg, #FF6B6B, #4ECDC4)', 'color': 'white'}"
></sui-tag>

Installation

npm install ui-lib

Basic Usage

Import the Tag component and use it in your template:

import { Tag } from 'ui-lib';

// In your template
<sui-tag value="Angular" severity="info"></sui-tag>

Common Use Cases

Categories and Labels

Display categories, labels, or keywords in content

Status Indicators

Show status information with different severity levels

Filter Tags

Display active filters with removal capability

User Tags

Show user roles, permissions, or attributes

Properties

Property Type Default Description
value string '' Text content of the tag
severity string 'info' Severity level: info, success, warning, danger, contrast
icon string '' Icon or emoji to display in the tag
removable boolean false Whether the tag can be removed
disabled boolean false Whether the tag is disabled
style any {} Inline styles for the component
styleClass string '' CSS class for the component

Events

onRemove

Emitted when the remove button is clicked (only when removable is true)

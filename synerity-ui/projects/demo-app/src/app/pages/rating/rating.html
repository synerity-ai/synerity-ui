Rating

Star rating component for collecting user feedback and ratings with customizable maximum values and interactive states.

Basic Rating

{{ basicRating }} 
<sui-rating
  [(ngModel)]="basicRating"
  (change)="onRatingChange($event)"
  [max]="5"
  [allowClear]="true"
  inputId="basic-rating">
</sui-rating>

Installation

Import the Rating component in your Angular module:

import { Rating } from '@synerity/ui-lib';

@NgModule({
  imports: [
    Rating
  ]
})

Basic Usage

Use the Rating component in your template:

<sui-rating
  [(ngModel)]="rating"
  (change)="onRatingChange($event)"
  [max]="5"
  [allowClear]="true"
  inputId="rating">
</sui-rating>

Form Integration

The Rating implements ControlValueAccessor and works seamlessly with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-rating
    formControlName="rating"
    [max]="5"
    [allowClear]="true">
  </sui-rating>
</form>

Component Overview

The Rating component provides an intuitive star-based rating system for collecting user feedback. It supports customizable maximum values, interactive states, and seamless integration with Angular forms through ControlValueAccessor implementation.

Props

Property Type Default Description
ngModel number 0 Two-way binding for the rating value
max number 5 Maximum number of stars
disabled boolean false Disables the rating interaction
readonly boolean false Makes the rating read-only
allowClear boolean true Whether clicking the same star clears the rating
inputId string - Unique identifier for the rating element

Events

Event Type Description
change number Fired when the rating value changes

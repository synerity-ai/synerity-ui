Rating

Star rating component for collecting user feedback and ratings with customizable maximum values and interactive states.

Basic Rating

{{ basicRating }} 
<sui-rating
  [(ngModel)]="basicRating"
  (change)="onRatingChange($event)"
  [max]="5"
  [allowClear]="true"
  inputId="basic-rating">
</sui-rating>

Basic Usage

The Rating component provides an intuitive way to collect user ratings and feedback.

1. Import the Component

import { Rating as SuiRating } from '@synerity/ui';

2. Add to Component Imports

@Component({
  imports: [CommonModule, FormsModule, SuiRating],
  // ...
})

3. Use in Template

<sui-rating
  [(ngModel)]="ratingValue"
  (change)="onRatingChange($event)"
  [max]="5"
  [allowClear]="true"
  inputId="my-rating">
</sui-rating>

API Reference

Properties

Property Type Default Description
max number 5 Maximum number of stars
disabled boolean false Whether the rating is disabled
readonly boolean false Whether the rating is read-only
allowClear boolean true Whether clicking the same star clears the rating

Events

Event Type Description
change EventEmitter<number> Emitted when the rating value changes

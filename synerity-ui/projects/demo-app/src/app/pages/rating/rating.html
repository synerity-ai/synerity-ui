Features

Customizable Max

Set any maximum number of stars

Clearable

Allow users to clear ratings by clicking the same star

Form Integration

Full Angular forms support with validation

Accessibility

Keyboard navigation and screen reader support

Props

Prop Type Default Description
ngModel number 0 Two-way binding for rating value
max number 5 Maximum number of stars
disabled boolean false Disable the rating component
readonly boolean false Make the rating readonly
allowClear boolean true Allow clearing rating by clicking same star

Events

Event Parameters Description
change number Callback to invoke when rating changes

Accessibility

  • • Proper labeling and value announcements
  • • Keyboard navigation support (Arrow keys, Home, End, Space)
  • • Screen reader compatibility
  • • ARIA attributes for rating input
  • • Focus management and visual indicators

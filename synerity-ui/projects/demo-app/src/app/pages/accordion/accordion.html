Accordion

Collapsible content sections for organizing related information in a compact space.

Basic Accordion

<sui-accordion 
  [tabs]="accordionTabs"
  [activeIndex]="activeIndex"
  [multiple]="false"
  (onOpen)="onAccordionOpen($event)"
  (onClose)="onAccordionClose($event)"
  (onActiveIndexChange)="onActiveIndexChange($event)">
</sui-accordion>

Installation

The Accordion component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Accordion component in your Angular application:

import { Accordion } from '@synerity/ui';

@Component({
  template: `
    <sui-accordion 
      [tabs]="accordionTabs"
      [activeIndex]="activeIndex"
      (onOpen)="onAccordionOpen($event)">
    </sui-accordion>
  `
})
export class MyComponent {}

Accordion Configuration

Define your accordion tabs data structure:

// Accordion tabs data
accordionTabs = [
  { 
    header: 'Section 1', 
    content: 'Content for section 1' 
  },
  { 
    header: 'Section 2', 
    content: 'Content for section 2' 
  }
];

// Track active section
activeIndex = 0;

Props

Prop Type Default Description
tabs Tab[] [] Array of accordion tab objects
activeIndex number | number[] 0 Index of active section(s)
multiple boolean false Whether multiple sections can be open
style any {} Inline styles for the accordion
styleClass string '' CSS class for the accordion

Events

Event Type Description
onOpen EventEmitter<any> Emitted when a section is opened
onClose EventEmitter<any> Emitted when a section is closed
onActiveIndexChange EventEmitter<number | number[]> Emitted when active index changes

Accordion

A collapsible content component that displays multiple panels in a vertical stack, allowing users to expand and collapse sections.

Basic Accordion

<sui-accordion 
  [tabs]="basicTabs"
  [activeIndex]="0"
  variant="default"
  size="medium"
  (onOpen)="onAccordionOpen($event)"
  (onClose)="onAccordionClose($event)"
  (onActiveIndexChange)="onActiveIndexChange($event)"
></sui-accordion>

Accordion Variants

<sui-accordion 
  [tabs]="multipleTabs"
  [multiple]="true"
  [activeIndex]="[0, 2]"
  variant="bordered"
  size="medium"
  (onOpen)="onAccordionOpen($event)"
  (onClose)="onAccordionClose($event)"
  (onActiveIndexChange)="onActiveIndexChange($event)"
></sui-accordion>

Installation

Import the Accordion component in your module or standalone component:

import { Accordion } from 'ui-lib';

Basic Usage

Use the Accordion component to create collapsible content sections:

<sui-accordion 
  [tabs]="accordionTabs"
  [activeIndex]="0"
  (onOpen)="onAccordionOpen($event)"
  (onClose)="onAccordionClose($event)"
  (onActiveIndexChange)="onActiveIndexChange($event)"
></sui-accordion>

Styling Guidelines

  • Use clear, descriptive headers for each accordion panel
  • Keep content concise and well-organized
  • Consider using icons in headers for better visual hierarchy
  • Ensure proper spacing and padding for readability

Props

Property Type Default Description
tabs Array<AccordionTab> [] Array of accordion tab objects
activeIndex number | number[] 0 Index of active tab(s)
multiple boolean false Whether multiple tabs can be open
variant 'default' | 'bordered' | 'minimal' 'default' Visual variant of the accordion
size 'small' | 'medium' | 'large' 'medium' Size of the accordion
styleClass string - Custom CSS class for styling

Events

Event Type Description
onOpen AccordionTab Emitted when a tab is opened
onClose AccordionTab Emitted when a tab is closed
onActiveIndexChange number | number[] Emitted when active index changes

Accessibility

  • Accordion headers are keyboard accessible with proper ARIA attributes
  • Screen readers announce the expanded/collapsed state of each panel
  • Focus management ensures proper keyboard navigation
  • ARIA labels provide context for assistive technologies

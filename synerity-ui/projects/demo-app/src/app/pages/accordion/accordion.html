Accordion

Collapsible content panels that allow users to expand and collapse sections of information.

Basic Accordion

<sui-accordion 
  [tabs]="accordionTabs"
  [activeIndex]="singleActiveIndex"
  [multiple]="false"
  (onOpen)="onAccordionOpen($event)"
  (onClose)="onAccordionClose($event)"
  (onActiveIndexChange)="onActiveIndexChange($event)">
</sui-accordion>

Multiple Accordion

<sui-accordion 
  [tabs]="accordionTabs"
  [activeIndex]="multipleActiveIndex"
  [multiple]="true"
  (onOpen)="onAccordionOpen($event)"
  (onClose)="onAccordionClose($event)"
  (onActiveIndexChange)="onActiveIndexChange($event)">
</sui-accordion>

Installation

npm install @synerity/ui

Basic Usage

<sui-accordion [tabs]="accordionTabs"></sui-accordion>

Import

import { Accordion } from '@synerity/ui';

Props

Prop Type Default Description
tabs Array<{ header: string; content?: any }> [] Array of accordion tab objects
multiple boolean false Whether multiple tabs can be open at once
activeIndex number | number[] 0 Index of active tab(s)

Events

Event Type Description
onOpen EventEmitter<any> Emitted when an accordion tab is opened
onClose EventEmitter<any> Emitted when an accordion tab is closed
onActiveIndexChange EventEmitter<number | number[]> Emitted when the active index changes

Accessibility

  • • Full keyboard navigation support (Arrow keys, Enter, Space)
  • • WCAG 2.1 AA compliant color contrast
  • • Proper ARIA attributes for screen readers
  • • Focus management and visual focus indicators
  • • Expandable/collapsible state announcements

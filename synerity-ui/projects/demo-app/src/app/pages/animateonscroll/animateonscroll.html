AnimateOnScroll

A component that animates elements when they come into view during scrolling.

Basic AnimateOnScroll

A basic animation that triggers when the element comes into view.

Scroll to see me animate! 
<sui-animate-on-scroll
  [animation]="basicAnimation"
  [threshold]="basicThreshold"
  [delay]="basicDelay"
  [duration]="basicDuration"
  [once]="basicOnce">
  <div class="bg-blue-500 text-white px-6 py-3 rounded-lg font-medium">
    Scroll to see me animate!
  </div>
</sui-animate-on-scroll>

Getting Started

Import the AnimateOnScroll component and add it to your template:

import {{ '{' }} AnimateOnScroll {{ '}' }} from 'ui-lib';

<sui-animate-on-scroll [animation]="'fadeInUp'">
  <div>Content to animate</div>
</sui-animate-on-scroll>

Basic animation usage:

<sui-animate-on-scroll
  [animation]="'fadeInUp'"
  [threshold]="0.1"
  [delay]="0"
  [duration]="1000"
  [once]="true">
  <div>Content to animate</div>
</sui-animate-on-scroll>

API Reference

Properties

Property Type Default Description
animation string 'fadeInUp' Animation type to apply
threshold number 0.1 Visibility threshold (0-1)
delay number 0 Animation delay in milliseconds
duration number 1000 Animation duration in milliseconds
once boolean true Whether to animate only once

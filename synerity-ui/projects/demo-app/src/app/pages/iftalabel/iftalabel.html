Features

Conditional Display

Show different labels based on conditions

Three States

True, false, and alternative label support

Dynamic Updates

Labels update automatically when conditions change

Flexible Content

Support for any text content in labels

Props

Prop Type Default Description
condition boolean true Condition to determine which label to show
trueLabel string '' Label to show when condition is true
falseLabel string '' Label to show when condition is false
altLabel string '' Alternative label to show when condition is false
style any {{ '{' }}{{ '}' }} Inline styles for the label
styleClass string '' CSS class for the label

Accessibility

  • • Semantic label structure for screen readers
  • • Dynamic content updates are announced
  • • Proper text contrast and readability
  • • Support for assistive technologies

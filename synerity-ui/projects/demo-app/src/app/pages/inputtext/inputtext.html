InputText

Text input components with validation states, sizes, and interactive features.

Basic Input

<!-- Basic Text Input -->
<sui-input-text 
  [(ngModel)]="inputValue"
  placeholder="Enter text here..."
  label="Basic Text Input">
</sui-input-text>

<!-- With Placeholder -->
<sui-input-text 
  placeholder="This is a placeholder text"
  label="With Placeholder">
</sui-input-text>

Input States

<!-- Normal State -->
<sui-input-text 
  placeholder="Normal input field"
  label="Normal Input">
</sui-input-text>

<!-- Disabled State -->
<sui-input-text 
  placeholder="Disabled input field"
  label="Disabled Input"
  [disabled]="true">
</sui-input-text>

<!-- Read-only State -->
<sui-input-text 
  placeholder="Read-only input field"
  label="Read-only Input"
  [readonly]="true">
</sui-input-text>

Installation

The InputText component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the InputText component in your Angular application:

import {{ '{' }} InputText {{ '}' }} from '@synerity/ui';

{{ '@' }}Component({{ '{' }}
  template: `
    <sui-input-text 
      [(ngModel)]="inputValue"
      placeholder="Enter text here..."
      label="Basic Text Input">
    </sui-input-text>
  `
{{ '}' }})
export class MyComponent {{ '{' }}
  inputValue = '';
{{ '}' }}

Styling with Tailwind CSS

InputText uses Tailwind CSS classes for styling. Here are the key classes:

w-full Full width
px-3 py-2 Horizontal and vertical padding
border border-gray-300 Border styling
rounded-md Border radius
focus:ring-2 focus:ring-blue-500 Focus ring

Props

Prop Type Default Description
placeholder string '' Placeholder text for the input
disabled boolean false Whether the input is disabled
readonly boolean false Whether the input is read-only
required boolean false Whether the input is required
size 'sm' | 'md' | 'lg' 'md' Size of the input field
variant 'default' | 'filled' | 'outlined' 'default' Visual style variant of the input
validationState 'none' | 'valid' | 'invalid' 'none' Validation state of the input
errorMessage string '' Error message to display when validation fails
helperText string '' Helper text to display below the input
label string '' Label text for the input field
type 'text' | 'email' | 'password' | 'tel' | 'url' 'text' HTML input type attribute
maxlength number | null null Maximum number of characters allowed
minlength number | null null Minimum number of characters required
pattern string | null null Regular expression pattern for validation
autocomplete string | null null Autocomplete attribute value
autofocus boolean false Whether the input should be focused on page load
spellcheck boolean true Whether to enable spell checking
prefix string | null null Prefix text or icon to display before the input
suffix string | null null Suffix text or icon to display after the input
clearable boolean false Whether to show a clear button
showPasswordToggle boolean false Whether to show password toggle button (for password type)

Events

Event Type Description
input EventEmitter<Event> Emitted when the input value changes
focus EventEmitter<FocusEvent> Emitted when the input receives focus
blur EventEmitter<FocusEvent> Emitted when the input loses focus
clear EventEmitter<void> Emitted when the clear button is clicked

InputText

Text input component with validation, sizing, and various input types for user data entry.

Basic Usage

Value: {{ basicValue || 'Empty' }}

Value: {{ placeholderValue || 'Empty' }}

<div class="space-y-4">
  <div>
    <label class="block text-sm font-medium text-gray-700 mb-2">Basic Input</label>
    <sui-input-text 
      [(ngModel)]="basicValue"

Properties

Property Type Default Description
placeholder string '' Placeholder text for the input
disabled boolean false Disables the input
readonly boolean false Makes the input read-only
required boolean false Makes the input required
size 'sm' | 'md' | 'lg' 'md' Size of the input
variant 'default' | 'filled' | 'outlined' 'default' Visual variant of the input
type 'text' | 'email' | 'password' | 'tel' | 'url' 'text' Input type
maxlength number null Maximum number of characters
pattern string null Regex pattern for validation
clearable boolean false Shows clear button when input has value

Events

Event Type Description
input Event Fired when the input value changes
focus FocusEvent Fired when the input receives focus
blur FocusEvent Fired when the input loses focus
clear void Fired when the clear button is clicked

Usage Example

<sui-input-text
  [(ngModel)]="value"
  placeholder="Enter text here"
  [required]="true"
  [clearable]="true"
  size="md"
  variant="outlined"

  (focus)="onFocus($event)"
  (blur)="onBlur($event)"
  (clear)="onClear()">
</sui-input-text>

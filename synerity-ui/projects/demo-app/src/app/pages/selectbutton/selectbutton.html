How to Use SelectButton

Installation

Import the SelectButton component from the UI library:

import { SelectButton } from 'ui-lib';

// In your component imports
imports: [SelectButton]

Basic Usage

Use the component with an array of options:

<sui-select-button
  [(ngModel)]="selectedValue"
  [options]="options"
  (change)="onSelectionChange($event)"
></sui-select-button>

Component Properties

  • options: Array of option objects with label and value
  • disabled: Boolean to disable the entire component
  • name: String for form control name
  • ngModel: Two-way binding for the selected value

Events

  • change: Emitted when selection changes

Option Structure

options = [
  { label: 'Option 1', value: 'option1' },
  { label: 'Option 2', value: 'option2', disabled: true },
  { label: 'Option 3', value: 'option3' }
];

API Documentation

Properties

Property Type Default Description
options Array<{label: string, value: any, disabled?: boolean}> [] Array of selectable options
disabled boolean false Disables the entire component
name string '' Name attribute for form control
ngModel any null Selected value (two-way binding)

Events

Event Type Description
change EventEmitter<any> Fired when selection changes

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Tab navigation between buttons
  • Focus Management: Clear focus indicators
  • Screen Reader: Descriptive labels for all options

Styling

The component uses CSS classes that can be customized:

  • .select-button-container: Main container
  • .select-button-option: Individual button styling
  • .select-button-option-selected: Selected button styling
  • .select-button-option-disabled: Disabled button styling

Examples

Basic Usage

Simple selection between multiple options:

export class ExampleComponent {
  selectedOption = 'option1';
  options = [
    { label: 'Option 1', value: 'option1' },
    { label: 'Option 2', value: 'option2' },
    { label: 'Option 3', value: 'option3' }
  ];
}

With Disabled Options

Some options can be disabled:

options = [
  { label: 'Available', value: 'available' },
  { label: 'Disabled', value: 'disabled', disabled: true },
  { label: 'Active', value: 'active' }
];

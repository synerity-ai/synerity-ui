How to Use SelectButton

Installation

Import the SelectButton component from the UI library:

import { SelectButton } from 'ui-lib';

// In your component imports
imports: [SelectButton]

Basic Usage

Use the component with an array of options:

<sui-select-button
  [(ngModel)]="selectedValue"
  [options]="options"
  (change)="onSelectionChange($event)"
></sui-select-button>

Form Integration

The SelectButton component supports two-way data binding with Angular forms:

<form [formGroup]="myForm">
  <sui-select-button
    formControlName="selectedOption"
    [options]="options"
  ></sui-select-button>
</form>

Documentation

Component Overview

The SelectButton component provides a group of toggle buttons that allow users to select from mutually exclusive options. It's ideal for forms where users need to choose one option from a set of predefined values.

Props

Property Type Default Description
options Array<{label: string, value: any, disabled?: boolean}> [] Array of selectable options
disabled boolean false Disables the entire component
name string '' Name attribute for form control
ngModel any null Selected value (two-way binding)

Events

Event Type Description
change EventEmitter<any> Fired when selection changes

Overlay Panel

A floating panel component that appears over other content with customizable positioning.

Installation

The Overlay Panel component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Overlay Panel component in your Angular application:

import { OverlayPanel } from '@synerity/ui';

@Component({
  template: `
    <div class="p-4 bg-gray-100 rounded-lg" 
      [visible]="panelVisible"
      (onShow)="onShow($event)"
      (onHide)="onHide($event)">
      <div>Panel content</div>
    </div class="p-4 bg-gray-100 rounded-lg">
  `
})
export class MyComponent {}

Styling with Tailwind CSS

Overlay Panel uses Tailwind CSS classes for styling. Here are the key classes:

div class="p-4 bg-gray-100 rounded-lg" Base class for the overlay panel container
div class="p-4 bg-gray-100 rounded-lg"-content Class for the panel content area
div class="p-4 bg-gray-100 rounded-lg"-arrow Class for the panel arrow indicator

Props

Prop Type Default Description
visible boolean false Whether the overlay panel is visible
style any {} Inline styles for the overlay panel container
styleClass string '' Style class for the overlay panel container

Events

Event Type Description
onShow EventEmitter<Event> Emitted when the overlay panel is shown
onHide EventEmitter<Event> Emitted when the overlay panel is hidden

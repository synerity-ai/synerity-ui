KeyFilter

A component that filters keyboard input based on predefined patterns or custom regular expressions.

Basic KeyFilter

<!-- Alphanumeric Only -->
<sui-key-filter
  [(ngModel)]="value"
  pattern="alphanum"
  placeholder="Letters and numbers only"
  (change)="onChange($event)"
></sui-key-filter>

<!-- Letters Only -->
<sui-key-filter
  [(ngModel)]="alphaValue"
  pattern="alpha"
  placeholder="Letters only (A-Z, a-z)"
  (change)="onChange($event)"
></sui-key-filter>

<!-- Integers Only -->
<sui-key-filter
  [(ngModel)]="numericValue"
  pattern="int"
  placeholder="Whole numbers only"
  (change)="onChange($event)"
></sui-key-filter>

KeyFilter Patterns

<!-- Email Format -->
<sui-key-filter
  [(ngModel)]="emailValue"
  pattern="^[a-zA-Z0-9@._-]*$"
  placeholder="email@domain.com"
  (change)="onChange($event)"
></sui-key-filter>

<!-- Phone Number -->
<sui-key-filter
  [(ngModel)]="phoneValue"
  pattern="^[0-9-()+ ]*$"
  placeholder="(555) 123-4567"
  (change)="onChange($event)"
></sui-key-filter>

<!-- Hex Color Code -->
<sui-key-filter
  [(ngModel)]="hexValue"
  pattern="^[A-Fa-f0-9#]*$"
  placeholder="#FF5733"
  (change)="onChange($event)"
></sui-key-filter>

KeyFilter States

<!-- Normal State -->
<sui-key-filter
  [(ngModel)]="value"
  pattern="alphanum"
  placeholder="Enter username"
  (change)="onChange($event)"
></sui-key-filter>

<!-- Disabled State -->
<sui-key-filter
  [(ngModel)]="disabledValue"
  pattern="alphanum"
  placeholder="Disabled input"
  [disabled]="true"
></sui-key-filter>

<!-- Required Field -->
<sui-key-filter
  [(ngModel)]="emailValue"
  pattern="^[a-zA-Z0-9@._-]*$"
  placeholder="Enter email (required)"
  required
  (change)="onChange($event)"
></sui-key-filter>

Installation

The KeyFilter component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Predefined Patterns

KeyFilter comes with several predefined patterns for common use cases:

int Integers only (0-9)
num Numbers with decimals and signs (0-9, ., -, +)
alpha Letters only (A-Z, a-z)
alphanum Letters and numbers (A-Z, a-z, 0-9)

Props

Prop Type Default Description
pattern string 'alphanum' Filter pattern or regex expression
disabled boolean false Disable the input
placeholder string '' Placeholder text for the input
name string '' Name attribute for the input
inputId string '' ID attribute for the input

Events

Event Type Description
change EventEmitter<string> Emitted when the input value changes

Key Filter

A component that filters keyboard input based on predefined patterns or custom regular expressions.

How to Use Key Filter

Installation

Import the KeyFilter component from the UI library:

import { KeyFilter } from 'ui-lib';

// In your component imports
imports: [KeyFilter]

Basic Usage

Use the KeyFilter component with different patterns:

<sui-key-filter
  [(ngModel)]="value"
  pattern="alphanum"
  placeholder="Enter text"
  (change)="onChange($event)"
></sui-key-filter>

Predefined Patterns

  • int: Integers only (0-9)
  • num: Numbers with decimals and signs (0-9, ., -, +)
  • alpha: Letters only (A-Z, a-z)
  • alphanum: Letters and numbers (A-Z, a-z, 0-9)

Custom Patterns

Use regular expressions for custom filtering:

<sui-key-filter
  [(ngModel)]="email"
  pattern="^[a-zA-Z0-9@._-]*$"
  placeholder="Enter email"
></sui-key-filter>

Common Use Cases

  • Username Input: Alphanumeric characters only
  • Phone Numbers: Numbers, spaces, and common phone symbols
  • Email Input: Letters, numbers, @, ., _, -
  • Credit Cards: Numbers and spaces
  • Currency Input: Numbers and decimal points
  • Product Codes: Uppercase letters and numbers

Best Practices

  • User Feedback: Provide clear placeholder text
  • Validation: Combine with form validation
  • Accessibility: Ensure screen reader compatibility
  • Testing: Test with various input scenarios
  • Performance: Use simple regex patterns when possible

Form Integration

The component implements ControlValueAccessor for seamless form integration:

<form (ngSubmit)="onSubmit()">
  <sui-key-filter
    [(ngModel)]="userData.username"
    pattern="alphanum"
    name="username"
    required
  ></sui-key-filter>
</form>

API Documentation

Properties

Property Type Default Description
disabled boolean false Disable the input
name string '' Name attribute for the input
inputId string '' ID attribute for the input
placeholder string '' Placeholder text
pattern string 'alphanum' Filter pattern or regex

Events

  • change: Emitted when the input value changes

Predefined Patterns

Pattern Description Regex
int Integers only /[0-9]/
num Numbers with decimals /[0-9.\-+]/
alpha Letters only /[A-Za-z]/
alphanum Letters and numbers /[A-Za-z0-9]/

Examples

Basic Usage

<sui-key-filter
  [(ngModel)]="username"
  pattern="alphanum"
  placeholder="Enter username"
  (change)="onUsernameChange($event)"
></sui-key-filter>

Custom Pattern

<sui-key-filter
  [(ngModel)]="phone"
  pattern="^[0-9-()+ ]*$"
  placeholder="(555) 123-4567"
  name="phone"
></sui-key-filter>

Form Integration

<form (ngSubmit)="onSubmit()">
  <sui-key-filter

    placeholder="email@domain.com"
    name="email"

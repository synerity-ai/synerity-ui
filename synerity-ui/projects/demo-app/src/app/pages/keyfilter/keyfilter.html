Available Patterns

int

Only digits (0-9)

num

Digits, decimal point, and +/- signs

alpha

Only letters (A-Z, a-z)

alphanum

Letters and numbers (A-Z, a-z, 0-9)

Custom Regex

Any valid regular expression pattern

Props

Prop Type Default Description
ngModel string '' Two-way binding for input value
pattern 'int' | 'num' | 'alpha' | 'alphanum' | string 'alphanum' Pattern to filter input characters
placeholder string '' Placeholder text for the input
disabled boolean false Disable the key filter component
name string '' Name attribute for form submission
inputId string '' ID attribute for the input element

Events

Event Parameters Description
change string Callback when input value changes

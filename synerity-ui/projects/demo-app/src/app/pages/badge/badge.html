Badge

Status indicators and notification counters for UI elements.

Basic Badge

<button class="relative px-4 py-2 bg-blue-600 text-white rounded-md">
  Messages
  <sui-badge [value]="badgeCount" class="absolute -top-2 -right-2"></sui-badge>
</button>

Badge Variants

<sui-badge [value]="count" severity="info"></sui-badge>
<sui-badge [value]="count" severity="success"></sui-badge>
<sui-badge [value]="count" severity="warning"></sui-badge>
<sui-badge [value]="count" severity="danger"></sui-badge>
<sui-badge [value]="count" severity="contrast"></sui-badge>

Badge Sizes

Sizes: Choose from small, normal, and large sizes to fit different UI contexts. 

<sui-badge [value]="count" size="small"></sui-badge>
<sui-badge [value]="count" size="normal"></sui-badge>
<sui-badge [value]="count" size="large"></sui-badge>

Installation

The Badge component is part of the Synerity UI library. Install it using npm:

npm install @synerity/ui

Basic Usage

Import and use the Badge component in your Angular application:

Badge with Elements

Use badges with buttons, icons, or other UI elements:

<button class="relative">
  Button Text
  <sui-badge [value]="count" class="absolute -top-2 -right-2"></sui-badge>
</button>

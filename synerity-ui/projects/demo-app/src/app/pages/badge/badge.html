Installation

npm install @synerity/ui

Basic Usage

<sui-badge value="New"></sui-badge>

Import

import { Badge } from '@synerity/ui';

Examples

Badge with Severity

Badge with Size

Number Badge

<sui-badge [value]="25" severity="danger"></sui-badge>

Custom Styled Badge

<sui-badge 
  value="Custom"
  severity="info"
  size="large"
  styleClass="custom-badge">
</sui-badge>

Props

Prop Type Default Description
value string | number '' Value to display in the badge
severity 'info' | 'success' | 'warning' | 'danger' | 'contrast' 'info' Visual severity level of the badge
size 'small' | 'normal' | 'large' 'normal' Size of the badge
style any {} Inline styles for the badge
styleClass string '' CSS class name for custom styling

Accessibility

  • • Proper ARIA labels for screen readers
  • • WCAG 2.1 AA compliant color contrast
  • • Semantic meaning through severity levels
  • • High contrast mode support

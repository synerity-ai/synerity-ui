Features

Rich Text Formatting

Bold, italic, underline, lists, and more

Customizable Toolbar

Show/hide toolbar and customize appearance

Flexible Sizing

Custom height and responsive design

State Management

Disabled, readonly, and interactive states

Props

Prop Type Default Description
value string '' HTML content of the editor
placeholder string 'Enter text...' Placeholder text when editor is empty
disabled boolean false Disable the editor
readonly boolean false Make the editor read-only
height string '200px' Height of the editor
showToolbar boolean true Show/hide the formatting toolbar
style any {{ '{' }}{{ '}' }} Inline styles for the editor
styleClass string '' CSS class for the editor

Events

Event Parameters Description
onChange string Callback when content changes
onFocus void Callback when editor gains focus
onBlur void Callback when editor loses focus

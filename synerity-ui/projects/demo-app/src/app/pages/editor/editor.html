Editor

A rich text editor component with toolbar, formatting options, and customizable themes.

Basic Editor

<sui-editor 
  [value]="basicContent"
  placeholder="Enter your text here..."
  height="300px"
  (onChange)="onEditorChange($event)"
  (onFocus)="onEditorFocus()"
  (onBlur)="onEditorBlur()"
></sui-editor>

Custom Editor

<sui-editor 
  [value]="customContent"
  placeholder="Advanced editor with custom content..."
  height="400px"
  theme="light"
  [showToolbar]="true"
  (onChange)="onEditorChange($event)"
  (onFocus)="onEditorFocus()"
  (onBlur)="onEditorBlur()"
></sui-editor>

Read-Only Editor

<sui-editor 
  [value]="readonlyContent"
  placeholder="This content is read-only..."
  height="250px"
  [readonly]="true"
  [showToolbar]="false"
  (onChange)="onEditorChange($event)"
  (onFocus)="onEditorFocus()"
  (onBlur)="onEditorBlur()"
></sui-editor>

How to Use Editor

Installation

Import the Editor component in your Angular module:

import { Editor } from '@synerity/ui-lib';

@NgModule({
  imports: [
    Editor
  ]
})
export class YourModule { }

Basic Usage

Use the Editor component in your template:

<sui-editor 
  [value]="content"
  placeholder="Enter your text here..."
  height="300px"
  (onChange)="onContentChange($event)">
</sui-editor>

Form Integration

The Editor implements ControlValueAccessor and works seamlessly with Angular reactive forms:

<form [formGroup]="myForm">
  <sui-editor
    formControlName="content"
    placeholder="Enter your content"
    height="400px">
  </sui-editor>
</form>

Editor Documentation

Component Overview

The Editor component provides a rich text editing interface with toolbar, formatting options, and customizable themes. It supports HTML content editing with a comprehensive set of formatting tools.

Props

Property Type Default Description
value string '' Editor content (HTML)
placeholder string 'Enter text...' Placeholder text
disabled boolean false Disable the editor
readonly boolean false Make editor read-only
height string '200px' Editor height
showToolbar boolean true Show toolbar
theme 'light' | 'dark' 'light' Editor theme
style any {} Custom styles
styleClass string '' Custom CSS class

Events

Event Type Description
onChange EventEmitter<string> Emitted when content changes
onFocus EventEmitter<void> Emitted when editor gains focus
onBlur EventEmitter<void> Emitted when editor loses focus

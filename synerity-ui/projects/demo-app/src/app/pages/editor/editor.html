Editor

A rich text editor component with toolbar, formatting options, and customizable themes.

Basic Editor

<sui-editor 
  [value]="basicContent"
  placeholder="Enter your text here..."
  height="300px"
  (onChange)="onEditorChange($event)"
  (onFocus)="onEditorFocus()"
  (onBlur)="onEditorBlur()"
></sui-editor>

Custom Editor

<sui-editor

  placeholder="Advanced editor with custom content..."
  height="400px"
  theme="light"
  [showToolbar]="true"

  (onBlur)="onEditorBlur()"

How to Use Editor

Installation

Import the Editor component from the UI library:

import { Editor } from 'ui-lib';

// In your component imports
imports: [Editor]

Basic Usage

Use the component with basic configuration:

<sui-editor 
  [value]="content"
  placeholder="Enter text..."
  (onChange)="onContentChange($event)"
></sui-editor>

Component Properties

  • value: Initial HTML content
  • placeholder: Placeholder text when empty
  • disabled: Boolean to disable the editor
  • readonly: Boolean to make editor read-only
  • height: Editor height (CSS value)
  • showToolbar: Boolean to show/hide toolbar
  • theme: Editor theme ('light' or 'dark')
  • style: Custom CSS styles
  • styleClass: Custom CSS class

Events

  • onChange: Emitted when content changes
  • onFocus: Emitted when editor gains focus
  • onBlur: Emitted when editor loses focus

Features

  • Rich Text Formatting: Bold, italic, underline, strikethrough
  • Headers and Paragraphs: Various heading levels
  • Lists: Ordered and unordered lists
  • Links and Images: Insert and edit links/images
  • Code Blocks: Insert and format code
  • Tables: Create and edit tables
  • Undo/Redo: Full undo/redo functionality
  • Keyboard Shortcuts: Standard editor shortcuts

Keyboard Shortcuts

  • Ctrl/Cmd + B: Bold text
  • Ctrl/Cmd + I: Italic text
  • Ctrl/Cmd + U: Underline text
  • Ctrl/Cmd + Z: Undo
  • Ctrl/Cmd + Y: Redo
  • Ctrl/Cmd + Shift + Z: Redo (alternative)

API Documentation

Properties

Property Type Default Description
value string '' Editor content (HTML)
placeholder string 'Enter text...' Placeholder text
disabled boolean false Disable the editor
readonly boolean false Make editor read-only
height string '200px' Editor height
showToolbar boolean true Show toolbar
theme 'light' | 'dark' 'light' Editor theme
style any {} Custom styles
styleClass string '' Custom CSS class

Events

Event Type Description
onChange EventEmitter<string> Fired when content changes
onFocus EventEmitter<void> Fired when editor gains focus
onBlur EventEmitter<void> Fired when editor loses focus

Methods

  • execCommand(command, value?): Execute formatting commands
  • isFormatActive(format): Check if format is active
  • insertLink(): Insert a link
  • insertImage(): Insert an image
  • insertCode(): Insert code block
  • insertText(html): Insert HTML text
  • onHeadingChange(event): Change heading format

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Keyboard Navigation: Full keyboard support
  • Focus Management: Proper focus handling
  • Screen Reader: Descriptive labels and announcements

Styling

The component uses CSS classes that can be customized:

  • .sui-editor: Main container
  • .sui-editor-toolbar: Toolbar styling
  • .sui-editor-content: Editor content area
  • .sui-editor-disabled: Disabled state styling
  • .sui-editor-readonly: Read-only state styling
  • .sui-editor-dark: Dark theme styling

Examples

Basic Usage

export class ExampleComponent {
  content = '<p>Initial content</p>';

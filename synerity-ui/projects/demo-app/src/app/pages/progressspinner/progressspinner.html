Progress Spinner

A loading indicator component with customizable size, animation, and styling options.

Basic Progress Spinner

A basic progress spinner with default settings.

Loading...
{{ loadingMessage }}

Basic: Default progress spinner with interactive loading simulation. 

<sui-progress-spinner 
  size="normal"
  strokeWidth="4"
  fill="transparent"
  animationDuration="2s">
</sui-progress-spinner>

Installation

npm install ui-lib

Basic Usage

Import the ProgressSpinner component and use it in your template:

import {{ '{' }} ProgressSpinner {{ '}' }} from 'ui-lib';

// In your template
<sui-progress-spinner></sui-progress-spinner>

Common Use Cases

Loading States

Show progress spinners during data loading, form submissions, or API calls

Button Loading

Display small spinners inside buttons during async operations

Page Loading

Use large spinners for full-page loading indicators

Properties

Property Type Default Description
strokeWidth string '2' Width of the spinner stroke
fill string 'transparent' Fill color of the spinner circle
animationDuration string '2s' Duration of the rotation animation
size 'small' | 'normal' | 'large' 'normal' Size of the progress spinner
style any {} Inline styles for the component
styleClass string '' CSS class for the component

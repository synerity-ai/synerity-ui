How to Use Progress Spinner

Installation

Import the ProgressSpinner component from the UI library:

import { ProgressSpinner } from 'ui-lib';

// In your component imports
imports: [ProgressSpinner]

Basic Usage

Use the component with basic configuration:

<sui-progress-spinner></sui-progress-spinner>

Component Properties

  • size: Size of the spinner ('small', 'normal', 'large')
  • strokeWidth: Width of the spinner stroke
  • fill: Fill color of the spinner background
  • animationDuration: Duration of the rotation animation
  • style: Custom CSS styles
  • styleClass: Custom CSS class

Common Use Cases

  • Loading States: Show loading indicators during async operations
  • Data Fetching: Indicate when data is being loaded
  • Form Submission: Show progress during form processing
  • Page Transitions: Display during route changes
  • File Uploads: Indicate upload progress

Best Practices

  • Appropriate Sizing: Use small spinners for inline content, large for full-page loading
  • Context: Always provide loading text or context when possible
  • Duration: Show spinners for operations that take more than 200ms
  • Accessibility: Ensure screen readers can announce loading state
  • Consistency: Use consistent spinner styling across your application

Conditional Rendering

<div *ngIf="isLoading" class="loading-container">
  <sui-progress-spinner size="normal"></sui-progress-spinner>
  <p>Loading data...</p>
</div>

API Documentation

Properties

Property Type Default Description
size 'small' | 'normal' | 'large' 'normal' Size of the spinner
strokeWidth string '2' Width of the spinner stroke
fill string 'transparent' Fill color of the background
animationDuration string '2s' Duration of the rotation animation
style any {} Custom CSS styles
styleClass string '' Custom CSS class

Methods

  • getSpinnerClass(): Returns CSS classes for the spinner
  • getSpinnerStyle(): Returns inline styles for the spinner
  • getCircleStyle(): Returns styles for the spinner circle
  • getSize(): Returns the numeric size value
  • getRadius(): Returns the radius of the spinner circle
  • getCircumference(): Returns the circumference of the circle

Accessibility

  • ARIA: Proper ARIA attributes for screen readers
  • Live Region: Announces loading state changes
  • Focus Management: Does not interfere with focus
  • Screen Reader: Descriptive labels and announcements

Styling

The component uses CSS classes that can be customized:

  • .sui-progress-spinner: Main container
  • .sui-progress-spinner-small: Small size styling
  • .sui-progress-spinner-normal: Normal size styling
  • .sui-progress-spinner-large: Large size styling
  • .sui-progress-spinner-circle: Spinner circle styling

ProgressSpinner

A loading spinner component with customizable size, color, and animation.

Basic ProgressSpinner

A basic progress spinner for loading states.

Size: {{ basicSize }} 
<sui-progress-spinner
  [visible]="basicVisible"
  [size]="basicSize">
</sui-progress-spinner>

Custom progress spinner with different styling and configuration.

  [size]="customSize"

Getting Started

Import the ProgressSpinner component and add it to your template:

import { ProgressSpinner } from 'ui-lib';

<sui-progress-spinner [visible]="true" [size]="medium">
</sui-progress-spinner>

Basic progress spinner usage:

<sui-progress-spinner
  [visible]="isLoading"
  [size]="large"
  [color]="#3b82f6"
  [strokeWidth]="4">
</sui-progress-spinner>

API Reference

Properties

Property Type Default Description
visible boolean true Controls spinner visibility
size string 'medium' Size of the spinner (small, medium, large)
color string '#3b82f6' Color of the spinner
strokeWidth number 3 Width of the spinner stroke
animation string 'spin' Animation type (spin, pulse, none)

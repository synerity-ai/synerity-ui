How It Works

CSS Custom Properties

The library defines CSS variables that apps can override: 

:root {
  --sui-color-primary-600: #9333ea;
  --sui-color-primary-700: #7c3aed;
  --sui-color-secondary-600: #2563eb;
  --sui-color-secondary-700: #1d4ed8;
}
App Theme Override

Apps can override any color by redefining the variables: 

.theme-corporate {
  --sui-color-primary-600: #1d4ed8;
  --sui-color-primary-700: #1e3a8a;
  --sui-color-secondary-600: #047857;
  --sui-color-secondary-700: #065f46;
}
Benefits
  • ✅ True theme inheritance - apps control all colors
  • ✅ No conflicts - library and app colors don't conflict
  • ✅ No synchronization needed
  • ✅ Backward compatible with existing Tailwind configs
  • ✅ Easy customization through CSS variables

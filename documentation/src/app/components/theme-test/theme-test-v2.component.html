How It Works

SCSS Mixin System

Components use theme mixins that generate CSS variables: 

.sui-button-primary {
  @include sui-button-theme();
  background-color: var(--sui-primary-600);
}
Runtime Theme Switching

Apps can switch themes by applying CSS classes: 

.theme-corporate {
  --sui-primary-600: #1d4ed8;
  --sui-secondary-600: #047857;
}
Benefits
  • ✅ No CSS specificity conflicts
  • ✅ Clean separation of concerns

  • ✅ Flexible runtime theme switching

Installation Guide

Install @synerity/ui

Choose your preferred package manager and follow the installation instructions. Get Synerity UI up and running in your Angular project in minutes.

Quick Start View Components

Package Managers

Choose your preferred package manager and install Synerity UI

NPM

Most popular package manager for Node.js

Terminal 
npm install @synerity/ui

Yarn

Fast, reliable, and secure package manager

Terminal 
yarn add @synerity/ui

PNPM

Fast, disk space efficient package manager

Terminal 
pnpm add @synerity/ui

Configuration

Configure your Angular application to use Synerity UI

1

Import Styles

Add the CSS to your angular.json or import in your styles.scss

styles.scss 
@import '@synerity/ui/styles';
2

Configure Angular

Update your angular.json to include the styles

angular.json 
{{ '{' }}
  "styles": [
    "node_modules/@synerity/ui/styles.css",
    "src/styles.scss"
  ]
{{ '}' }}
3

Import Components

Import the components you need in your Angular modules

component.ts 
import {{ '{' }} Button, Card {{ '}' }} from '@synerity/ui';

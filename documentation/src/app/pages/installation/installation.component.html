Installation

Detailed installation instructions for Synerity UI

NPM Installation

Install Synerity UI using npm:

npm install @synerity/ui

Yarn Installation

Install Synerity UI using yarn:

yarn add @synerity/ui

PNPM Installation

Install Synerity UI using pnpm:

pnpm add @synerity/ui

Configuration

After installation, you need to configure your Angular application to use Synerity UI.

1. Import Styles

Add the CSS to your angular.json or import in your styles.scss:

@import '@synerity/ui/styles';

2. Configure Angular

Update your angular.json to include the styles:

{{ '{' }}
  "styles": [
    "node_modules/@synerity/ui/styles.css",
    "src/styles.scss"
  ]
{{ '}' }}

Getting Started

Learn how to integrate Synerity UI into your Angular application

1. Installation

Install Synerity UI using npm:

npm install @synerity/ui

2. Import Styles

Add the CSS to your angular.json or import in your styles.scss:

@import '@synerity/ui/styles';

3. Start Using Components

You can now use Synerity UI components in your templates:

<button class="btn btn-primary">Click me</button>

4. Explore Components

Browse our component library to see all available components and their usage examples.

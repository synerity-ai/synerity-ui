Quick Setup Steps

Follow these simple steps to integrate Synerity UI into your Angular application

1

Install Package

Install Synerity UI using npm, yarn, or pnpm

Terminal 
npm install @synerity/ui
2

Import Styles

Add the CSS to your angular.json or import in styles.scss

styles.scss 
@import '@synerity/ui/styles';
3

Use Components

Import and use Synerity UI components in your templates

component.ts 
import {{ '{' }} Button {{ '}' }} from '@synerity/ui';
template.html 
<sui-button variant="primary">Click me</sui-button>
4

Quick Setup

Get your theme configuration up and running in minutes

1

Install TailwindCSS

Set up TailwindCSS in your Angular project

Terminal 
npm install -D tailwindcss postcss autoprefixer
2

Configure TailwindCSS

Create tailwind.config.js with Synerity UI theme variables

tailwind.config.js 
{{ tailwindConfigCode }}
3

Import Styles

Add Synerity UI styles to your global stylesheet

styles.scss 
@import '@synerity/ui/styles';
4

Customize Theme

Override CSS variables to match your brand colors

styles.scss 
{{ customThemeCode }}
